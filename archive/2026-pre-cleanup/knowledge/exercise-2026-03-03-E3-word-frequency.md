# 練習 E-3：實作單字頻率統計 (Word Frequency)

### 1. 目標
使用 code_eval 實作一個單字頻率統計函式，輸入為文字字串，輸出為單字出現次數的 Map（或物件）。

### 2. 實作邏輯
- 將文字轉為小寫。
- 使用正則表達式 \w+ 提取所有單字。
- 使用 Map 累加每個單字的出現次數。

### 3. 測試輸入 (Health API JSON)
{"status":"ok","uptime":17142,"version":"2.4.1","timestamp":"2026-03-03T21:45:00Z"}

### 4. 測試結果
成功統計出 status, ok, uptime, version 等單字的出現頻率。

### 5. 結論
透過 Map 結構可高效處理大量文本的單字統計，適用於日誌分析與關鍵字提取。
