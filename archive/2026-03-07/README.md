# Simple Python CLI Tool

## Description
This is a basic Python command-line interface (CLI) tool that demonstrates how to handle command-line arguments.
- When executed without any arguments, it prints "Hello, World!".
- When executed with arguments, it concatenates them into a single string and prints the result.

## How to Run
1. Ensure you have Python installed on your system.
2. Navigate to the directory containing `main.py`.
3. Execute the script from your terminal.

## Example Usage

*   **Without arguments:**
    ```bash
    python main.py
    ```
    Output:
    ```
    Hello, World!
    ```

*   **With arguments:**
    ```bash
    python main.py This is a test
    ```
    Output:
    ```
    This is a test
    ```
