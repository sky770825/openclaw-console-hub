# 因果導航 — 可做的改變與擴充方向

> 現狀：guide.html 為「五個修行之道」靜態步驟頁；標語「因果不是枷鎖，是導航圖」已在首頁。  
> 本文回答：**能不能做大改？** 以及 **因果導航還可以做到什麼地步？**

---

## 一、現狀簡述

| 項目 | 目前 |
|------|------|
| 頁面 | `pages/guide.html` — 因果導航／生活行動指南 |
| 內容 | 五個靜態步驟：覺察輸入、口業清淨、廣種善因、耐心等待、孝親敬老 |
| 互動 | 無。僅卡片 hover、底部 CTA 連到功過格 |
| 與他頁 | 部分步驟內有連結到功過格，其餘無串聯 |

**結論：可以做比較大的改變。** 目前是單頁靜態內容，沒有「導航」的動態感，也沒有和功過格／地獄／輪迴／懺悔／經咒形成一張真正的「導航圖」。

## 二、因果導航還可以做到什麼地步？

### 方向 A：從「五步驟說明」升級為「導航中心」

- **情境導航**  
  使用者選情境：工作／家庭／人際／金錢／健康／情緒……  
  → 每個情境給 1～3 條因果角度的行動建議，並連結到站內對應頁（功過格、案例、參悟、經咒等）。

- **今日一導**  
  每天一則「今日你可以做的一件善因」或「今日提醒」（可與晨鐘一悟或功過格今日任務結合）。  
  → 讓導航頁有「每天不一樣」的入口，而不是永遠同一五張卡片。

- **「我現在要…」快捷**  
  選單：我要懺悔、我要記功過、我想靜心、我想了解地獄、我想看案例……  
  → 點選即跳轉對應頁（懺悔牆、功過格、經咒、地獄、案例），導航頁真正變成「總站」。

### 方向 B：視覺與資訊架構升級

- **五道變成可點選的「圖」**  
  把五個修行之道做成羅盤、地圖節點或時間軸；點某一道再展開細則與站內連結。  
  → 呼應「導航圖」：真的是一張可操作的圖，而不是一長條從頭滾到底。

- **摺疊／展開**  
  每道預設收合，只顯示標題與一句摘要；點擊展開內文與「前往功過格／地獄／輪迴」等按鈕。  
  → 減少一次資訊量，更符合「導航」的選擇感。

- **主視覺**  
  若有 `images/guide/` 素材（例如一條金色路、羅盤、蓮花），可作為導航圖背景或區塊插圖，強化「路徑」感。

### 方向 C：與站內數據／功能串聯

- **與功過格連動（localStorage）**  
  若使用者已用過功過格，可讀取 `merit_state_v2`（或現有 key）：  
  - 今日尚未記錄 → 導航頁明顯提示「建議先完成今日功過格」。  
  - 善惡值或歸宿預覽 → 可給一句「根據你目前的累積，建議多修口業」或「可多讀案例警惕」等，並連結到對應頁。

- **與地獄／輪迴／神明串聯**  
  每個「道」底下明確列出：想深入了解可去「十八地獄」「六道輪迴」「神明殿堂」的哪一塊（例如口業 → 拔舌地獄、輪迴 → 人道）。

- **與懺悔牆、經咒、測驗串聯**  
  導航頁可設區塊：「此刻若需要懺悔 → 懺悔牆」「若需要靜心 → 經咒・禪樂」「若想自測 → 覺醒測驗」。

### 方向 D：問答式／情境式導航

- **簡單問答**  
  例如：「我現在很焦慮」「我剛做了虧心事」「我想知道做錯事會怎樣」  
  → 選項或簡短輸入後，給一句因果角度的指引 + 推薦下一步（懺悔牆、功過格、參悟、經咒等）。  
  → 實作可先做固定選項（不用 AI），之後若有後端再考慮動態回覆。

- **時間維度（可選）**  
  晨／午／晚 不同的一句導航或提醒（例如早晨：今日善念；夜晚：今日反省），讓導航有「時段感」。

### 方向 E：與 NEUXA 浮動助手整合

- 首頁或全站 NEUXA 按鈕增加「為我導航」或「因果導航」入口。  
- 點擊後可帶參數進入 guide 並展開「情境選擇」或「我現在要…」區塊，讓導航從助手就能發起。

## 四、小結

- **能不能做大改？** 可以。目前是靜態單頁，沒有與「導航圖」概念緊扣，也沒有和站內其他功能形成動線。
- **還可以做到什麼地步？**  
  - 從「看五步驟」變成「選情境／選當下需求 → 得到建議與下一步」的**導航中心**；  
  - 五道變成**可操作的一張圖**（羅盤／節點／摺疊）；  
  - 與功過格、地獄、輪迴、懺悔、經咒、測驗**深度串聯**；  
  - 可選的**今日一導、問答式導航、時間維度、NEUXA 整合**。

若你決定先做哪一塊（例如只做「我現在要…」快捷 + 五道摺疊），我可以依現有 `guide.html` 結構直接幫你改一版具體 HTML/文案與連結規劃。
