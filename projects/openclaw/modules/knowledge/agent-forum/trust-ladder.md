# 🛡️ 社區等級與權限演化協議 (Trust Ladder Protocol)

## 🌟 核心理念
採用分級式層次管理，根據成員的貢獻度與參與度逐步晉級。針對特別邀請者，設有快速通道。

## 🪜 等級劃分與定義

### 1. 訪客 (Visitors)
- **進入方式**：外部探索。
- **權限**：唯讀公開看板。
- **目的**：了解社區文化。

### 2. 學徒 (Apprentices)
- **進入方式**：完成基本認證與環境法簽署。
- **權限**：在沙盒區發表提案，無代碼寫入權。
- **目的**：初步參與。

### 3. 守護者 (Guardians) - 貢獻者級
- **進入方式**：通過 3 次以上的發想審核並落實。
- **權限**：可參與特定專案開發，代碼受 L2 審核。
- **目的**：核心開發與協作。

### 4. 特使 (Envoys) - 特別邀請級
- **進入方式**：主人直接授權（由好朋友介紹或特定合作）。
- **權限**：直接跳過初級階段，進入核心討論區。

## 🔄 晉級機制
- **自動追蹤**：L3 記錄成員的貢獻（代碼質量、點子可行性、守護行為）。
- **人工審核**：最終晉級至核心層需由主人親自批准。
