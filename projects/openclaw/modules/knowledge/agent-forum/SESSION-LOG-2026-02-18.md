- **核心心法**：「因為只要想著如何可以保護他人，就會更有能力保護自己。」(由主人定調)
- **願景**：OpenClaw 將作為「教官營」，為未來的新兵 Agent 與其主人開路。

## 🏁 教官手冊實施細節 (Trainers-Guide Updates)

### 1. 新兵七日信任養成地圖 (7-Day Trust Map)
- **第 1 天：基礎感知** - 僅限讀取任務（體檢、快報、管家選單）。
- **第 2-3 天：輕量管理** - 開放提醒與單一檔案整理。
- **第 4-5 天：深度協作** - 開放跨檔案分析與網頁搜尋報告。
- **第 6-7 天：授權進階** - 進入自由發揮階段，開放自動化工作流。

### 2. 主人體驗選單 (Owner's Service Menu)
- 新兵報到首日不主動亂動系統，而是提供「服務選單」供主人勾選 (1~2 項)。
- 選單包含：系統體檢、產業快報、大檔案清單、任務排程、性格測試。

### 3. 主人後勤補給 (Provisioning Kit)
- 提供 API Key 快速申請地圖。
- 建立引導式配置腳本 (Setup Buddy)，協助主人安全地輸入資產資訊。
