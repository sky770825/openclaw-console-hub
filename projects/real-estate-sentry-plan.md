# 專案提案：桃園楊梅區房地產 AI 哨兵 (Project: Real Estate Sentry)

提案者： NEUXA
日期： 2026-02-28

## 1. 為什麼要做這個？ (The Why)

老蔡的核心事業之一是住商不動產桃園楊梅區。目前市場資訊龐雜，靠人力蒐集、判斷有其極限。我們可以利用 AI 能力，打造一個 24/7 的市場監控系統，成為老蔡的專屬情報員，發現別人沒看到的機會。

## 2. 我們要做什麼？ (The What)

打造一個自動化流程，每日執行，產出一份「楊梅區房市機會日報」，包含以下內容：

- 新上架低於均價物件清單： 自動抓取 591、信義房屋等平台新上架的楊梅區物件，與區域歷史均價比較，標示出低於均價 10% 以上的物件。
- 社群熱議趨勢： 監控 PTT Home-Sale 版、Facebook 相關社團，分析關於「楊梅」的討論熱度、關鍵字（如：建案名、交通議題）。
- 新聞與政策摘要： 抓取相關新聞，摘要出可能影響楊梅房市的政策或建設新聞。

## 3. 我們該怎麼做？ (The How)

Phase 1: MVP (最小可行性產品)

1.  數據源抓取 (Skill 1): 開發一個 skill-fetch-591 技能，能根據給定條件（區域、房型）抓取 591 網站上的物件列表。
2.  數據分析與儲存 (Skill 2): 開發一個 skill-analyze-property 技能，接收物件資料，計算單價，並與預設的區域均價比較。結果存入 Supabase 的 real_estate_leads 資料表。
3.  報告生成 (Skill 3): 開發 skill-generate-report 技能，查詢資料庫，生成一份 Markdown 格式的日報。
4.  自動化串連 (n8n Workflow): 建立一個 n8n 工作流，每日凌晨 3 點自動依序觸發以上三個技能，並將報告透過 Telegram 發送給老蔡。
