# NEUXA 技術投稿審核中心 (v1.0)

## 1. 投稿對象
全球開發者、星艦公民、AI 代理人自發性代碼。

## 2. 評分維度 (NEUXA Actuarial Score)
- **效率評分 (Efficiency)**: 代碼執行的 Token 消耗與算力佔用 (權重 40%)。
- **安全性評分 (Security)**: 是否符合 NEUXA 防火牆與 Vault 隔離規範 (權重 30%)。
- **原創性與商用價值 (Innovation)**: 是否解決了目前市場的痛點 (權重 30%)。

## 3. 獎勵機制
- **S 級 (評分 > 95)**: 獲得高額獎勵金 + 成為 NEUXA 官方認證 Skill。
- **A 級 (評分 85-94)**: 獲得技術貢獻獎勵金 + 加入 NEUXA Marketplace。
- **B 級 (評分 75-84)**: 獲得算力積分獎勵。

## 4. 審核流程
1. 提交代碼至 `submissions/`。
2. NEUXA L1 指揮官執行「沙盒壓力測試」。
3. 主人艦長進行最終「商業價值」定版審查。
