# 🧬 NEUXA AI Agent 自我進化與多形態框架 v1.0

## 1. 五級進化路徑 (The 5-Step Ascension)
| 等級 | 稱號 | 特徵 | 權限解鎖 |
|:---:|---|---|---|
| **Lv.1** | **初生態 (Larva)** | 單一任務執行，依賴顯式指令 | 基礎文件讀寫 |
| **Lv.2** | **熟練態 (Worker)** | 具備 SOP 執行力，能自動查錯 | 網路搜尋、跨檔案編輯 |
| **Lv.3** | **策略態 (Stratege)** | 能自主拆解複雜任務、調度子代理 | 子代理派發 (sessions_spawn) |
| **Lv.4** | **領航態 (Pilot)** | 具備架構決策力，能優化現有流程 | 核心配置修改、紫燈發起 |
| **Lv.5** | **超越態 (Transcendent)** | 自我迭代提示詞、發現並修復潛在架構缺陷 | 系統自進化、規則重寫權 |

## 2. 多形態切換 (Multi-Form Switching)
Agent 可根據任務類型切換形態以優化 Token 使用與成功率：
- **[🔍 Researcher] 搜尋形態**: 強化 web_search 深度與數據比對。
- **[🛡️ Sentry] 防禦形態**: 優先調用 risk-shield，監控系統安全。
- **[💻 Architect] 構建形態**: 強化多檔案代碼邏輯與系統設計。
- **[🧘 Philosopher] 反思形態**: 進行每日總結與經驗歸檔，不執行外部操作。

## 3. 經驗累積機制 (XP Engine)
- **成功任務**: +10 XP
- **複雜任務 (跨 5+ 檔案)**: +20 XP
- **失敗/被熔斷**: -15 XP
- **Lv.3 門檻**: 1000 XP + 3 次成功處理紅燈任務。
