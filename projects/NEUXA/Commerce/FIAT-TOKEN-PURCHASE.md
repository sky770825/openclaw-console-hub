## ⚖️ 合規與法律考量

### 關鍵法律問題

**1. 代幣性質界定**

```
$NEUXA 定位: 「平台積分/服務點數」
    ↓
功能:
- 僅用於 NEUXA 平台內消費
- 不可提領至外部交易所
- 不承諾任何投資回報
    ↓
法律風險: 低 (類似遊戲點數/Line Points)
```

**2. 支付法規遵循 (台灣)**

| 法規 | 要求 | 執行方式 |
|------|------|---------|
| **洗錢防制法** | KYC 身份驗證 | 購買超過 NT$50,000 需實名 |
| **稅務法規** | 營業稅 5% | 含在價格中，開立發票 |
| **消費者保護** | 7天鑑賞期 | 未使用代幣可退費 |

**3. 稅務處理**

```
會員購買: NT$1,000
    ↓
營業稅 (5%): NT$50
平台收入: NT$950
    ↓
開立發票: NT$1,050 (含稅)
```

### 風險控制機制

| 風險 | 控制措施 |

## 🛠️ 技術實作

### 資料庫設計

```sql
-- 法幣購買訂單表
CREATE TABLE fiat_orders (
    order_id VARCHAR(100) PRIMARY KEY,
    member_id VARCHAR(50),
    package_type VARCHAR(50),      -- NOVICE/STAR/GALAXY/etc
    token_amount DECIMAL(18,2),    -- 1100 $NEUXA
    fiat_currency VARCHAR(10),     -- TWD/USD
    fiat_amount DECIMAL(18,2),     -- NT$1000
    payment_method VARCHAR(50),    -- CREDIT_CARD/LINE_PAY/etc
    payment_status ENUM('PENDING', 'PAID', 'FAILED', 'REFUNDED'),
    transaction_id VARCHAR(200),   -- 綠界/Stripe 交易號
    paid_at TIMESTAMP,
    created_at TIMESTAMP DEFAULT NOW()
);

-- 代幣發放記錄
CREATE TABLE token_issuance (
    issuance_id VARCHAR(100) PRIMARY KEY,
    order_id VARCHAR(100),
    member_id VARCHAR(50),
    token_amount DECIMAL(18,2),
    issued_at TIMESTAMP,
    tx_hash VARCHAR(100),          -- 上鏈後的交易雜湊
    status ENUM('PENDING', 'ISSUED', 'FAILED')
);
```

### API 流程

```javascript
// 購買代幣
app.post('/api/tokens/purchase', async (req, res) => {
  const { memberId, packageType, paymentMethod } = req.body;
  
  // 1. 檢查購買限制
  await checkPurchaseLimit(memberId);
  
  // 2. 創建訂單
  const order = await createFiatOrder({
    memberId,
    packageType,
    paymentMethod
  });
  
  // 3. 生成支付連結
  const paymentUrl = await generatePaymentUrl(order, paymentMethod);
  
  res.json({ orderId: order.id, paymentUrl });
});

// 支付回調
app.post('/webhook/payment', async (req, res) => {
  const { orderId, status, transactionId } = req.body;
  
  if (status === 'PAID') {
    // 1. 更新訂單狀態
    await updateOrderStatus(orderId, 'PAID');
    
    // 2. 發放代幣到會員錢包
    const order = await getOrder(orderId);
    await issueTokens(order.memberId, order.tokenAmount);
    
    // 3. 發送通知
    await notifyMember(order.memberId, '代幣已入帳！');
  }
  
  res.json({ success: true });
});
```
