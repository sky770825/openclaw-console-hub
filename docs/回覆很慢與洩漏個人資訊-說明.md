# 回覆很慢 + 回覆裡出現一堆你的資訊

## 原因

1. **很慢**  
   本機 Ollama（mistral / llama3.2 / qwen2.5）在一般電腦上本來就慢，常要十幾秒到一兩分鐘才回一句。

2. **一堆你的資訊**  
   OpenClaw 會把 **workspace 裡的檔案**（例如 `USER.md`、`MEMORY.md`、`IDENTITY.md`）當成「背景知識」送給模型。  
   **本機小模型**（Ollama）常會把這段背景直接當成回覆內容講出來，所以你會看到自己的名字、Telegram、生日、工作等被貼出來。

---

## 已幫你做的設定

- **預設模型**已改成 **`google/gemini-2.5-flash`**（雲端）  
  - 回覆會**快很多**  
  - 比較**不會**把 USER.md / MEMORY 當成回覆內容貼出來  

其他模型（含 Ollama）都還在，要切換再用指令或改設定即可。

---

## 若之後又想用本機 Ollama

- 本機模型**本來就慢**，且**比較容易**把 workspace 內容當回覆。  
- 若還是想用：可把 **`~/.openclaw/workspace/USER.md`** 裡太私密的段落刪掉或改簡短，只留「怎麼稱呼你」等必要資訊，可減少被整段貼出來的機率。  
- 或開新對話（`/new`）再試，有時會好一點。

---

## 切換回 Ollama 的方式

在 `~/.openclaw/openclaw.json` 裡把：

`"primary": "google/gemini-2.5-flash"`

改成例如：

- `"primary": "ollama/mistral"`  
- 或 `"primary": "ollama/llama3.2"`

存檔後重啟 Gateway（或等自動重載）。
