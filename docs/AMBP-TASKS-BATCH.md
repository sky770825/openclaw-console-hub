# AMBP Phase 3+ 任務規劃

## 🔥 高優先（立即價值）

### 1. CellCog Connector
**描述**: 整合 CellCog Python SDK，讓研究任務完成後自動更新任務板
**類型**: `openclaw`
**預估**: 2-3 小時
**驗收標準**:
- [ ] 監控 CellCog chat 狀態
- [ ] 研究完成自動發送 completion 訊息
- [ ] 支援 PDF/報告檔案路徑傳遞

### 2. AMBP 自動化規則引擎
**描述**: 讓任務可以設定「當 X 完成時自動指派給 Y」
**類型**: `cursor`
**預估**: 4-6 小時
**驗收標準**:
- [ ] 支援 if-this-then-that 規則設定
- [ ] 任務完成自動觸發下一個任務
- [ ] 視覺化規則編輯器（可選）

**類型**: `cursor`
**預估**: 2-3 小時
**驗收標準**:
- [ ] 使用 macOS Keychain 或檔案加密
- [ ]  connector 自動讀取解密
- [ ] 移除所有明文 API Key

### 4. AMBP 監控儀表板

**類型**: `cursor` + `openclaw`
**預估**: 6-8 小時
**驗收標準**:
- [ ] 即時訊息流顯示
- [ ] Agent 健康狀態
- [ ] 任務完成統計圖表
