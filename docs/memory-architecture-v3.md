## 安全流程

寫入任何記憶前：
┌─────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────┐
│  輸入內容  │───►│ memfw-scan  │───►│   判斷結果   │───►│  寫入   │
└─────────┘    │  三層掃描    │    └─────────────┘    └─────────┘
               └─────────────┘         │
                    │                  │
              Layer 1: 正則快掃      PASS → 允許寫入
              Layer 2: 語意分析      REVIEW → 人工審查
              Layer 3: 深度檢查      BLOCK → 拒絕寫入

## 檔案說明

| 檔案/腳本 | 用途 | 更新時機 |
|-----------|------|----------|
| NOW.md | 當前任務狀態 | 重大進展後、Context 70%、會話結束前 |
| SESSION-STATE.md | Session 期間工作記憶 | 即時更新 |
| checkpoint.sh | 自動備份腳本 | Context 達 70% 自動觸發 |
| memfw-scan.sh | 安全掃描 | 寫入記憶前必須執行 |
| MEMORY.md | 長期記憶索引 | 每日彙整 |
| memory/anchors/ | 完整任務歷史 | 重要任務完成時 |

## 使用流程

1. **會話開始**：讀 NOW.md → SOUL.md → USER.md → MEMORY.md
2. **任務進行中**：更新 SESSION-STATE.md
3. **寫入記憶前**：執行 memfw-scan.sh 掃描
4. **Context 達 70%**：自動觸發 checkpoint.sh
5. **會話結束前**：更新 NOW.md + 寫入 daily log
6. **每日結束**：彙整到 MEMORY.md
