# 日誌與快取清理建議

## OpenClaw 本機（~/.openclaw）

- **logs/** — 約 6.4MB，一般可保留；若需輪替可手動刪除舊的 `.jsonl` / `.log`。
- **browser/** — 約 1.1GB（Chrome 擴充與 OpenClaw 瀏覽用快取）。可安全刪除：
  - 僅刪除 `.old` 備份（約 78 個）：  
    `find ~/.openclaw/browser -type f -name '*.old' -delete`
  - 若不需要保留瀏覽狀態，可整包刪除後由 OpenClaw 重建：  
    `rm -rf ~/.openclaw/browser/chrome-extension/data ~/.openclaw/browser/openclaw/*`  
    （依實際子目錄調整，建議先 `ls ~/.openclaw/browser` 確認）

## 任務面版 / server

- 專案內 **logs** 已在 `.gitignore`，本機產生的 log 不會被提交。
- 若 server 有寫入 log 檔，可依需求設定輪替（例如 logrotate 或依日期的檔名）。
