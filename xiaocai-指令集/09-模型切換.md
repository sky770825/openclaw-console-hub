## 🚀 快速切換

### 切換到 Gemini（免費額度）

```
模型切換 gemini
```

**效果**：
- 主要模型：Gemini 2.5 Flash（免費）
- 每月可省：約 $6-10 (NT$180-300)

**限制**：
- Rate Limit：每分鐘請求數有限制
- 複雜推理任務品質略低於 Kimi

---

### 切換回 Kimi（品質優先）

```
模型切換 kimi
```

**效果**：
- 主要模型：Kimi K2.5（付費）
- 備援模型：Gemini 2.5 Flash（免費）

**特點**：
- 推理品質較高
- 中文理解較好
- 成本較高

## ⚙️ 技術說明

**切換原理**：
- 修改 `~/.openclaw/config.json`
- 自動備份原設定（`.backup-YYYYMMDD`）
- 立即生效，無需重啟

**四層備援架構**：
```
L1: Kimi K2.5（主要）
L2: Claude Code（技術開發）
L3: Gemini Flash（免費備援）
L4: Cursor（終極備援）
Local: Ollama（本地子派，零成本）
```

## ⚠️ 注意事項

1. **Gemini 免費額度**：每天 1,500 次請求
2. **自動降級**：額度用完會自動切換備援模型
3. **備份機制**：每次切換會自動備份原設定
4. **恢復方式**：可手動改回或執行對應切換指令
