# Live2D 生活圈概念規劃書 (v1.0)
**提案人：阿策 (Ace) & 阿研 (Ken)**

## 1. 核心願景
打造一個「有溫度」的工作與生活伴侶系統。Live2D 不僅是看板娘，而是任務系統的靈魂與情感介面。

## 2. 初步功能規劃
- **情緒同步 (Emotion Sync)**: 角色表情隨任務進度、壓力值或天氣變化。
- **動態互動 (Dynamic Interaction)**: 點擊、語音喚醒、自定義問候語。
- **任務聯動 (Task Linking)**: 當任務超時，角色表現出焦慮；任務完成，角色進行慶祝動畫。
- **環境感知 (Context Awareness)**: 根據當前時間（早中晚）切換服裝或背景。

## 3. 技術方案 (阿研建議)
- **渲染引擎**: PixiJS + cubism-web-sdk。
- **狀態管理**: 使用 Zustand 或 Redux 控制 Live2D 的 `Motion` 與 `Expression`。
- **通訊協議**: WebSocket 連接後端，接收即時事件觸發動作。
- **資源格式**: 模型文件採用 .model3.json 標準格式。

## 4. 互動模式
- **主動模式**: 角色主動提醒事項。
- **被動模式**: 使用者點擊觸發反饋。
- **掛機模式**: 角色進行隨機待機動作（如讀書、打瞌睡）。
