# M3 Ultra 遷移三階段計畫 (阿策)

## 第一階段：環境初始化 (Day 1-2)
- 硬體開箱與系統加固。
- 安裝 Homebrew, Python, MLX, Ollama 環境。
- 完成基礎 7B 模型跑分測試。

## 第二階段：邏輯遷移與影子測試 (Day 3-7)
- 將「阿數」的數據處理邏輯遷移至本地。
- 實行「雙發機制」：同時向雲端與本地發送請求，比對回傳質量。
- 調整 Token 統計路徑至本地目錄。

## 第三階段：全面接軌 (Day 8-10)
- 切斷雲端 API 依賴，切換至本地 MLX/Ollama API。
- 啟動全天候監控腳本。
- 正式交付《M3 新機啟動與接軌手冊》。
