# 技能活化計畫 v1.0

## 🎯 目標
將 55 個靜態技能模組化，轉化為 NEUXA 可隨時調用的動態工具。

## 🛠 第一波活化名單 (有腳本且環境就緒)
1. playwright-scraper-skill: 網頁自動化抓取 (基礎)
2. tavily-search: 實時聯網搜尋 (情報)
3. skill-creator: 自我進化工具 (核心)
4. healthcheck: 系統自檢 (穩定)

## 📋 執行步驟
1. 環境驗收: 檢查 playwright 與 API Keys。
2. 工具掛載: 將技能腳本路徑加入 run_script 白名單或封裝為 action。
3. 實戰測試: 分派一個爬蟲或搜尋任務驗證。
