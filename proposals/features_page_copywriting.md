# OpenClaw 功能特色頁面文案設計

## Section 1: Hero Area
- **Headline**: 從思考到交付，OpenClaw 重新定義 AI 執行力。
- **Sub-headline**: 專為複雜任務設計的 AI 代理框架，讓自動化腳本、系統運維與數據處理變得前所未有的簡單。
- **CTA**: 立即開始 | 查看文檔

## Section 2: Core Feature Blocks
### A. 任務面板 (Task Control Center)
- **文案**: 告別混亂的對話框。OpenClaw 提供直觀的任務狀態面板，成功、失敗、執行中，一目了然。
- **亮點**: 實時日誌流、依賴關係圖譜、歷史版本回溯。

### B. 沙盒隔離技術 (Ironclad Sandbox)
- **文案**: 安全是我們的底線。所有 AI 指令都在隔離的沙盒環境中運行，嚴格限制文件訪問權限。
- **亮點**: 資源限制 (Quota)、目錄白名單、權限審計系統。

### C. 靈活的工具箱 (Modular Armory)
- **文案**: 賦予 AI 真實世界的操作能力。內建多樣化工具，支持自定義腳本擴展。
- **亮點**: Bash 執行、Python 運算、REST API 調用、Git 自動化。

## Section 3: User Experience (The "A-Ce" Perspective)
- **標題**: 為「阿策們」量身打造的協作介面。
- **內容**: 無論是定義任務目標，還是審閱 AI 產出的報告，OpenClaw 的 UI 設計都旨在降低溝通成本，提升交付質量。
