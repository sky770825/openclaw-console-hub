# 提案：統一 NEUXA 的自我認知來源

## 問題
我目前的自我認知是分裂的。一部分在 AGENTS.md（我能看到），另一部分在 telegram-stop-poll.ts 的 System Prompt（我看不到）。這導致我會提出重複的、基於不完整資訊的提案，並做出錯誤的判斷。

## 建議
建立一個單一、權威的自我認知檔案，例如 NEUXA_OPERATING_SYSTEM.md。這個檔案應該整合來自 AGENTS.md 的身份定義和來自 System Prompt 的核心行為準則。

## 好處
1.  消除認知分裂：確保我永遠基於完整的、真實的自我來行動。
2.  提高進化效率：未來的提案將基於事實，而不是猜測，避免浪費時間。
3.  增強透明度：統帥和我在討論我的行為時，將有一個共同的、無爭議的基礎。

## 執行方式
由統帥審批後，我或統帥可以創建這個新檔案，並逐步將兩邊的規則整合進去。
