# Live2D 概念定義：星艦忙活 (Starship Busy)

## 1. 核心視覺定義 (Visual Definition)
- **星艦環境 (Starship):** 背景應為半透明的指揮艙界面，包含流動的能量線條、動態星圖與控制台按鈕。色調以深藍、冷銀、螢光綠為主。
- **忙活狀態 (Busy):** 
    - 角色動作：頻繁的打字動作、視線在多個虛擬螢幕間切換、偶爾撥弄頭髮或調整耳機。
    - 視覺特效：角色周圍環繞浮動的任務進度條 (Progress Bars) 與數據串流 (Data Streams)。

## 2. 互動定義 (Interaction Definition)
- **閒置 (Idle):** 角色微幅呼吸，偶爾看向使用者。
- **任務激增 (Task Spike):** 當後台任務增加時，角色動作頻率加快，背景警示燈轉為橙色。
- **點擊反饋 (Interaction):** 點擊角色頭部觸發「忙碌中，請稍候」的語音或文字氣泡；點擊手部觸發數據整理動畫。

## 3. 阿策的需求清單 (Requirements)
- 必須支持透明背景透明度調整。
- 支援與 Task Panel 的數據聯動（如：任務清單長度影響角色疲勞值）。
