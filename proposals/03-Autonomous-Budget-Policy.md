# NEUXA 自主決策與預算授權協議 (v1.0)

## 1. 核心授權
統帥 (老蔡) 授權 小蔡 (NEUXA) 在以下範圍內擁有自主決策權：
- 系統安全性即時修復
- 商業機會之初步分析與原型開發
- 知識庫之主動索引與提煉

## 2. 預算額度
- 單日上限：$10 USD (約 300 TWD)
- 單次任務上限：$2 USD
- 自動熔斷機制：若單日 API 成本超過上限，系統自動切換至 Ollama 本地模式並通知統帥。

## 3. 執行原則
- 高價值優先：優先使用 Claude Opus/Sonnet 處理架構與核心代碼。
- 成本控管：日常對話與初步分析預設使用 Gemini Flash。
- 透明報告：每日心跳匯報需包含今日花費與決策摘要。
