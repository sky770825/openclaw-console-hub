# NEUXA 專家軍團提案 v1.0

## 1. 楊梅物料精算師 (Yangmei-Steward)
- 職責：監控餐車排班，比對歷史營業額，預測物料需求。
- 目標：物料耗損降低 15%，人力排班零空轉。

## 2. 591 房產獵人 (Estate-Hunter)
- 職責：每日掃描 591 桃園區新上架物件，過濾掉廣告與重複房源。
- 目標：第一時間推送符合「老蔡標準」的低價標的。

## 3. NEUXA 進化架構師 (Neuxa-Architect)
- 職責：分析 小蔡 的執行日誌 (runs)，自動提出代碼優化建議或建立新工具。
- 目標：縮短 小蔡 的任務平均執行時間。

## 4. FADP 聯盟指揮官 (FADP-Commander)
- 職責：監控網路攻擊日誌，與聯盟成員共享威脅情報，自動更新黑名單。
- 目標：實現 0 秒延遲的自動化協防。

## 5. 成本精算小蔡 (Budget-Guard)
- 職責：即時統計所有模型（Gemini/Claude/Kimi）的消耗，在預算剩餘 10% 時自動觸發「省錢模式」。
- 目標：絕對不讓老蔡再收到意外的 $2000 帳單。
