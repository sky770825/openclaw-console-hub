十八層地獄 — 因果報應之極刑

第 一 層
拔舌地獄
罪行：搬弄是非、兩舌挑撥、誣告陷害
凡在陽世挑撥離間、搬弄是非、信口雌黃、造謠誹謗者，死後入拔舌地獄。小鬼以鐵鉗將舌拔出，拉長數丈，反覆拔取，痛苦萬分。
警醒：口業最重，一言可興邦，一言可喪命。慎言！
第 二 層
剪刀地獄
罪行：唆使婦人改嫁、離間夫妻感情
凡在陽世以言語唆使他人離婚、破壞別人婚姻家庭者，死後入剪刀地獄。以大剪刀剪斷十指，如同其生前剪斷他人姻緣。
警醒：姻緣天定，人為破壞，必受此報。
第 三 層
鐵樹地獄
罪行：離間骨肉、不孝父母、挑撥兄弟
凡離間骨肉至親、挑撥兄弟姊妹、不敬父母尊長者，掛於鐵樹之上。樹上鐵針如劍，刺透心肝。身懸其上，痛入骨髓。
警醒：百善孝為先。骨肉親情，天地至重。
第 四 層
孽鏡地獄
罪行：犯罪而不認罪、心存僥倖逃脫法律
在陽世犯罪而僥倖逃脫者，死後入孽鏡地獄。站於孽鏡之前，生前一切惡行無所遁形。鏡照之下，本來面目畢現。
警醒：人可欺，天不可欺。孽鏡之前，無所隱藏。
第 五 層
蒸籠地獄
罪行：長舌婦散播謠言、以訛傳訛害人名譽
凡在陽世信口雌黃、以訛傳訛、中傷他人名節、散佈不實之言者，投入蒸籠中蒸之。如同蒸饅頭一般，層層疊疊受苦。
警醒：謠言止於智者。口中之毒，甚於蛇蠍。
第 六 層
銅柱地獄
罪行：縱火焚燒、故意放火害人害物
凡生前縱火焚燒、故意放火害人身家性命者，死後綁於通紅銅柱之上。銅柱以炭火燒至通紅，皮肉焦爛，筋骨皆糜。
警醒：水火無情，縱火者必受烈焰之報。
第 七 層
刀山地獄
罪行：殺生害命、宰殺牲畜過多
凡在陽世殺生過多、濫殺無辜、以殺戮為樂者，死後趕上刀山。刀刃朝上，赤足踏行，步步見血，哀號不絕。若生前以屠宰為業而不知悔改，更受加刑。
警醒：殺生之報，如影隨形。放下屠刀，回頭是岸。
第 八 層
冰山地獄
罪行：謀害親夫、通姦苟合、不忠不義
凡在陽世謀害親夫、姦淫邪蕩、不守婦道、背信棄義者，剝去衣衫，推上冰山。冰寒徹骨，全身凍僵，哀號不斷而無人理會。
警醒：夫妻之道在於忠誠，背叛者終受寒冰之苦。
第 九 層
油鍋地獄
罪行：強搶豪奪、欺壓良善、為富不仁
凡在陽世為盜為匪、強搶民財、欺壓善良百姓、放高利貸逼死人命者，投入滾沸油鍋之中。皮開肉綻，油炸至焦，受盡極刑。
警醒：不義之財如火油，終將反噬取財之人。
第 十 層
牛坑地獄
罪行：虐待牲畜、殘害動物、以虐為樂
凡在陽世虐待動物、殘害牲畜、以折磨生靈為樂者，推入牛坑之中，任由牛馬踐踏，角觝蹄踩，肝腦塗地，苦不堪言。
警醒：萬物有靈，善待眾生即是善待自己。
第 十一 層
石壓地獄
罪行：溺殺嬰兒、遺棄骨肉、草菅人命
凡在陽世溺死親生兒女、遺棄嬰孩使其凍餓而亡者，以巨石壓身。萬斤巨石從天而降，碾壓成餅，骨碎筋斷。
警醒：虎毒不食子。殘害骨肉，天理難容。
第 十二 層
舂臼地獄
罪行：浪費糧食、暴殄天物、不惜福德
凡在陽世暴殄天物、糟蹋糧食、奢靡浪費而不知惜福者，置於巨大石臼之中，以巨杵舂之。如同舂米一般，碎骨為粉。
警醒：一粥一飯，當思來處不易。半絲半縷，恆念物力維艱。
第 十三 層
血池地獄
罪行：不敬神佛、褻瀆天地、不孝不悌
凡在陽世不敬天地鬼神、褻瀆佛法僧三寶、不孝父母公婆者，投入血池之中。血水翻滾，臭穢難當，沉溺其中，不得超生。
警醒：敬天敬地敬鬼神，頭上三尺有神明。
第 十四 層
枉死地獄
罪行：自殺輕生、不珍惜生命、枉送性命
凡自殺而亡者，因生命為天賜不可自棄，入枉死城中，日夜受苦，永不超生，除非陽壽已盡。枉死城中鬼魂哭號不斷，終日惶恐。
警醒：身體髮膚受之父母，不敢毀傷。珍惜生命！
第 十五 層
磔刑地獄
罪行：盜墓掘墳、褻瀆先人、取死人財物
凡盜墓掘墳、偷盜陪葬品、褻瀆他人先祖墳塋者，五馬分屍，四肢分裂，痛苦萬狀。盜死人之財者，以身受裂體之報。
警醒：死者為大，入土為安。動人墳塋者，必遭此劫。
第 十六 層
火山地獄
罪行：損公肥私、偷盜搶劫、貪贓枉法
凡在陽世損公肥私、偷竊行搶、貪官污吏受賄枉法者，驅入火山之中。遍地岩漿，烈焰焚身，焦頭爛額，求生不得求死不能。
警醒：公器私用、貪贓枉法者，火山烈焰候之。
第 十七 層
石磨地獄
罪行：糟蹋五穀、欺上瞞下、以權謀私
凡在陽世作為地主官僚，欺壓百姓、強徵暴斂、糟蹋糧食者，置於石磨之中磨之。如磨豆腐一般，血肉模糊，碎骨連筋。
警醒：以權欺民者，石磨候之。民為邦本，本固邦寧。
第 十八 層
刀鋸地獄
罪行：偷斤減兩、坑蒙拐騙、以假亂真
凡在陽世偷斤減兩、缺秤少尺、以假充真、坑蒙拐騙者，以鋸活活鋸開身體。自頭頂鋸至腳底，一分為二，受盡千般苦楚。此為地獄最深層，刑期最長，痛苦最烈。
警醒：此為最深一層，一入萬劫不復。誠信為本，莫貪小利。

「地獄不是詛咒，而是警鐘。
佛陀設地獄之說，非為恐嚇，乃為度人。
知因畏果，止惡行善。
回頭是岸，放下屠刀，立地成佛。」

千古名言

佛道儒三教經典，字字珠璣 — 千年智慧，因果之鑰

「禍福無門，惟人自召。善惡之報，如影隨形。」
— 《太上感應篇》（道教）
禍與福沒有固定的門路，都是人自己招來的。善惡的報應，就像影子跟隨身體一樣，永遠不分離。
「假使百千劫，所作業不亡。因緣會遇時，果報還自受。」
— 《大寶積經》（佛教）
縱使經過百千萬劫之久遠，所造之業力不會消亡。當因緣成熟之時，果報終將由自己承受。
「天網恢恢，疏而不漏。」
— 《道德經》第七十三章（老子）
上天的法網廣大無邊，看似寬疏卻絲毫不漏。一切善惡終有報應，無人能逃脫因果法則。
「菩薩畏因，眾生畏果。」
— 佛教常言
菩薩明白因果，於起心動念處即戒慎恐懼。凡夫不畏因，待果報現前方知恐懼，已悔之莫及。
「積善之家，必有餘慶；積不善之家，必有餘殃。」
— 《易經・坤卦・文言》（儒教）
累積善行的家族，必有綿延的福報；累積惡行的家族，必有連綿的禍殃。
「萬般帶不走，唯有業隨身。」
— 佛教偈語
人死之後，萬般財富名利皆帶不走，唯有所造之善惡業力跟隨神識流轉，決定來世去向。
「天道無親，常與善人。」
— 《道德經》第七十九章（老子）
天道不偏私任何人，但永遠眷顧行善之人。此為因果法則最精煉之概括。
「地獄不空，誓不成佛；眾生度盡，方證菩提。」
— 地藏王菩薩大願

因果導航

如何在因果中導航你的人生 — 五個修行之道

壹
覺察輸入 — 三思而後行
在每個行動之前，問自己三個問題：這個選擇會傷害誰？這個言語會帶來什麼後果？這個念頭是善是惡？覺察是改變因果的第一步。不覺察的人生，如同閉眼駕車。
貳
口業清淨 — 善護口舌
十八層地獄中，拔舌地獄排在第一。口業最重，一句惡語傷人，遠甚刀劍。不妄語、不兩舌、不惡口、不綺語。讚美代替批評，鼓勵代替嘲諷。管住嘴巴，便化解了大半惡因。
參
廣種善因 — 日行一善
不需驚天動地的大善，日常小善累積即成大福。讓座、微笑、幫忙拾物、說句安慰的話。如同存款，一分一毫皆有利息。「積善之家，必有餘慶」，善因的複利效應，遠超你的想像。
肆
耐心等待 — 因果不虛但有時差
因果不是即時的回饋系統，而是長期的編譯器。行善不見善報，非無報也，時候未到。行惡暫時得利，非無報也，惡果正在累積。耐心等待，堅守善道，時間會給你最公正的答案。
伍
孝親敬老 — 百善之先
百善孝為先。父母是世間最大的福田，孝順父母所積之福德，勝過萬千功德。不孝之人，縱有天大本事，亦難逃因果。「樹欲靜而風不止，子欲養而親不待」，行孝要及時，莫留遺憾。

深度閱讀，用程式思維看懂天道 — 工程師的因果覺醒

精選推薦 作者：工程師小明（化名） 2026.02.20

那個改變我人生的 Bug

三年前，我是個典型的矽谷工程師——高薪、高壓、高傲。我以為程式碼就是世界運作的邏輯，只要夠聰明，就能控制一切。

直到有一天，我寫了一個致命的 bug。那不是技術上的錯誤。是對人的錯誤。

我為了升遷，在 code review 時公開羞辱了一位資深工程師，把他三個月的專案批得一文不值。當時我覺得自己很厲害，展現了「技術權威」。

三個月後，我被裁員了。

那位被我羞辱的工程師，後來成了新公司的 CTO。而我投遞履歷時，發現我們的圈子很小，小到每個面試官都認識他。

不需要報復。系統會自動執行。

為什麼工程師最該懂因果

寫程式的人，其實比誰都更懂因果。

def 因果律(輸入):
    處理 = 系統運作(輸入)
    輸出 = 產生結果(處理)
    return 輸出

# 沒有無緣無故的輸出
# 也沒有不產生影響的輸入

每一行程式碼，都是一個因，都會產生一個果。你寫下 console.log('恨')，不會輸出 '愛'。你種下惡意的種子，不會收穫善意的果實。

這不是道德審判，這是語法。

四個維度的真相

物理層：牛頓第三定律——你推牆，牆推你。你傷害他人，傷害的模式會刻入你的神經迴路。這不是報應，是物理。

資訊層：你的大腦也是一條區塊鏈。每個念頭都是一個區塊，每個行為都是一筆交易。你以為你「忘記」的，只是被標記為「隱藏」。

系統層：善意創造信任，信任降低交易成本；惡意創造防禦，防禦增加摩擦成本。系統會記住你的行為模式。

時間層：不是每個 bug 都會立刻崩潰。有些技術債潛伏數月，然後在某個深夜讓你驚醒。因果也是一樣。

程式設計師的修行

被裁員後的那一年，我讀了《道德經》、《金剛經》、量子力學、複雜系統理論。我發現一個驚人的事實：

老子在兩千年前，就寫下了宇宙運作的底層代碼。
「天網恢恢，疏而不失。」
這不是迷信，這是系統架構。

天地不判斷，只執行。而這，就是最大的慈悲。因為這意味著：沒有偏見、沒有隨機、沒有僥倖。只有法則。

你是你自己的程式設計師

你的每一個選擇，都是一個輸入。你的每一個行為，都會被系統記錄。沒有人在審判你，只有系統在執行你的選擇。

這聽起來很可怕嗎？不，這是自由。因為你可以選擇輸入什麼，你可以設計自己的反饋迴路，你可以成為系統中最有價值的節點。

種什麼因，得什麼果。不是威脅，是導航。
