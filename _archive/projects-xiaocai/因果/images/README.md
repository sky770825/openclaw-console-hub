# 圖片目錄

**因果真相旗艦專案** — 本目錄位於專案根目錄下 `images/`。

請將**自己製圖**或**參考圖片製圖**產出的檔案，依專案根目錄 `IMAGE-PROMPTS.md` 的「路徑」欄位放入對應資料夾。

- **自己製圖**：用該文件裡的「意境指令／Prompt」貼到 AI 繪圖工具（DALL·E、Midjourney、Stable Diffusion 等）生成後，存到下表路徑。
- **參考圖片製圖**：同一文件開頭有「參考圖片製圖」說明與**神明／地獄參考圖直連**，可下載或貼 URL 當風格／內容參考再生成，存到下表路徑。

| 資料夾／檔案 | 說明 |
|--------------|------|
| `hero-hall.jpg` | 首頁 Hero（可選） |
| `hell/` | 地獄頁 18 層（tongue-removal, finger-cutting, … oil-cauldron, saw-hell 等），見 IMAGE-PROMPTS 表二 |
| `realms/hero.jpg` | 六道輪迴主視覺 |
| `wall/`, `guide/`, `cases/`, `audio/`, `merit/`, `quiz/`, `simulator/`, `faq/`, `whitepaper/` | 各頁 `hero.jpg` 主視覺 |
| `deities/` | 神明圖（jade-emperor 等），可自製或沿用參考圖風格 |

完整 prompt、參考圖連結與工具用法見專案根目錄 **IMAGE-PROMPTS.md**。
