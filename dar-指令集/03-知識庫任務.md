任務：補充 {名稱} 知識庫到 ≥5KB

照 SOP-1 執行：
1. `ls -la knowledge/{名稱}/` 確認現狀
2. 建立任務板任務（帶 projectPath: projects/openclaw/modules/knowledge/）
3. `POST /api/tasks/:id/run` 建立 Run
4. 用 Cursor Agent Mode 研究 + 撰寫
5. 品質要求：≥5KB、≥5 表格、≥2 真實 URL、不可編造
6. 寫入 RESULT.md
7. 回報：`PATCH /api/tasks/:id/progress { status: 'review' }`
8. 回報時附上：檔案路徑 + `wc -c` 大小

模板在 `knowledge/TEMPLATE.md`，照著填。
