# OpenClaw Installation Guide

## Prerequisites
- Node.js 18 or higher
- npm
- Git
- Docker (Optional, for local Postgres)

## Installation Steps
1. Place `install-openclaw.sh` in your target installation directory.
2. Run the script:
   ```bash
   bash install-openclaw.sh
   
3. Follow the prompts for database configuration.

## Starting the System
Once installed, use the generated start script:
./start-dev.sh
```
- Server: http://localhost:3011
- Frontend: http://localhost:5173
