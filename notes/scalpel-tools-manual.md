# Scalpel (手術刀) 工具使用手冊

這套工具由主人於 2026-03-02 整合，旨在提供精準的代碼編輯與系統操作能力。

## 核心武器
1. patch_file: 精準修補代碼行，避免整檔覆寫風險。
2. grep_project: 全域搜尋關鍵字，快速定位邏輯。
3. find_symbol: 尋找函數、變數、類別的定義位置。
4. pty_exec: 支援互動式終端指令 (如 npm install)。

## 使用場景
- 當檔案很大（>500行）只需改 1 行時 -> 用 patch_file。
- 當不知道某個功能在哪定義時 -> 用 find_symbol。
- 當需要安裝套件或執行 git 指令時 -> 用 pty_exec。
