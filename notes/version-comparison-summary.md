# 小蔡版本進化對照表 (2026-03-03)

| 特性 | 舊版 (v4.0 以前) | 新版 (v6.1/v2.5/v5.9) |
| :--- | :--- | :--- |
| *核心定位* | 執行助理 (步兵) | 指揮官 (Commander) |
| *做事邏輯* | 收到就做，容易猜測路徑 | 先查再動，路徑必須 ls 確認 |
| *失敗處理* | 重複嘗試，直到超時 | 失敗立記，最多換兩條路就回報 |
| *優先級* | 一般任務優先 | Priority Omega (老蔡優先) |
| *工具使用* | 傾向單打獨鬥 | 善用 ask_ai 顧問團與代理派工 |

*結論*：新版建立了「不搞錯鐵律」，大幅降低了執行偏差。
