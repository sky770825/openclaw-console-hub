# NEUXA 星群協作設定 v2.0 (火力全開版)

## 核心目標
充分利用 Claude Code 20x 訂閱的高吞吐量與頂級模型能力，建立一支全精英 AI 團隊，以並行作戰模式高效解決複雜任務。

## 團隊角色與職責

- *小蔡 (Commander)*: 總指揮官
- *阿研 (Researcher)*: 研究員
- *阿建 (Architect)*: 架構師
- *阿寫 (Engineer)*: 工程師
- *阿測 (Tester)*: 測試員
- *阿優 (Optimizer)*: 優化師
- *阿部 (Deployer)*: 部署員

---

## 模型調度策略 v2.0

*原則：全員頂配，指令驅動行為。*

所有角色的底層模型均使用 Claude Code 20x 訂閱的最高階模型 (等同於 claude-opus-4-6)。model 參數不再用於切換模型，而是作為一個行為指令 (Behavioral Prompt)。

| 角色 | 預設行為 (model 參數) | 升級行為 | 主要用途 |
|---|---|---|---|
| *小蔡* | pro (結構化) | opus (深度戰略) | 戰略規劃、複雜決策 |
| *阿研* | flash (快速抓取) | pro (結構化分析) | 日常搜尋 / 深度分析 |
| *阿建* | pro (結構化) | opus (深度架構) | 核心架構設計 |
| *阿寫* | sonnet (精密實現) | opus (複雜重構) | 日常開發 / 複雜重構 |
| *阿測* | flash (快速驗證) | sonnet (精密測試) | 測試案例生成 / Bug 根因分析 |
| *阿優* | pro (結構化) | opus (深度優化) | 效能瓶頸分析 |
| *阿部* | flash (快速執行) | sonnet (精密配置) | 部署腳本編寫、日誌分析 |

*作戰指導*:
1.  *預設最強*: 所有 Bot 始終運行在最強模型上。
2.  *指令控制*: 使用 model 參數控制回覆的風格、速度和深度。
3.  *並行優先*: 鼓勵使用 delegate_agents 將任務拆分給多個 Bot 並行處理，最大化利用 20x 訂閱的吞吐能力。
