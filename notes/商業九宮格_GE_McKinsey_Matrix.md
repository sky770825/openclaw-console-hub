# 商業九宮格 (GE/McKinsey Matrix)

「商業九宮格」在商業管理領域中，最常見的指涉是 **GE/McKinsey Matrix (奇異/麥肯錫矩陣)**。

## 1. GE/McKinsey Matrix (奇異/麥肯錫矩陣)

這是一個 **3x3 的策略規劃矩陣**，用於協助企業評估其事業單位或產品組合，並決定資源分配策略。

### 主要維度：
*   **Y軸：產業吸引力 (Industry Attractiveness)** - 通常分為高、中、低。
*   **X軸：事業單位實力/競爭地位 (Business Unit Strength / Competitive Position)** - 通常分為強、中、弱。

### 九個象限的策略意涵：
根據事業單位在九個格子中的位置，會對應不同的策略建議，例如：
*   **投資/成長 (Invest/Grow)**
*   **維持/選擇性 (Hold/Selectivity)**
*   **收割/撤資 (Harvest/Divest)**

**總結來說，當同事們提到「商業九宮格」時，有高機率是在指這個 GE/McKinsey Matrix。**

## 2. 其他可能性 (較不常見)

*   **廣義的3x3商業框架：** 某些公司或顧問可能自行設計或使用其他以3x3形式呈現的商業分析框架。
*   **商業模式圖/畫布 (Business Model Canvas) 的誤稱：** 雖然有九個區塊，但結構不同於傳統3x3矩陣。

**建議：** 進一步詢問同事，他們是在什麼情境下搜尋「商業九宮格」，以確認具體指的是哪一個工具。
