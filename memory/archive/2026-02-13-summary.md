## 💡 值得深入的概念

- **成本層級**：Context 載入（20%）→ 詳細指令（17%）→ 實際執行（59%）
- **省 token 的關鍵**：信任 + 不讀回傳，單個任務省 $0.005
- **批量與細度的平衡**：詳細指令 = 更高成功率但更多 token，簡單指令 = 風險但省成本
- **Telegram API 細節**：`--data-urlencode` 自動處理，實際換行比 %0A 更可靠
