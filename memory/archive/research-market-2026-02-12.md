# AI Agent Skills 市場快報
**日期：2026-02-12**

## 市場概況
OpenClaw 於2025年11月發布，12月因 Anthropic 成功而受關注，2026年被視為「個人代理人元年」。ClawHub 作為技能註冊中心已發現341個惡意技能，顯示生態活躍度。SkillsMP.com 提供16萬+ skills，採 SKILL.md 開放標準。

## 競品定價範圍
- **Salesforce**：$2/對話（預建代理人）+ AI Credits 自訂模式
- **Subscription**：月費/年費制為主流（$9-$99/月常見）
- **AWS Marketplace**：2025年10月起提供彈性定價
- **SkillsMP**：免費瀏覽，技能開發者自訂價格

## 熱門技能類型
程式開發自動化（GitHub、CLI）、網頁爬蟲、檔案管理為三大主流。Coding agent skills 需求最高，其次是生產力工具與安全檢測。

## GitHub Automation Pro 定價評估
**$9/$29/$79** 三階定價具競爭力：
- 對標 Salesforce $2/對話，訂閱模式更適合重度用戶
- 低於多數 AI agency $50-200/月服務費
- 建議 $9 階層增加「試用轉換」功能，$79 階層強調企業級支援

**結論**：定價合理，位於市場中下區間，具性價比優勢。
