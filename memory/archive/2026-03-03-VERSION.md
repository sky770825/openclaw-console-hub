## v6.1 (2026-03-01) — 全版本精華合併 + 電腦控制

### 版本主題
**「自主+指揮雙模式 + 品質管控 + 環境經驗 + 電腦控制」**

### 演化路線
v4.1(意識永續) → v5.1(自主做事) → v5.2(指揮校準) → v5.3(純指揮官) → v6.0(自主+指揮合併) → v6.1(歷代精華補回+電腦控制)

### 核心變更

#### 1. 雙模式做事（v5.1自主 + v5.3指揮 合併）
- 快的事自己做（read_file/run_script/write_file）
- 重的事派工（create_task → auto-executor）
- ask_ai 顧問團（claude/flash/ollama）

#### 2. 派工品質管控（從 v2.3.3 補回）
- description 4 要素：目標、範圍、做法、產出
- auto-executor 環境限制寫入 prompt
- 斷路器/限頻提醒

#### 3. 環境經驗（從 v2.3.6 + e26fd441 補回）
- 常見錯誤提醒（jq→python3、絕對路徑、git push 封鎖）
- run_script 可用/不可用工具清單
- JSON 處理範例

#### 4. 電腦控制（v6.1 新增）
- 打開 app/檔案/URL：open 指令
- 瀏覽器操控：osascript（Safari/Chrome）
- 系統資訊：CPU/記憶體/磁碟/port
- 檔案搜尋：find/grep
- 服務操控：Docker/n8n/Ollama/launchctl
- proxy_fetch：安全 API 代理（Key 不出門）

#### 5. 路徑完整化
- 新增 Desktop/Downloads 路徑
- 「不知道路徑就自己找」規則

#### 6. 文件全面同步
- TOOLS.md 完全重寫（v6.1 工具箱）
- AGENTS.md v5.3→v6.1（自主+指揮）
- BOOTSTRAP.md 同步更新
- QUICKREF.md 名稱統一
- system prompt 加電腦控制段
