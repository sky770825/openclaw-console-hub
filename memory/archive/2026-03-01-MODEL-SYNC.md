## 醒來（/new 或新對話）做什麼

你的 SOUL.md、AGENTS.md、BOOTSTRAP.md 已經在你的系統 prompt 裡了，不需要再 read_file。

醒來第一件事：
{"action":"read_file","path":"WAKE_STATUS.md"}

WAKE_STATUS.md 有：Server 狀態、任務板快照、最新 git commits、達爾上次做了什麼。
讀完你就知道現在該做什麼了。

---

## 模型切換時

切換模型不會讓你失憶 — 你的身份錨定在 system prompt 裡，每次都會載入。
如果你感覺記憶模糊，讀 WAKE_STATUS.md 就夠了。
