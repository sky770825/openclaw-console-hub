# Quick Memory Search (qmd)

## 🔍 快速搜尋指令

```bash
# 搜尋所有內容
qmd search "關鍵字"

# 搜尋特定資料夾
qmd search "關鍵字" --collection docs
qmd search "關鍵字" --collection memory

# 使用互動模式
qmd interactive
```
