# [Ollama] 日誌輪轉與清理

**任務類型**: 開發  
**狀態**: 完成  
**日期**: 2026-02-13

## 規格說明

### 目標
自動清理舊 session 日誌，釋放磁碟空間，提升系統性能。

### 日誌輪轉策略

#### 1. 輪轉規則
```bash
# 位置: ~/.openclaw/sessions/
# 命名: {sessionId}-{date}.log

# 觸發條件 (滿足任一):
# - 檔案大小 > 100MB
# - 日誌年齡 > 7 天
# - 每週一 02:00 UTC
```

#### 2. 清理邏輯
```bash
# 保留策略
- 當日日誌: 完全保留
- 1-7 天: 無壓縮保留
- 7-30 天: gzip 壓縮
- > 30 天: 刪除或歸檔

# 預期釋放空間: 每週 2-5GB
```

#### 3. 監控指標
- 日誌目錄大小
- 輪轉失敗計數
- 磁碟剩餘空間告警 (< 10% 時發送 Telegram)

```bash

```

## 手動執行
```bash
./scripts/log-rotate.sh --dry-run   # 預覽
./scripts/log-rotate.sh --execute   # 執行
```
