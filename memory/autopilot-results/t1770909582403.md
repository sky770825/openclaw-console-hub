## 什麼是 AI Agent？

**AI Agent（AI 代理）** 是一種能夠「自主思考並執行任務」的人工智慧系統。與傳統的 ChatGPT 對話不同，AI Agent 可以：

- 🧠 **自主規劃** — 根據目標拆解成步驟
- 🔧 **使用工具** — 搜尋網路、查詢資料庫、發送郵件
- 🔄 **循環執行** — 觀察結果 → 調整策略 → 繼續執行
- 📊 **記憶累積** — 記住之前的對話和結果

### 簡單比喻
> 如果 ChatGPT 是「會回答問題的圖書館員」，那 AI Agent 就是「能夠幫你跑腿辦事的助理」。

## 常見問題 FAQ

### Q1: 我需要會寫程式才能使用 AI Agent 嗎？
**A:** 不一定！有許多無程式碼平台如 n8n、Zapier 都可以建立 AI Agent。但學習基礎 Python 會讓你更靈活。

### Q2: AI Agent 和傳統自動化有什麼不同？
**A:** 傳統自動化是「固定流程」，AI Agent 是「智能決策」。例如：傳統 RPA 只能按預設步驟執行，AI Agent 可以根據結果動態調整策略。

### Q3: 建立自己的 AI Agent 成本高嗎？
**A:** 入門成本很低：
- 使用 OpenRouter 免費模型：$0
- 使用本地 Ollama：$0（只需電腦）
- 使用 GPT-4 API：每千次呼叫約 $0.01-0.03

### Q4: AI Agent 安全嗎？
**A:** 需要注意：
- 限制 Agent 能使用的工具權限
- 設定執行預算上限
- 人工審核重要決策
