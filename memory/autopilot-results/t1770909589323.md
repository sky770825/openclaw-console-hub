### 1. 帳戶註冊與驗證


```
✓ 美國銀行帳戶 (ACH Transfer)
✓ 歐盟: SEPA 銀行帳戶
✓ 國際: Wise (TransferWise) 支援
```

**香港/台灣替代方案**：

- **PayPal** (提現費用 2%)
- **Wise** (推薦，低費率)

**提現規則**：
- 最小額度: $10 USD
- 提現週期: 月底結算
- 費率: 3.49% + $0.50/筆
- Gumroad 傭金: 10%
