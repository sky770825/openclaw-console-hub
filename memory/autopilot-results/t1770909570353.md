# ClawFlow CLI 工具技術規格

## 核心功能
```typescript
// 專案初始化
$ clawflow init my-project
$ cd my-project

// Skill 建立
$ clawflow skill create github-automation
$ clawflow skill create email-sender

// 本地測試
$ clawflow dev
$ clawflow test

// 部署
$ clawflow deploy
$ clawflow publish
```

## 命令集合

### 初始化
```bash
clawflow init [project-name]  # 建立新專案
clawflow config               # 設定工作目錄
```

### Skill 開發
```bash
clawflow skill create <name>      # 建立新 Skill
clawflow skill generate <type>    # 生成範本代碼
clawflow skill validate           # 驗證 Skill 結構
clawflow skill test               # 執行單元測試
clawflow skill lint               # 程式碼檢查
```

### 執行與調試
```bash
clawflow dev                  # 啟動開發伺服器 (hot reload)
clawflow run <skill-name>     # 執行單一 Skill
clawflow debug <skill-name>   # 調試模式
clawflow logs [--follow]      # 查看日誌
```

### 依賴與版本
```bash
clawflow install <skill>      # 安裝 Skill 依賴
clawflow update              # 更新依賴
clawflow version             # 版本管理
```

### 構建與部署
```bash
clawflow build               # 構建生產版本
clawflow bundle              # 打包為可分發格式
clawflow deploy              # 部署到 OpenClaw
clawflow publish             # 發佈至 ClawHub
clawflow rollback            # 回滾版本
```

## 技術架構

```
my-project/
├── clawflow.json            # 專案配置
├── skills/
│   ├── github-automation/
│   │   ├── skill.ts
│   │   ├── manifest.json
│   │   └── tests/
│   └── email-sender/

```

### 配置文件
```json
{
  "name": "my-project",
  "version": "1.0.0",
  "openclaw": {
    "version": ">=2.0.0"
  },
  "skills": {

  },
  "scripts": {
    "dev": "clawflow dev",
    "test": "clawflow test",
    "build": "clawflow build"
  }
}
```

## 技術棧
- **CLI Framework**: Commander.js / Yargs
- **打包**: esbuild / Webpack
- **測試**: Jest / Vitest
- **驗證**: Zod / Joi
- **模板**: Mustache / EJS
