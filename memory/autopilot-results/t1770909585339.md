### 🎓 教學推文 (Tutorial)
```
用 ClawFlow 5 分鐘建立自動化工作流程：

1️⃣ 安裝: npm install -g clawflow
2️⃣ 初始化: clawflow init my-project
3️⃣ 加入 Skill: clawflow add github email
4️⃣ 編輯 workflow.yaml 定義任務
5️⃣ 執行: clawflow run

就是這麼簡單。

完整文件: docs.clawflow.dev
```
