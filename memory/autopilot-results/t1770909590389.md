### 2. CI/CD Pipeline 建立
**GitHub Actions 工作流程**：

a) **Test & Lint** (PR 觸發)
```yaml
name: Test & Lint
on: [pull_request, push]
jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: '20'
          cache: 'npm'
      - run: npm ci && npm run lint && npm test
```

b) **Release** (Tag 觸發)
```yaml
name: Release
on:
  push:
    tags: ['v*']
jobs:
  publish:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
      - run: npm ci && npm publish --access public
```

c) **CodeQL** (安全掃描)
- 週一次自動掃描
- PR 合併時額外掃描
