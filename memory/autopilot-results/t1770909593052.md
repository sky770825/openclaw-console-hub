## 🚀 快速上手相關

### Q1: ClawFlow 對新手友善嗎？
**A:** 非常友善！我們設計的核心原則就是「5 分鐘快速上手」。
- 無需 JavaScript/TypeScript 知識，視覺化編輯器支援
- 提供 20+ 快速範例 Skill
- 社群 Discord 有 24/7 支援

### Q2: 我需要安裝 Docker 嗎？
**A:** 不需要。ClawFlow 在本地直接執行，無需容器化。
- 最低需求: Node.js 18+
- 可選: Ollama (本地 LLM)
- 完全獨立運行，無外部依賴

### Q3: 沒有 AI/LLM 背景可以用 ClawFlow 嗎？
**A:** 完全可以！ClawFlow 抽象化了複雜概念。
- 無需理解 Token、Prompt Engineering 等概念
- 預設模型選擇已優化，開箱即用
- 複雜任務需要時，文件提供進階說明

## 🔒 隱私與安全相關

### Q4: 我的資料會被上傳到雲端嗎？
**A:** 絕對不會。ClawFlow 本地優先架構確保：
- 所有資料永遠留在您的機器
- 無雲端帳戶需求 (除非您選擇)
- 完全離線運行可能

### Q5: ClawFlow 如何保護我的 API Key？
**A:** 多層安全機制：
- Key 儲存於本地 `.env` 檔案，從不上傳
- 支援環境變數隔離
- 可配置 Key 加密存儲
- 建議使用 1Password/Bitwarden 管理敏感資訊

### Q6: 可以在公司內網部署嗎？
**A:** 可以。ClawFlow 支援：
- 私有部署 (VPS/Docker)
- 內網隔離環境
- Proxy 設定支援
- 企業版提供審計日誌

## 💻 技術相關

### Q7: ClawFlow 支援哪些程式語言？
**A:** 核心支援：
- **JavaScript/TypeScript** (Skill 開發)
- **Python** (LangChain/Agent 整合)
- **Bash/Shell** (工具呼叫)

### Q8: 我可以使用自己的 LLM 模型嗎？
**A:** 完全可以。支援：
- **本地**: Ollama (Llama、Mistral、DeepSeek...)
- **API**: OpenAI、Anthropic、Google、OpenRouter
- **自訓練模型**: 透過 OpenAI 兼容接口
- **多模型混合**: 同時使用多個模型

| 部署 | 本地優先 | 雲端導向 | 雲端導向 | 可自裝 |
| 價格 | 完全免費 | 監控付費 | 免費+付費 | 免費+付費 |

## 📦 Skill 相關

### Q10: 如何找到我需要的 Skill？
**A:** 多個來源：
- **ClawHub**: https://clawhub.com (官方市場)
- **GitHub**: 搜尋 `clawflow-skill`
- **社群**: Discord #skill-showcase

### Q11: 能用付費的 Skill 嗎？
**A:** 可以。購買選項：
- ClawHub 市場內建支付
- Gumroad 直接購買
- License Key 啟用付費 Skill
- 企業版支援 Team License

### Q12: 如何貢獻我自己的 Skill？
**A:** 簡單三步：
1. 建立 Skill: `clawflow skill create my-skill`
2. 打包: `clawflow skill package`
3. 發布: 提交至 ClawHub 或自行發行

## 📚 整合相關

### Q16: ClawFlow 能整合 Notion/Slack/GitHub 嗎？
**A:** 都可以。預裝 Skill：
- **GitHub**: Pull Request 審查、Issue 管理
- **Notion**: 頁面 CRUD、資料庫查詢
- **Slack**: 訊息、Thread、表情回應
- **Email**: SMTP/IMAP 完整支援
- **Discord**: 訊息、頻道管理
- **Google Sheets**: 讀寫、圖表

### Q17: 能自訂整合嗎？
**A:** 完全可以。建立客製 Skill：
```javascript
// 3 行程式碼建立新整合
const mySkill = new Skill({
  async run(input) {
    return await this.http.post('https://api.example.com', input);
  }
});
```

### Q18: ClawFlow 支援 Webhook 嗎？
**A:** 支援。接收與發送都可以：
- 接收: `POST /webhook/:skillName`
- 發送: `this.webhook.send(url, data)`
- 自動驗證: HMAC-SHA256 簽名
- 重試機制: 指數退避
