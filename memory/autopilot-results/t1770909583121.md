## 前置需求

- Node.js 18+ 已安裝
- 終端機/命令列基本操作
- (選用) Ollama 已安裝，用於本地 LLM

---

## Step 1: 安裝 ClawFlow (1 分鐘)

```bash
# 全域安裝 ClawFlow CLI
npm install -g clawflow

# 驗證安裝
clawflow --version
# 預期輸出: clawflow/1.x.x
```

---

## Step 2: 初始化專案 (1 分鐘)

```bash
# 建立新專案目錄
mkdir my-ai-assistant
cd my-ai-assistant

# 初始化 ClawFlow
clawflow init
```

初始化過程會詢問：
```
? 專案名稱: my-ai-assistant
? 描述: 我的第一個 AI 助手
? 使用本地 Ollama? (Y/n): y
? 預設模型: llama3.2:latest
```

完成後會產生：
```
my-ai-assistant/
├── clawflow.config.yaml    # 主要設定
├── skills/                 # Skill 目錄
│   └── hello-world/        # 範例 Skill
└── .env                    # 環境變數
```

## Step 3: 建立第一個 Skill (2 分鐘)

```bash
# 建立新 Skill
clawflow skill create daily-summary

# 進入 Skill 目錄
cd skills/daily-summary
```

編輯 `skill.yaml`：
```yaml
name: daily-summary
description: 生成每日工作摘要
version: 1.0.0
author: your-name

inputs:
  - name: date
    type: string
    description: 日期 (YYYY-MM-DD)
    required: false

outputs:
  - name: summary
    type: string
    description: 工作摘要
```

編輯 `index.js`：
```javascript
const { Skill } = require('clawflow-sdk');

module.exports = new Skill({
  async run({ date }) {
    const targetDate = date || new Date().toISOString().split('T')[0];
    
    // 取得今日 GitHub 提交
    const commits = await this.tools.github.getCommits({ since: targetDate });
    
    // 使用 LLM 生成摘要
    const summary = await this.llm.chat([
      { role: 'system', content: '你是一位專業的技術助理，請用繁體中文簡潔摘要今日工作內容。' },
      { role: 'user', content: `今日提交: ${JSON.stringify(commits)}` }
    ]);
    
    return { summary };
  }
});
```

## Step 4: 測試執行 (1 分鐘)

```bash
# 回到專案根目錄
cd ../..

# 執行 Skill
clawflow run daily-summary
```

## 下一步：擴展功能

### 加入定時執行
編輯 `clawflow.config.yaml`：
```yaml
schedules:
  - name: daily-report
    skill: daily-summary
    cron: "0 18 * * 1-5"  # 週一至週五 18:00
    notify:
      slack: "#daily-updates"
```

### 加入更多資料來源
```javascript
// 整合 Notion、Slack、Jira...
const notionTasks = await this.tools.notion.getTasks({ filter: 'today' });
const slackMessages = await this.tools.slack.getMentions();
const jiraIssues = await this.tools.jira.getAssigned();
```

### 部署到伺服器
```bash
# 打包專案
clawflow build

# 部署到 Railway/VPS
clawflow deploy --target railway
```
