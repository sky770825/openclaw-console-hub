## 核心 API
```typescript
// 建立訂閱
POST /api/subscriptions
{
  "plan": "pro",
  "billing": "monthly",
  "payment_method": "stripe"
}

// Webhook 處理
POST /webhooks/stripe
- invoice.paid → 開立發票
- subscription.cancelled → 降級帳號
```
