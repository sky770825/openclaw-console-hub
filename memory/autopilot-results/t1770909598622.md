## 規格說明

### 目標
整合所有 .env 檔案到統一位置，簡化配置與安全管理。

### 統一結構
```
~/.openclaw/config/
├── .env.master              # 主配置檔
├── .env.secrets             # 敏感資訊 (API Keys)
├── .env.development         # 開發環境
├── .env.production          # 生產環境
├── .env.local               # 本機覆蓋 (gitignore)
└── loader.js                # 環境變數載入器
```

### 核心機制

#### 1. 優先級系統
```
.env.local > .env.{NODE_ENV} > .env.secrets > .env.master > 系統環境變數
```

#### 2. 安全管理
- 敏感值加密存儲 (AES-256)
- 存取控制 (chmod 600)
- 審計日誌 (誰、何時、改了什麼)

#### 3. 自動驗證
```bash
# 啟動時檢查必要變數
node -e "require('./config/validator.js').validate()"
```
