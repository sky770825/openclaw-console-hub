## Step 1: 安裝 (1 分鐘)

### 對於 macOS/Linux 用戶

```bash
# 安裝 Node.js (如果尚未安裝)
# https://nodejs.org (建議 18.0+)

# 安裝 ClawFlow
npm install -g clawflow

# 驗證安裝
clawflow --version
# 輸出: clawflow/1.0.0
```

### 對於 Windows 用戶

```bash
# 開啟 PowerShell (系統管理員模式)
npm install -g clawflow

# 驗證
clawflow --version
```

### 可選: 安裝 Ollama (本地 LLM)

```bash
# 訪問 https://ollama.ai
# 下載並安裝

# 驗證
ollama --version

# 拉取一個模型 (自動執行一次)
ollama pull llama3.2:latest
```

## Step 2: 建立專案 (1.5 分鐘)

```bash
# 建立專案目錄
mkdir my-first-agent
cd my-first-agent

# 初始化
clawflow init

# 選擇選項
# 使用本地 Ollama? [Y/n] → y
# 預設模型? → llama3.2:latest
```

**專案結構:**
```
my-first-agent/
├── clawflow.config.yaml   # 設定檔
├── skills/                # Skill 目錄
│   └── hello-world/       # 範例 Skill
│       ├── skill.yaml
│       └── index.js
└── .env                   # 環境變數
```

---

## Step 3: 執行範例 Skill (1.5 分鐘)

```bash
# 執行預設的 hello-world Skill
clawflow run hello-world

# 預期輸出:
# 🚀 執行 Skill: hello-world
# 🤖 Agent 回應: Hello! I'm ClawFlow, your AI assistant.
# ✅ 執行成功 (耗時: 1.2s)
```

**看看 Skill 如何運作:**

```bash
# 打開 Skill 檔案
cat skills/hello-world/index.js
```

你會看到：
```javascript
const { Skill } = require('clawflow-sdk');

module.exports = new Skill({
  async run(input) {
    return "Hello! I'm ClawFlow, your AI assistant.";
  }
});
```

## Step 4: 修改並執行 (1 分鐘)

編輯 `skills/hello-world/index.js`：

```javascript
const { Skill } = require('clawflow-sdk');

module.exports = new Skill({
  async run(input) {
    const currentTime = new Date().toLocaleString('zh-TW');
    
    // 使用 LLM 生成回應
    const response = await this.llm.chat([
      { role: 'system', content: '你是一位友善的 AI 助手。用繁體中文回應。' },
      { role: 'user', content: `現在時間是 ${currentTime}，用一句話問候我` }
    ]);
    
    return response;
  }
});
```

執行：
```bash
clawflow run hello-world

# 預期輸出:
# 🚀 執行 Skill: hello-world
# 🤖 Agent 回應: 你好呀！現在是下午 3 點 45 分，祝你有個愉快的一天！
# ✅ 執行成功 (耗時: 2.1s)
```

## 核心概念 (理解)

### Skill（技能）
一個獨立的、可重複使用的工作單位。

```
Skill = 函式 + 工具 + LLM
```

例如：
- `github-pr-reviewer`: 審查 GitHub PR
- `slack-summarizer`: 摘要 Slack 訊息
- `notion-sync`: 同步 Notion 資料

### Agent（代理）
執行 Skill 的智能角色。

```
Agent = Skill 執行器 + 決策引擎
```

一個 Agent 可以：
- 執行多個 Skill
- 根據結果做決策
- 與其他 Agent 協作

### 工作流程 (Workflow)
多個 Skill 的有序組合。

```
Workflow = Skill1 → Skill2 → Skill3
```

例如：
1. 抓取 GitHub 數據
2. 分析程式碼變更
3. 生成週報
4. 發送到 Slack

## 常用指令速查

```bash
# 列出所有 Skill
clawflow list

# 執行特定 Skill
clawflow run <skill-name>

# 使用參數執行
clawflow run <skill-name> --query "搜尋內容"

# 查看 Skill 詳情
clawflow info <skill-name>

# 安裝 Skill 市場的 Skill
clawflow skill install github-reviewer

# 建立新 Skill
clawflow skill create my-awesome-skill

# 啟動定時任務
clawflow schedule add daily-report "0 18 * * *" daily-report

# 查看執行日誌
clawflow logs --tail 50

# 停止執行中的任務
clawflow stop <task-id>
```

## 遇到問題？

| 問題 | 解決方案 |
|------|---------|
| `command not found: clawflow` | 重新安裝: `npm install -g clawflow` |
| `Error: Cannot connect to Ollama` | 檢查 Ollama 是否執行: `ollama serve` |
| `Permission denied` | 檢查檔案權限或用 `sudo` |
| 找不到 Skill | 先執行 `clawflow list` 列出所有 Skill |
