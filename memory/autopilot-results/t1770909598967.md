## 規格說明

### 目標
支援增量備份與雲端同步選項，提升備份效率與安全性。

### 核心功能

#### 1. 增量備份 (Incremental Backup)
```bash
# 方案: rsync + 時間戳追蹤
rsync -av --delete --link-dest=../backup-$(date -d yesterday +%Y%m%d) \
  ~/.openclaw/workspace/ /backup/openclaw/backup-$(date +%Y%m%d)/

# 優點: 只備份變更檔案，節省空間 60-80%
# 保留最近 7 天快照，自動清理老備份
```

#### 2. 雲端同步選項
- **AWS S3**: `aws s3 sync` (按需轉移重要檔案)
- **Google Drive**: `rclone mount` (自動同步)
- **Supabase**: 結構化資料備份

#### 3. 備份驗證
```bash
# 計算 SHA256 checksum，驗證備份完整性
find . -type f -exec sha256sum {} \; > /backup/manifest.sha256
```
