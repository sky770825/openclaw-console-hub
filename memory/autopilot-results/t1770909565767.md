## 功能規格

### 核心功能
1. **AI 自動分類**
   - 使用本地 Ollama 或 OpenAI API
   - 支援自定義分類標籤
   - 語意理解而非關鍵字匹配
   - 多語言支援

2. **優先級標記**
   - P0 (緊急) / P1 (重要) / P2 (一般) / P3 (低)
   - 根據內容 urgency 自動判斷
   - 寄件者白名單優先
   - 關鍵字加權

3. **分類類別**
   - 工作/專案相關
   - 通知/系統郵件
   - 行銷/促銷
   - 社交/私人
   - 垃圾郵件

4. **智能摘要**
   - 自動生成郵件摘要
   - 提取 Action Items
   - 辨識截止日期
