## 文章內文

Hi HN! 👋

ClawFlow 是我過去半年打造的開源專案 — 一個為開發者設計的 AI Agent 自動化框架。

## 為什麼做這個？

我每天花大量時間在重複性工作：
- 整理 GitHub Issues/PRs
- 監控網站變化並截圖
- 定時發送報表 Email
- 在 Slack/Telegram 發送通知

現有工具要不是太複雜（n8n），就是太貴（Zapier），而且都不支援本地 AI。

## ClawFlow 是什麼？

開箱即用的自動化框架，特色：

🔧 **Skill 插件生態** — GitHub/Email/Notion/Web Scraper/即時通訊，隨插即用
🤖 **本地 AI 優先** — 原生支援 Ollama，無需雲端 API
⚡ **開發者友好** — TypeScript、CLI、視覺化編輯器
🔒 **隱私保障** — 完全本地執行，資料不外流

## 快速開始

```bash
npm install -g clawflow
clawflow init my-automation
clawflow add github email
clawflow run
```

## 技術亮點

- Playwright 瀏覽器自動化
- BullMQ 任務佇列
- 熱插拔 Skill 架構
- YAML + TypeScript 雙模式
