## 📦 核心工具

### 1. Suricata - 網路守門員
```bash
# 功能
- 即時入侵檢測 (IDS)
- 自動攻擊阻擋 (IPS)  
- 深度封包檢測
- 保護整個網路段

# 適用場景
- 家庭網路保護
- 小型辦公室
- 伺服器機房
```

### 2. Zeek - 網路分析師
```bash
# 功能
- 流量行為分析
- 異常模式檢測
- 自動取證記錄
- 支援多種協定

# 適用場景
- 進階威脅獵捕
- 事件調查分析
- 合規監控
```

### 3. Wazuh - 端點保鑣
```bash
# 功能
- 主機監控 (EDR)
- 日誌分析 (SIEM)
- 威脅檢測
- 合規管理

# 適用場景
- 多主機監控
- 企業資安
- 雲端環境
```

### 4. MISP - 情報分享網
```bash
# 功能
- 威脅情報共享
- IOC 資料庫
- 自動化交換
- 社群協防

# 適用場景
- 資安社群
- 產業資訊分享
- 跨組織協防
```

## 💼 服務模式

### 個人守護 (Personal Guardian)
- **對象**: 個人用戶、自由工作者
- **內容**: AI系統保護、個人資料防護
- **價格**: 免費基礎版 / 進階訂閱

### 企業守護 (Business Guardian)  
- **對象**: 小型企業、工作室
- **內容**: 網路監控、端點保護、事件響應
- **價格**: 月費制

### 社群守護 (Community Guardian)
- **對象**: 開源專案、社群組織
- **內容**: 威脅情報分享、協防機制
- **價格**: 免費
