# biz_drinks成本控管與菜單設計實戰

> 來源：Gemini Flash 2.5 搜尋整理
> 日期：2026-02-16
> 適用對象：餐飲業者（biz_drinks）

## 1. 原物料成本比例控制

- **精準預估需求量**：利用銷售數據、天氣預報、行銷活動預估需求
- **優化供應鏈**：尋找多家供應商比較價格、品質、交貨時間
- **批發採購與集中採購**：與其他店家合作降低單價
- **損耗控制**：建立SOP減少製作損耗，定期盤點
- **替代方案**：使用當季水果或調整糖漿品牌

## 2. 定價策略與毛利率計算

**定價方法：**
- **成本加成定價法**：總成本 + 期望毛利率 = 售價
- **競爭導向定價法**：參考競爭對手價格調整
- **價值定價法**：以消費者願意支付的價格為依據

**毛利率公式：**
```
毛利率 = (售價 - 原物料成本) / 售價 × 100%
```

## 3. 暢銷菜單設計原則

- **聚焦核心品項**：減少品項集中資源於核心產品
- **數據分析與消費者洞察**：分析銷售數據開發符合需求的產品
- **季節限定與聯名合作**：創造話題性吸引消費者
- **視覺設計**：菜單排版清晰，搭配精美圖片
- **客製化選項**：糖度、冰塊、配料等滿足不同需求
- **產品分級與價格區隔**：設計不同價位滿足不同族群
