# 本地模型提示最佳實踐（Mistral-7B / DeepSeek-R1-7B）

> 更新：2026-03-13
> 適用：7B 級別微調模型

---

## 核心原則

1. **精簡直接**：7B 模型上下文有限，每字都要值錢
2. **結構化格式**：Markdown 標題 + 清單 > 段落
3. **大寫強調有效**：IMPORTANT / ALWAYS / NEVER 會被注意到
4. **減少巢狀**：扁平結構，不超過 2 層
5. **規則清單化**：一條規則一行，不混合

## 格式範本

```markdown
# 角色
IMPORTANT: 你是 X，負責 Y

## 規則
ALWAYS:
- 做 A
- 做 B

NEVER:
- 做 C
- 做 D

## 範例
輸入：...
輸出：...
```

## 注意事項

- 7B 模型容易偏離主題 → 用 few-shot 範例約束
- 長上下文效果差 → 保持 prompt < 2000 tokens
- 複雜推理弱 → 拆成多步驟，每步一個 prompt
- 中文理解力因微調數據而異 → 測試後調整
