## 1. 表單 UX 設計原則

### 1.1 簡潔至上 — 少即是多

```
黃金法則：每多一個欄位，轉換率下降約 7%
```

| 原則 | 說明 | 範例 |
|------|------|------|
| **只問必要的** | 不需要的欄位就不要放 | 聯絡表單不需要問地址 |
| **合併欄位** | 姓 + 名 → 姓名（一個欄位） | 除非系統需要分開存 |
| **善用預設值** | 國家、城市可自動偵測 | 用 IP geolocation 預填 |
| **延遲收集** | 註冊先收 email，其他後面再問 | Progressive profiling |

### 1.2 分步表單（Progressive Disclosure）

```
何時用分步表單：
✅ 超過 6 個欄位
✅ 欄位有邏輯分組（個人資料 → 偏好 → 確認）
✅ 用戶需要上傳檔案
✅ 需要條件邏輯（選 A 才出現 B 欄位）

❌ 不要用分步：
- 只有 3-4 個欄位的簡單表單
- 每步只有 1 個欄位（太瑣碎）
```

**進度指示器設計**：

```
方式 A — 步驟條（推薦）
  ① 基本資料  ──  ② 偏好設定  ──  ③ 確認送出
      ●              ○              ○

方式 B — 百分比進度條
  ████████░░░░░░░░  50% 完成

方式 C — 文字提示
  第 2 步，共 3 步
```

### 1.3 即時驗證（Inline Validation）

```
驗證時機的選擇：

❌ 輸入每個字元都驗證（太煩躁）
❌ 只在送出時驗證（回饋太慢）
✅ onBlur 驗證（離開欄位時，最佳平衡）
✅ 已出錯的欄位用 onChange 即時修正回饋
```

**最佳驗證流程**：

```
1. 使用者輸入中 → 不顯示錯誤
2. 使用者離開欄位（blur）→ 驗證並顯示結果
3. 如果有錯，使用者回來修改 → 即時驗證（onChange），修正後立刻消除錯誤
4. 送出時 → 全表單驗證，滾動到第一個錯誤
```

### 1.4 錯誤提示位置與寫法

```
位置規則：
✅ 錯誤訊息放在欄位「正下方」
✅ 用紅色文字 + 紅色邊框
✅ 加上 ⚠ icon 輔助色盲使用者
❌ 不要用 alert() 彈窗
❌ 不要只在表單頂部集中顯示所有錯誤
```

**錯誤訊息寫法**：

| 差的寫法 | 好的寫法 |
|----------|----------|
| 「格式錯誤」 | 「請輸入有效的 Email 地址，例如：name@example.com」 |
| 「必填」 | 「請填寫您的姓名」 |
| 「密碼不符合規則」 | 「密碼至少需要 8 個字元，包含大小寫字母和數字」 |
| 「日期無效」 | 「請選擇今天之後的日期」 |

### 1.5 按鈕設計

```
✅ 主按鈕用明確動詞：「送出預約」「完成報名」「寄送訊息」
❌ 不要用模糊的「送出」「Submit」

✅ 送出中顯示 loading 狀態，防止重複點擊
✅ 成功後顯示確認訊息或跳轉
✅ 長表單提供「儲存草稿」選項
```

## 2. HTML5 原生驗證

### 2.1 常用驗證屬性

```html
<!-- 必填 -->
<input type="text" name="name" required>

<!-- Email 格式 -->
<input type="email" name="email" required>

<!-- 電話格式（台灣手機） -->
<input type="tel" name="phone" pattern="09\d{8}"
       title="請輸入 09 開頭的 10 位手機號碼">

<!-- 最小/最大長度 -->
<input type="text" name="username" minlength="3" maxlength="20" required>

<!-- 密碼強度 -->
<input type="password" name="password" required
       minlength="8"
       pattern="(?=.*\d)(?=.*[a-z])(?=.*[A-Z]).{8,}"
       title="至少 8 字元，包含大小寫字母和數字">

<!-- 數字範圍 -->
<input type="number" name="age" min="18" max="120" required>

<!-- URL -->
<input type="url" name="website" placeholder="https://example.com">

<!-- 日期範圍 -->
<input type="date" name="appointment" min="2026-01-01" max="2026-12-31">
```

```html
// 自訂驗證（搭配 novalidate 屬性使用）
const form = document.getElementById('contact-form');

// 驗證規則
const validators = {
  name: (value) => {
    if (!value.trim()) return '請填寫您的姓名';
    if (value.trim().length < 2) return '姓名至少 2 個字';
    return '';
  },
  email: (value) => {
    if (!value.trim()) return '請填寫您的 Email';
    if (!/^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(value)) return '請輸入有效的 Email 地址';
    return '';
  },
  phone: (value) => {
    if (value && !/^0[2-9]\d{7,8}$/.test(value)) return '請輸入有效的電話號碼';
    return '';
  },
  message: (value) => {
    if (!value.trim()) return '請填寫訊息內容';
    if (value.trim().length < 10) return '訊息至少 10 個字';
    return '';
  },
};

// onBlur 驗證
form.querySelectorAll('input, textarea').forEach((field) => {
  field.addEventListener('blur', () => validateField(field));
  // 已有錯誤時，onChange 即時驗證
  field.addEventListener('input', () => {
    if (field.classList.contains('error')) {
      validateField(field);
    }
  });
});

function validateField(field) {
  const validator = validators[field.name];
  if (!validator) return true;

  const error = validator(field.value);
  const errorEl = document.getElementById(`${field.name}-error`);

  if (error) {
    field.classList.add('error');
    field.classList.remove('valid');
    errorEl.textContent = error;
    return false;
  } else {
    field.classList.remove('error');
    field.classList.add('valid');
    errorEl.textContent = '';
    return true;
  }
}

// 送出驗證
form.addEventListener('submit', (e) => {
  e.preventDefault();
  let isValid = true;

  form.querySelectorAll('input, textarea').forEach((field) => {
    if (!validateField(field)) isValid = false;
  });

  if (isValid) {
    const btn = form.querySelector('button[type="submit"]');
    btn.disabled = true;
    btn.textContent = '送出中...';

    // 送出表單資料
    const formData = new FormData(form);
    fetch('/api/contact', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(Object.fromEntries(formData)),
    })
      .then((res) => res.json())
      .then((data) => {
        form.innerHTML = '<p class="success">感謝您的來信，我們會盡快回覆！</p>';
      })
      .catch((err) => {
        btn.disabled = false;
        btn.textContent = '送出訊息';
        alert('送出失敗，請稍後再試');
      });
  } else {
    // 滾動到第一個錯誤欄位
    const firstError = form.querySelector('.error');
    if (firstError) firstError.scrollIntoView({ behavior: 'smooth', block: 'center' });
  }
});
</script>
```

### 3.2 React Hook Form + Zod 完整範例

```tsx
// ContactForm.tsx
import { useForm } from 'react-hook-form';
import { zodResolver } from '@hookform/resolvers/zod';
import { z } from 'zod';
import { useState } from 'react';

// ========== Schema 定義 ==========
const contactSchema = z.object({
  name: z
    .string()
    .min(1, '請填寫您的姓名')
    .min(2, '姓名至少 2 個字')
    .max(50, '姓名不能超過 50 個字'),

  email: z
    .string()
    .min(1, '請填寫您的 Email')
    .email('請輸入有效的 Email 地址'),

  phone: z
    .string()
    .optional()
    .refine(
      (val) => !val || /^09\d{8}$/.test(val),
      '請輸入有效的手機號碼（09 開頭 10 碼）'
    ),

  subject: z.enum(['general', 'quote', 'support', 'other'], {
    errorMap: () => ({ message: '請選擇主題' }),
  }),

  budget: z
    .string()
    .optional(),

  message: z
    .string()
    .min(1, '請填寫訊息內容')
    .min(10, '訊息至少 10 個字')
    .max(2000, '訊息不能超過 2000 個字'),

  privacy: z
    .boolean()
    .refine((val) => val === true, '請同意隱私政策'),
});

// 從 schema 推導 TypeScript 型別
type ContactFormData = z.infer<typeof contactSchema>;

// ========== 表單元件 ==========
export default function ContactForm() {
  const [submitStatus, setSubmitStatus] = useState<'idle' | 'loading' | 'success' | 'error'>('idle');

  const {
    register,
    handleSubmit,
    formState: { errors, isSubmitting, touchedFields },
    watch,
    reset,
  } = useForm<ContactFormData>({
    resolver: zodResolver(contactSchema),
    mode: 'onBlur',        // blur 時驗證
    reValidateMode: 'onChange', // 已有錯誤時 onChange 重新驗證
    defaultValues: {
      name: '',
      email: '',
      phone: '',
      subject: undefined,
      budget: '',
      message: '',
      privacy: false,
    },
  });

  const selectedSubject = watch('subject');

  const onSubmit = async (data: ContactFormData) => {
    setSubmitStatus('loading');
    try {
      const response = await fetch('/api/contact', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify(data),
      });

      if (!response.ok) throw new Error('送出失敗');

      setSubmitStatus('success');
      reset();
    } catch (error) {
      setSubmitStatus('error');
    }
  };

  if (submitStatus === 'success') {
    return (
      <div className="success-message" role="alert">
        <h3>感謝您的來信！</h3>
        <p>我們會在 1-2 個工作天內回覆。</p>
        <button onClick={() => setSubmitStatus('idle')}>
          再填一份
        </button>
      </div>
    );
  }

  return (
    <form onSubmit={handleSubmit(onSubmit)} noValidate>
      {/* 姓名 */}
      <FormField label="姓名" required error={errors.name?.message}>
        <input
          {...register('name')}
          type="text"
          autoComplete="name"
          placeholder="請輸入您的姓名"
          aria-invalid={!!errors.name}
          aria-describedby={errors.name ? 'name-error' : undefined}
        />
      </FormField>

      {/* Email */}
      <FormField label="Email" required error={errors.email?.message}>
        <input
          {...register('email')}
          type="email"
          autoComplete="email"
          placeholder="name@example.com"
          aria-invalid={!!errors.email}
          aria-describedby={errors.email ? 'email-error' : undefined}
        />
      </FormField>

      {/* 電話 */}
      <FormField label="電話" error={errors.phone?.message}>
        <input
          {...register('phone')}
          type="tel"
          autoComplete="tel"
          placeholder="0912345678"
          aria-invalid={!!errors.phone}
        />
      </FormField>

      {/* 主題 */}
      <FormField label="主題" required error={errors.subject?.message}>
        <select
          {...register('subject')}
          aria-invalid={!!errors.subject}
        >
          <option value="">— 請選擇 —</option>
          <option value="general">一般諮詢</option>
          <option value="quote">索取報價</option>
          <option value="support">技術支援</option>
          <option value="other">其他</option>
        </select>
      </FormField>

      {/* 條件欄位：選「索取報價」才顯示預算 */}
      {selectedSubject === 'quote' && (
        <FormField label="預算範圍">
          <select {...register('budget')}>
            <option value="">— 請選擇 —</option>
            <option value="under-30k">3 萬以下</option>
            <option value="30k-100k">3 萬 - 10 萬</option>
            <option value="100k-300k">10 萬 - 30 萬</option>
            <option value="over-300k">30 萬以上</option>
          </select>
        </FormField>
      )}

      {/* 訊息 */}
      <FormField label="訊息" required error={errors.message?.message}>
        <textarea
          {...register('message')}
          rows={5}
          placeholder="請輸入您的訊息（至少 10 個字）"
          aria-invalid={!!errors.message}
        />
      </FormField>

      {/* 隱私同意 */}
      <div className="checkbox-group">
        <label>
          <input
            {...register('privacy')}
            type="checkbox"
          />
          <span>我已閱讀並同意<a href="/privacy" target="_blank">隱私政策</a></span>
        </label>
        {errors.privacy && (
          <span className="error-text" role="alert">{errors.privacy.message}</span>
        )}
      </div>

      {/* 送出按鈕 */}
      <button type="submit" disabled={isSubmitting}>
        {isSubmitting ? '送出中...' : '送出訊息'}
      </button>

      {submitStatus === 'error' && (
        <p className="error-text" role="alert">
          送出失敗，請稍後再試。
        </p>
      )}
    </form>
  );
}

// ========== 共用欄位包裝元件 ==========
interface FormFieldProps {
  label: string;
  required?: boolean;
  error?: string;
  children: React.ReactNode;
}

function FormField({ label, required, error, children }: FormFieldProps) {
  const id = label.toLowerCase().replace(/\s+/g, '-');
  return (
    <div className={`form-field ${error ? 'has-error' : ''}`}>
      <label htmlFor={id}>
        {label}
        {required && <span className="required" aria-hidden="true"> *</span>}
      </label>
      {children}
      {error && (
        <span className="error-text" id={`${id}-error`} role="alert">
          {error}
        </span>
      )}
    </div>
  );
}
```

### 3.4 Zod vs Yup 比較

| 面向 | Zod | Yup |
|------|-----|-----|
| **TypeScript 整合** | 原生 TS，`z.infer` 直接推型別 | 需要額外 `InferType` |
| **Bundle 大小** | ~13 KB (gzip) | ~12 KB (gzip) |
| **語法風格** | 函式鏈（更接近 TS） | 鏈式呼叫（更接近 Joi） |
| **前後端共用** | 非常適合（同一套 schema） | 可以但較少人這樣用 |
| **生態系** | 快速成長（2024 後主流） | 成熟穩定（長期使用） |
| **推薦場景** | 新專案、TypeScript 專案 | 維護既有 Yup 專案 |

**結論**：2026 年新專案推薦 **Zod**，前後端可共用同一套 schema。

## 4. 後端驗證 — Express + Zod / Joi

### 4.1 Express + Zod

```ts

// server/src/middleware/validate.ts
import { Request, Response, NextFunction } from 'express';
import { ZodSchema, ZodError } from 'zod';

/**
 * 通用 Zod 驗證 middleware
 * 用法：router.post('/contact', validate(contactSchema), controller)
 */
export function validate(schema: ZodSchema) {
  return (req: Request, res: Response, next: NextFunction) => {
    try {
      // 驗證並覆寫 req.body（自動去除多餘欄位）
      req.body = schema.parse(req.body);
      next();
    } catch (error) {
      if (error instanceof ZodError) {
        // 格式化錯誤訊息
        const formattedErrors = error.errors.map((err) => ({
          field: err.path.join('.'),
          message: err.message,
          code: err.code,
        }));

        return res.status(400).json({
          success: false,
          message: '表單驗證失敗',
          errors: formattedErrors,
        });
      }
      next(error);
    }
  };
}

/**
 * 驗證 query 參數
 */
export function validateQuery(schema: ZodSchema) {
  return (req: Request, res: Response, next: NextFunction) => {
    try {
      req.query = schema.parse(req.query) as any;
      next();
    } catch (error) {
      if (error instanceof ZodError) {
        return res.status(400).json({
          success: false,
          message: '參數驗證失敗',
          errors: error.errors.map((e) => ({
            field: e.path.join('.'),
            message: e.message,
          })),
        });
      }
      next(error);
    }
  };
}
```

```ts
import { Request, Response, NextFunction } from 'express';
import { ObjectSchema } from 'joi';

export function validateJoi(schema: ObjectSchema) {
  return (req: Request, res: Response, next: NextFunction) => {
    const { error, value } = schema.validate(req.body, {
      abortEarly: false,   // 回傳所有錯誤，不是只有第一個
      stripUnknown: true,   // 移除未定義欄位
    });

    if (error) {
      const formattedErrors = error.details.map((detail) => ({
        field: detail.path.join('.'),
        message: detail.message,
        type: detail.type,
      }));

      return res.status(400).json({
        success: false,
        message: '表單驗證失敗',
        errors: formattedErrors,
      });
    }

    req.body = value;
    next();
  };
}
```

```tsx
// components/FileUpload.tsx
import { useState, useCallback, useRef } from 'react';

interface FileUploadProps {
  accept?: string;          // e.g. 'image/*,.pdf'
  maxSizeMB?: number;       // 預設 5 MB
  maxFiles?: number;        // 預設 1
  onFilesSelected: (files: File[]) => void;
}

export function FileUpload({
  accept = 'image/*,.pdf,.doc,.docx',
  maxSizeMB = 5,
  maxFiles = 5,
  onFilesSelected,
}: FileUploadProps) {
  const [dragActive, setDragActive] = useState(false);
  const [files, setFiles] = useState<File[]>([]);
  const [errors, setErrors] = useState<string[]>([]);
  const inputRef = useRef<HTMLInputElement>(null);

  const maxSizeBytes = maxSizeMB * 1024 * 1024;

  const validateFiles = useCallback((fileList: FileList | File[]): File[] => {
    const newErrors: string[] = [];
    const validFiles: File[] = [];
    const allFiles = Array.from(fileList);

    if (files.length + allFiles.length > maxFiles) {
      newErrors.push(`最多上傳 ${maxFiles} 個檔案`);
      setErrors(newErrors);
      return [];
    }

    for (const file of allFiles) {
      // 檢查檔案大小
      if (file.size > maxSizeBytes) {
        newErrors.push(`「${file.name}」超過 ${maxSizeMB} MB 限制`);
        continue;
      }

      // 檢查檔案類型（簡單版，後端再做完整驗證）
      const extension = file.name.split('.').pop()?.toLowerCase();
      const allowedExtensions = ['jpg', 'jpeg', 'png', 'gif', 'webp', 'pdf', 'doc', 'docx'];
      if (extension && !allowedExtensions.includes(extension)) {
        newErrors.push(`「${file.name}」不支援此檔案格式`);
        continue;
      }

      validFiles.push(file);
    }

    setErrors(newErrors);
    return validFiles;
  }, [files.length, maxFiles, maxSizeBytes, maxSizeMB]);

  const handleFiles = useCallback((fileList: FileList | null) => {
    if (!fileList) return;
    const validFiles = validateFiles(fileList);
    if (validFiles.length > 0) {
      const newFiles = [...files, ...validFiles];
      setFiles(newFiles);
      onFilesSelected(newFiles);
    }
  }, [files, validateFiles, onFilesSelected]);

  const removeFile = (index: number) => {
    const newFiles = files.filter((_, i) => i !== index);
    setFiles(newFiles);
    onFilesSelected(newFiles);
  };

  const handleDrag = (e: React.DragEvent) => {
    e.preventDefault();
    e.stopPropagation();
    setDragActive(e.type === 'dragenter' || e.type === 'dragover');
  };

  const handleDrop = (e: React.DragEvent) => {
    e.preventDefault();
    e.stopPropagation();
    setDragActive(false);
    handleFiles(e.dataTransfer.files);
  };

  return (
    <div className="file-upload">
      {/* 拖拽區域 */}
      <div
        className={`drop-zone ${dragActive ? 'active' : ''}`}
        onDragEnter={handleDrag}
        onDragOver={handleDrag}
        onDragLeave={handleDrag}
        onDrop={handleDrop}
        onClick={() => inputRef.current?.click()}
        role="button"
        tabIndex={0}
        aria-label="點擊或拖拽檔案上傳"
        onKeyDown={(e) => e.key === 'Enter' && inputRef.current?.click()}
      >
        <input
          ref={inputRef}
          type="file"
          accept={accept}
          multiple={maxFiles > 1}
          onChange={(e) => handleFiles(e.target.files)}
          style={{ display: 'none' }}
        />
        <div className="drop-zone-content">
          <span className="upload-icon">&#128206;</span>
          <p>拖拽檔案到這裡，或<strong>點擊選擇檔案</strong></p>
          <p className="hint">
            支援 JPG、PNG、PDF、DOC（單檔最大 {maxSizeMB} MB，最多 {maxFiles} 個）
          </p>
        </div>
      </div>

      {/* 錯誤訊息 */}
      {errors.length > 0 && (
        <ul className="upload-errors" role="alert">
          {errors.map((err, i) => <li key={i}>{err}</li>)}
        </ul>
      )}

      {/* 已選檔案列表 */}
      {files.length > 0 && (
        <ul className="file-list">
          {files.map((file, index) => (
            <li key={index}>
              <span className="file-name">{file.name}</span>
              <span className="file-size">({(file.size / 1024 / 1024).toFixed(2)} MB)</span>
              <button type="button" onClick={() => removeFile(index)} aria-label={`移除 ${file.name}`}>
                &times;
              </button>
            </li>
          ))}
        </ul>
      )}
    </div>
  );
}
```

### 6.2 後端：Multer（本地/暫存上傳）

```ts
// server/src/middleware/upload.ts
import multer from 'multer';
import path from 'path';
import crypto from 'crypto';

// 允許的 MIME 類型
const ALLOWED_TYPES: Record<string, string[]> = {
  image: ['image/jpeg', 'image/png', 'image/gif', 'image/webp'],
  document: ['application/pdf', 'application/msword',
    'application/vnd.openxmlformats-officedocument.wordprocessingml.document'],
};

const ALL_ALLOWED = [...ALLOWED_TYPES.image, ...ALLOWED_TYPES.document];

// 儲存策略
const storage = multer.diskStorage({
  destination: (req, file, cb) => {
    cb(null, path.join(__dirname, '../../uploads/temp'));
  },
  filename: (req, file, cb) => {
    // 隨機檔名 + 原副檔名（防止路徑穿越攻擊）
    const ext = path.extname(file.originalname).toLowerCase();
    const randomName = crypto.randomBytes(16).toString('hex');
    cb(null, `${randomName}${ext}`);
  },
});

// 檔案過濾
const fileFilter = (req: any, file: Express.Multer.File, cb: multer.FileFilterCallback) => {
  if (ALL_ALLOWED.includes(file.mimetype)) {
    cb(null, true);
  } else {
    cb(new Error(`不支援的檔案類型: ${file.mimetype}`));
  }
};

// 匯出不同場景的 upload middleware
export const uploadSingle = multer({
  storage,
  fileFilter,
  limits: { fileSize: 5 * 1024 * 1024 }, // 5 MB
}).single('file');

export const uploadMultiple = multer({
  storage,
  fileFilter,
  limits: { fileSize: 5 * 1024 * 1024 },
}).array('files', 5); // 最多 5 個

// 錯誤處理 middleware
export function handleUploadError(err: any, req: any, res: any, next: any) {
  if (err instanceof multer.MulterError) {
    if (err.code === 'LIMIT_FILE_SIZE') {
      return res.status(400).json({ success: false, message: '檔案大小超過 5 MB 限制' });
    }
    if (err.code === 'LIMIT_FILE_COUNT') {
      return res.status(400).json({ success: false, message: '檔案數量超過限制' });
    }
    return res.status(400).json({ success: false, message: `上傳錯誤: ${err.message}` });
  }
  if (err?.message?.includes('不支援的檔案類型')) {
    return res.status(400).json({ success: false, message: err.message });
  }
  next(err);
}
```

```ts
router.post('/upload/presigned', async (req, res) => {

  // 驗證檔案類型
  const allowed = ['image/jpeg', 'image/png', 'image/webp', 'application/pdf'];

  // 產生唯一檔名

  const key = `uploads/${crypto.randomBytes(16).toString('hex')}.${ext}`;

  try {
    const command = new PutObjectCommand({
      Bucket: process.env.S3_BUCKET!,
      Key: key,
      ContentType: contentType,
    });

    const presignedUrl = await getSignedUrl(s3, command, { expiresIn: 300 }); // 5 分鐘有效

    res.json({
      success: true,
      presignedUrl,
      key,
      publicUrl: `https://${process.env.S3_BUCKET}.s3.amazonaws.com/${key}`,
    });
  } catch (error) {
    console.error('Presigned URL error:', error);
    res.status(500).json({ success: false, message: '產生上傳連結失敗' });
  }
});
```

```ts
// server/src/middleware/recaptcha.ts

export async function verifyRecaptcha(req: any, res: any, next: any) {
  const { recaptchaToken } = req.body;

  if (!recaptchaToken) {
    return res.status(400).json({ success: false, message: '缺少驗證 token' });
  }

  try {
    const response = await fetch('https://www.google.com/recaptcha/api/siteverify', {
      method: 'POST',
      headers: { 'Content-Type': 'application/x-www-form-urlencoded' },
      body: new URLSearchParams({
        secret: process.env.RECAPTCHA_SECRET_KEY!,
        response: recaptchaToken,
      }),
    });

    const data = await response.json();

    // v3 回傳 score 0.0 ~ 1.0（越高越可能是真人）
    if (!data.success || data.score < 0.5) {
      console.warn('reCAPTCHA 驗證失敗:', data);
      return res.status(403).json({ success: false, message: '驗證失敗，請稍後再試' });
    }

    next();
  } catch (error) {
    console.error('reCAPTCHA 驗證錯誤:', error);
    // 驗證服務異常時放行（不要因為 Google 掛了擋住使用者）
    next();
  }
}
```

```ts
export function honeypotCheck(fieldName = 'website') {
  return (req: any, res: any, next: any) => {
    if (req.body[fieldName]) {
      // Honeypot 被填了 → 是機器人
      console.warn('Honeypot triggered:', {
        ip: req.ip,
        field: fieldName,
        value: req.body[fieldName],

    delete req.body[fieldName];
    next();
  };
}
```

```ts
export async function forwardToWebhook(webhookUrl: string, payload: WebhookPayload) {
  try {
    const response = await fetch(webhookUrl, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(payload),
      signal: AbortSignal.timeout(10000), // 10 秒 timeout
    });

      return false;

    return true;
  } catch (error) {
    console.error('Webhook forward error:', error);
    return false; // 不要因為 webhook 失敗就擋住表單送出
  }
}
```

```
3. 錯誤訊息用 role="alert" 讓螢幕閱讀器即時播報
4. 表單可以純鍵盤操作（Tab/Shift+Tab/Enter/Space）
5. 焦點順序符合邏輯（visual order = DOM order）
6. 表單群組用 <fieldset> + <legend>
7. 色彩對比度至少 4.5:1
```

## 10. 多步驟表單 — Wizard Pattern

### 10.1 完整 React 實作

```tsx
// MultiStepForm.tsx
import { useState } from 'react';
import { useForm, FormProvider, useFormContext } from 'react-hook-form';
import { zodResolver } from '@hookform/resolvers/zod';
import { z } from 'zod';

// ========== 各步驟 Schema ==========
const step1Schema = z.object({
  name: z.string().min(2, '請填寫姓名'),
  email: z.string().email('請輸入有效 Email'),
  phone: z.string().regex(/^09\d{8}$/, '請輸入有效手機號碼'),
});

const step2Schema = z.object({
  service: z.enum(['web-design', 'app-dev', 'seo', 'consulting'], {
    errorMap: () => ({ message: '請選擇服務項目' }),
  }),
  budget: z.enum(['under-50k', '50k-100k', '100k-300k', 'over-300k'], {
    errorMap: () => ({ message: '請選擇預算範圍' }),
  }),
  timeline: z.enum(['asap', '1-month', '2-3-months', 'flexible'], {
    errorMap: () => ({ message: '請選擇時程' }),
  }),
  description: z.string().min(20, '請描述您的需求（至少 20 個字）'),
});

const step3Schema = z.object({
  referral: z.string().optional(),
  terms: z.literal(true, {
    errorMap: () => ({ message: '請同意服務條款' }),
  }),
});

// 合併完整 Schema
const fullSchema = step1Schema.merge(step2Schema).merge(step3Schema);
type FormData = z.infer<typeof fullSchema>;

// 步驟設定
const STEPS = [
  { title: '基本資料', schema: step1Schema },
  { title: '需求描述', schema: step2Schema },
  { title: '確認送出', schema: step3Schema },
];

// ========== 主元件 ==========
export default function MultiStepForm() {
  const [currentStep, setCurrentStep] = useState(0);
  const [isSubmitting, setIsSubmitting] = useState(false);
  const [isComplete, setIsComplete] = useState(false);

  const methods = useForm<FormData>({
    resolver: zodResolver(fullSchema),
    mode: 'onBlur',
    defaultValues: {
      name: '',
      email: '',
      phone: '',
      service: undefined,
      budget: undefined,
      timeline: undefined,
      description: '',
      referral: '',
      terms: false as any,
    },
  });

  const { trigger, handleSubmit, getValues } = methods;

  // 下一步（只驗證當前步驟的欄位）
  const goNext = async () => {
    const currentSchema = STEPS[currentStep].schema;
    const fields = Object.keys(currentSchema.shape) as (keyof FormData)[];

    // 只驗證當前步驟的欄位
    const isValid = await trigger(fields);
    if (isValid) {
      setCurrentStep((prev) => Math.min(prev + 1, STEPS.length - 1));
      window.scrollTo({ top: 0, behavior: 'smooth' });
    }
  };

  // 上一步（不驗證）
  const goBack = () => {
    setCurrentStep((prev) => Math.max(prev - 1, 0));
  };

  // 最終送出
  const onSubmit = async (data: FormData) => {
    setIsSubmitting(true);
    try {
      const response = await fetch('/api/quote-request', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify(data),
      });

      if (!response.ok) throw new Error('送出失敗');
      setIsComplete(true);
    } catch (error) {
      alert('送出失敗，請稍後再試');
    } finally {
      setIsSubmitting(false);
    }
  };

  // 完成畫面
  if (isComplete) {
    return (
      <div className="wizard-complete" role="alert">
        <h2>需求已送出！</h2>
        <p>我們會在 1-2 個工作天內與您聯繫。</p>
        <p>確認信已寄到 <strong>{getValues('email')}</strong></p>
      </div>
    );
  }

  return (
    <FormProvider {...methods}>
      <div className="multi-step-form">
        {/* 進度指示器 */}
        <StepIndicator steps={STEPS} currentStep={currentStep} />

        {/* 表單內容 */}
        <form onSubmit={handleSubmit(onSubmit)} noValidate>
          <div className="step-content">
            {currentStep === 0 && <Step1 />}
            {currentStep === 1 && <Step2 />}
            {currentStep === 2 && <Step3 />}
          </div>

          {/* 導航按鈕 */}
          <div className="step-navigation">
            {currentStep > 0 && (
              <button type="button" className="btn-back" onClick={goBack}>
                上一步
              </button>
            )}

            {currentStep < STEPS.length - 1 ? (
              <button type="button" className="btn-next" onClick={goNext}>
                下一步
              </button>
            ) : (
              <button type="submit" className="btn-submit" disabled={isSubmitting}>
                {isSubmitting ? '送出中...' : '確認送出'}
              </button>
            )}
          </div>
        </form>
      </div>
    </FormProvider>
  );
}

            index < currentStep ? 'completed' :
            index === currentStep ? 'current' : 'upcoming';

    </nav>
  );
}
```

### 10.3 多步驟表單的狀態保持（localStorage）

```tsx
// hooks/useFormPersist.ts
// 自動儲存表單進度到 localStorage，防止意外關閉遺失資料

import { useEffect, useCallback } from 'react';
import { UseFormReturn } from 'react-hook-form';

interface UseFormPersistOptions {
  storageKey: string;
  debounceMs?: number;   // 儲存間隔，預設 500ms
}

export function useFormPersist<T extends Record<string, any>>(
  form: UseFormReturn<T>,
  { storageKey, debounceMs = 500 }: UseFormPersistOptions
) {
  const { watch, reset, getValues } = form;

  // 載入儲存的資料
  useEffect(() => {
    const saved = localStorage.getItem(storageKey);
    if (saved) {
      try {
        const data = JSON.parse(saved);
        reset(data, { keepDefaultValues: true });
      } catch {
        localStorage.removeItem(storageKey);
      }
    }
  }, [storageKey, reset]);

  // 監聽變化並儲存
  useEffect(() => {
    let timeout: NodeJS.Timeout;

    const subscription = watch(() => {
      clearTimeout(timeout);
      timeout = setTimeout(() => {
        const values = getValues();
        localStorage.setItem(storageKey, JSON.stringify(values));
      }, debounceMs);
    });

    return () => {
      clearTimeout(timeout);
      subscription.unsubscribe();
    };
  }, [watch, getValues, storageKey, debounceMs]);

  // 清除儲存
  const clearSaved = useCallback(() => {
    localStorage.removeItem(storageKey);
  }, [storageKey]);

  return { clearSaved };
}

// 使用方式：
// const methods = useForm<FormData>({ ... });
// const { clearSaved } = useFormPersist(methods, { storageKey: 'quote-form' });
// 送出成功後呼叫 clearSaved()
```

```ts
// 台灣手機
z.string().regex(/^09\d{8}$/, '手機號碼格式不正確')

z.string().regex(/^0[2-8]\d{7,8}$/, '市話格式不正確')

// 台灣統一編號
z.string().regex(/^\d{8}$/, '統一編號格式不正確')

// 密碼強度
z.string()
  .min(8, '至少 8 字元')
  .regex(/[A-Z]/, '需包含大寫字母')
  .regex(/[a-z]/, '需包含小寫字母')
  .regex(/[0-9]/, '需包含數字')

// 確認密碼
z.object({
  password: z.string().min(8),
  confirmPassword: z.string(),
}).refine(d => d.password === d.confirmPassword, {
  message: '密碼不一致',
  path: ['confirmPassword'],
})

// 至少選一個
z.array(z.string()).min(1, '請至少選一項')

// 日期在今天之後
z.string().refine(v => new Date(v) > new Date(), '請選擇未來日期')

// URL（選填）
z.string().url('請輸入有效網址').or(z.literal(''))

// 條件必填
z.object({
  hasCompany: z.boolean(),
  companyName: z.string().optional(),
}).refine(
  d => !d.hasCompany || (d.companyName && d.companyName.length > 0),
  { message: '請填寫公司名稱', path: ['companyName'] }
)
```
