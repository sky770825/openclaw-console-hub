## 怎麼用

1. **不會做某件事** → 先查上面的「快速對應」表
2. 找不到？查 **28-高級智能體能力索引.md**（萬能查詢表）
3. 知道類別 → 直接開對應的 .md 檔
4. 找到你要的章節 → 複製指令，改參數就能用

**28 號是你的萬能索引。涵蓋：生產力工具、雲端平台、通訊、開發、macOS、AI 模型、數據處理。**
