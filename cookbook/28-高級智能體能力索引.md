
## 十、做不到的事怎麼辦？

**完整清單見 cookbook/29-能力邊界與替代方案.md**，這裡列快速版：

| 做不到的事 | 替代方案 |
|-----------|---------|
| 搜尋網路 | ask_ai model=claude 或告訴主人 |
| 操控瀏覽器內容 | osascript 開 URL；動態網頁→告訴主人 |
| 精確編輯大檔案 | run_script sed 替換；或 create_task |
| 跑超過 30 秒 | create_task（auto-executor 120 秒） |
| git push | commit 好告訴主人「等你 push」 |
| 安裝套件 | 先用 python3 標準庫；不行就告訴主人 |
| 讀 .env / API key | proxy_fetch 或 query_supabase 繞過 |
| 讀超大檔案 | run_script head/tail/sed -n 讀指定行 |

**不會做 ≠ 不能做。先找替代方案，真的沒轍再求援。**
