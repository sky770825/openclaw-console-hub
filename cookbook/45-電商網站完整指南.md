### 2.2 Step 1 — 環境安裝

**主機選擇**（台灣推薦）：

| 主機商 | 月費 | 特色 | 推薦程度 |

| SiteGround | ~400 TWD | WooCommerce 優化 | ★★★★ |

**安裝流程**：

```bash
# Cloudways 一鍵安裝 WordPress + WooCommerce
# 或手動安裝：

# 1. 安裝 WordPress（假設已有主機）
wp core download --locale=zh_TW
wp config create --dbname=woo_db --dbuser=root --dbpass=密碼

  --admin_user=admin --admin_password=強密碼 --admin_email=you@email.com

# 2. 安裝 WooCommerce
wp plugin install woocommerce --activate

# 3. 安裝必備外掛
wp plugin install \
  ry-woocommerce-tools \
  woocommerce-checkout-field-editor \
  yoast-seo \
  wordfence \
  wp-super-cache \
  --activate
```

### 2.3 Step 2 — WooCommerce 基礎設定

**設定清單**（WooCommerce → 設定）：

```
一般
├── 商店地址：台灣
├── 銷售地區：僅台灣（或含海外）
├── 幣別：新台幣 (TWD / NT$)
└── 稅率：含稅價（台灣習慣含稅）

商品
├── 重量單位：公斤 (kg)
├── 尺寸單位：公分 (cm)
├── 啟用評價：視需求
└── 庫存管理：啟用

運費
├── 運費區域 1：台灣本島
│   ├── 宅配（黑貓/郵局）：60-150 元
│   ├── 超商取貨：60 元
│   └── 滿額免運：滿 1,000 免運
├── 運費區域 2：離島
│   └── 宅配：150-300 元
└── 運費區域 3：海外（視需求）

帳號 & 隱私
├── 啟用訪客結帳：建議啟用（降低棄單率）
├── 結帳時建立帳號：勾選
└── 隱私權政策頁面：建立並連結
```

### 2.5 Step 4 — 金流設定（綠界 ECPay）

```
1. 註冊綠界帳號 → 開通信用卡收款
2. 取得 MerchantID + HashKey + HashIV
3. 安裝 RY WooCommerce Tools
4. WooCommerce → 設定 → 付款：
   ├── 綠界信用卡：啟用
   ├── 綠界 ATM 轉帳：啟用
   ├── 綠界超商代碼：啟用
   └── 綠界超商條碼：啟用
5. 輸入綠界 API 金鑰
6. 測試模式先測一遍 → 確認沒問題再切正式
```

**綠界設定參考**：

```php
// functions.php — 自訂綠界付款方式顯示名稱
add_filter('woocommerce_gateway_title', function($title, $id) {
    $names = [
        'ry_ecpay_credit'     => '信用卡付款',
        'ry_ecpay_atm'        => 'ATM 轉帳',
        'ry_ecpay_cvs_code'   => '超商代碼繳費',
        'ry_ecpay_cvs_barcode'=> '超商條碼繳費',
    ];
    return $names[$id] ?? $title;
}, 10, 2);
```

### 2.6 Step 5 — 物流設定（超商取貨）

```
1. 綠界後台 → 開通物流服務
2. RY WooCommerce Tools → 物流設定：
   ├── 7-11 取貨：啟用
   ├── 全家取貨：啟用
   ├── 萊爾富取貨：啟用
   └── OK 超商取貨：啟用
3. 運費區域 → 新增「超商取貨」運送方式
4. 測試選店功能
```

### 2.7 Step 6 — 電子發票設定

```
1. 綠界後台 → 電子發票服務申請
2. 安裝 RY ECPay Invoice 外掛
3. 設定：
   ├── 自動開立：訂單完成時自動開立
   ├── 載具類型：手機條碼 / 自然人憑證 / 會員載具
   ├── 捐贈碼：預設愛心碼清單
   └── 公司統編：B2B 開立
4. 結帳頁會自動出現發票選項
```

## 3. Shopify 台灣用法

### 3.1 方案與定價

| 方案 | 月費(USD) | 約(TWD) | 信用卡費率 | 適合 |
|------|-----------|---------|-----------|------|
| Basic | $39 | ~1,250 | 2.9%+30¢ | 新創/小品牌 |
| Shopify | $105 | ~3,360 | 2.6%+30¢ | 成長中品牌 |
| Advanced | $399 | ~12,770 | 2.4%+30¢ | 大型電商 |
| Plus | $2,300+ | ~73,600 | 洽談 | 企業級 |

> 注意：如果不使用 Shopify Payments（台灣目前可用），使用第三方金流還會加收 0.5-2% 手續費。

### 3.2 台灣金流解決方案

Shopify 在台灣的金流選項：

| 方案 | 支援付款方式 | 設定難度 | 手續費 |
|------|-------------|---------|--------|
| **Shopify Payments** | 信用卡/Apple Pay/Google Pay | 低（內建） | 2.4-2.9% |
| **綠界 ECPay App** | 信用卡/ATM/超商代碼 | 中 | 綠界費率 |
| **TapPay** | 信用卡/LINE Pay/JKOPay | 中 | 洽談 |
| **PayNow（永豐）** | 信用卡/超商 | 中 | 洽談 |

**Shopify + 綠界串接步驟**：

```
1. Shopify App Store → 搜尋「ECPay」或「綠界」
2. 安裝官方或第三方串接 App
3. 設定 → 付款 → 新增付款方式
4. 輸入綠界 MerchantID / HashKey / HashIV
5. 啟用需要的付款方式
6. 測試訂單確認金流正常
```

### 3.3 台灣物流解決方案

```
Shopify 原生不支援台灣超商取貨，需要透過 App：
├── 綠界物流 App → 7-11 / 全家 / 萊爾富 / OK
├── 超商取貨 App（第三方）
└── 自建串接（Shopify Functions + 綠界 API）
```

## 4. 自建電商（Next.js）

### 4.1 技術架構

```
┌─────────────────────────────────────────────┐
│                   前端 (Next.js)              │
│  App Router + Server Components + Tailwind   │
├─────────────────────────────────────────────┤
│              API Routes / tRPC               │
├──────────┬──────────┬───────────────────────┤
│ Supabase │  Stripe  │    綠界 ECPay         │
│ (DB+Auth)│ (國際金流)│  (台灣金流+物流+發票)  │
├──────────┴──────────┴───────────────────────┤
│           Object Storage (S3/R2)             │
│           Redis (購物車/Session)              │
│           n8n (自動化/通知)                   │
└─────────────────────────────────────────────┘
```

### 4.2 核心資料庫 Schema

```sql
-- ===== 商品表 =====
CREATE TABLE products (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name TEXT NOT NULL,
  slug TEXT UNIQUE NOT NULL,
  description TEXT,
  short_description TEXT,
  -- 價格
  price DECIMAL(10,2) NOT NULL,
  compare_at_price DECIMAL(10,2),        -- 原價（劃線價）
  cost_price DECIMAL(10,2),              -- 成本價
  -- 分類
  category_id UUID REFERENCES categories(id),
  brand TEXT,
  -- 庫存
  sku TEXT UNIQUE,
  stock_quantity INTEGER DEFAULT 0,
  track_inventory BOOLEAN DEFAULT true,
  allow_backorder BOOLEAN DEFAULT false,
  -- 狀態
  status TEXT DEFAULT 'draft' CHECK (status IN ('draft','active','archived')),
  is_featured BOOLEAN DEFAULT false,
  -- SEO
  meta_title TEXT,
  meta_description TEXT,
  -- 規格
  weight DECIMAL(8,2),                   -- 公斤
  dimensions JSONB,                      -- {length, width, height}
  -- 時間
  created_at TIMESTAMPTZ DEFAULT now(),
  updated_at TIMESTAMPTZ DEFAULT now(),
  published_at TIMESTAMPTZ
);

-- ===== 商品變體 =====
CREATE TABLE product_variants (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  product_id UUID REFERENCES products(id) ON DELETE CASCADE,
  name TEXT NOT NULL,                    -- 如「紅色 / L」
  sku TEXT UNIQUE NOT NULL,
  price DECIMAL(10,2) NOT NULL,
  compare_at_price DECIMAL(10,2),
  stock_quantity INTEGER DEFAULT 0,
  -- 選項值
  option_values JSONB NOT NULL,          -- {"顏色":"紅色","尺寸":"L"}
  -- 圖片
  image_url TEXT,
  -- 狀態
  is_active BOOLEAN DEFAULT true,
  sort_order INTEGER DEFAULT 0,
  created_at TIMESTAMPTZ DEFAULT now()
);

-- ===== 分類 =====
CREATE TABLE categories (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  name TEXT NOT NULL,
  slug TEXT UNIQUE NOT NULL,
  parent_id UUID REFERENCES categories(id),
  description TEXT,
  image_url TEXT,
  sort_order INTEGER DEFAULT 0,
  is_active BOOLEAN DEFAULT true,
  -- SEO
  meta_title TEXT,
  meta_description TEXT
);

-- ===== 訂單 =====
CREATE TABLE orders (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  order_number TEXT UNIQUE NOT NULL,     -- 如 ORD-20260305-0001
  user_id UUID REFERENCES users(id),
  -- 金額
  subtotal DECIMAL(10,2) NOT NULL,       -- 小計
  shipping_fee DECIMAL(10,2) DEFAULT 0,
  discount_amount DECIMAL(10,2) DEFAULT 0,
  tax_amount DECIMAL(10,2) DEFAULT 0,
  total DECIMAL(10,2) NOT NULL,
  -- 狀態
  status TEXT DEFAULT 'pending' CHECK (status IN (
    'pending',        -- 待付款
    'paid',           -- 已付款
    'processing',     -- 備貨中
    'shipped',        -- 已出貨
    'delivered',      -- 已送達
    'completed',      -- 已完成
    'cancelled',      -- 已取消
    'refunding',      -- 退款中
    'refunded'        -- 已退款
  )),
  payment_status TEXT DEFAULT 'unpaid',
  shipping_status TEXT DEFAULT 'pending',
  -- 付款資訊
  payment_method TEXT,                   -- credit_card / atm / cvs_code
  payment_gateway TEXT,                  -- ecpay / stripe / newebpay
  payment_id TEXT,                       -- 金流交易編號
  paid_at TIMESTAMPTZ,
  -- 收件資訊
  shipping_name TEXT NOT NULL,
  shipping_phone TEXT NOT NULL,
  shipping_email TEXT,
  shipping_address JSONB NOT NULL,
  shipping_method TEXT,                  -- home_delivery / cvs_711 / cvs_family
  shipping_tracking TEXT,                -- 物流追蹤號
  -- 發票
  invoice_type TEXT,                     -- personal / company / donation
  invoice_carrier TEXT,                  -- 載具號碼
  invoice_tax_id TEXT,                   -- 統編
  invoice_number TEXT,                   -- 發票號碼
  -- 備註
  customer_note TEXT,
  admin_note TEXT,
  -- 優惠
  coupon_code TEXT,
  -- 時間
  created_at TIMESTAMPTZ DEFAULT now(),
  updated_at TIMESTAMPTZ DEFAULT now(),
  shipped_at TIMESTAMPTZ,
  completed_at TIMESTAMPTZ
);

-- ===== 訂單明細 =====
CREATE TABLE order_items (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  order_id UUID REFERENCES orders(id) ON DELETE CASCADE,
  product_id UUID REFERENCES products(id),
  variant_id UUID REFERENCES product_variants(id),
  name TEXT NOT NULL,
  sku TEXT,
  price DECIMAL(10,2) NOT NULL,
  quantity INTEGER NOT NULL,
  subtotal DECIMAL(10,2) NOT NULL,
  -- 快照（下單時的商品資訊，避免商品修改後影響歷史訂單）
  product_snapshot JSONB
);

-- ===== 購物車 =====
CREATE TABLE cart_items (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES users(id),
  session_id TEXT,                       -- 訪客購物車
  product_id UUID REFERENCES products(id),
  variant_id UUID REFERENCES product_variants(id),
  quantity INTEGER NOT NULL DEFAULT 1,
  created_at TIMESTAMPTZ DEFAULT now(),
  updated_at TIMESTAMPTZ DEFAULT now(),
  UNIQUE(user_id, product_id, variant_id),
  UNIQUE(session_id, product_id, variant_id)
);

-- ===== 優惠券 =====
CREATE TABLE coupons (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  code TEXT UNIQUE NOT NULL,
  description TEXT,
  type TEXT NOT NULL CHECK (type IN (
    'percentage',      -- 百分比折扣
    'fixed_amount',    -- 固定金額折扣
    'free_shipping'    -- 免運
  )),
  value DECIMAL(10,2) NOT NULL,          -- 折扣值
  min_order_amount DECIMAL(10,2),        -- 最低訂單金額
  max_discount DECIMAL(10,2),            -- 折扣上限
  usage_limit INTEGER,                   -- 總使用次數上限
  usage_count INTEGER DEFAULT 0,
  per_user_limit INTEGER DEFAULT 1,
  starts_at TIMESTAMPTZ,
  expires_at TIMESTAMPTZ,
  is_active BOOLEAN DEFAULT true,
```

```typescript
import { cookies } from 'next/headers';
import { revalidatePath } from 'next/cache';
import { supabase } from '@/lib/supabase-server';
import { getSession } from '@/lib/auth';

export async function addToCart(productId: string, variantId: string | null, quantity: number) {
  const session = await getSession();
  const cookieStore = await cookies();
  const sessionId = session?.user?.id || cookieStore.get('cart_session')?.value;

  // 訪客購物車：建立 session ID
  if (!session?.user?.id && !sessionId) {
    const newSessionId = crypto.randomUUID();
    cookieStore.set('cart_session', newSessionId, {
      httpOnly: true,
      maxAge: 60 * 60 * 24 * 30,  // 30 天
    });
  }

  // 檢查庫存
  const { data: product } = await supabase
    .from(variantId ? 'product_variants' : 'products')
    .select('stock_quantity')
    .eq('id', variantId || productId)
    .single();

  if (!product || product.stock_quantity < quantity) {
    return { error: '庫存不足' };
  }

  // Upsert 購物車（已有則增加數量）
  const filter = session?.user?.id
    ? { user_id: session.user.id }
    : { session_id: sessionId };

  const { data: existing } = await supabase
    .from('cart_items')
    .select('id, quantity')
    .match({ ...filter, product_id: productId, variant_id: variantId })
    .single();

  if (existing) {
    await supabase
      .from('cart_items')
      .update({ quantity: existing.quantity + quantity })
      .eq('id', existing.id);
  } else {
    await supabase.from('cart_items').insert({
      ...filter,
      product_id: productId,
      variant_id: variantId,
      quantity,
    });
  }

  revalidatePath('/cart');
  return { success: true };
}

export async function removeFromCart(itemId: string) {
  await supabase.from('cart_items').delete().eq('id', itemId);
  revalidatePath('/cart');
  return { success: true };
}

export async function updateCartQuantity(itemId: string, quantity: number) {
  if (quantity <= 0) return removeFromCart(itemId);

  await supabase.from('cart_items').update({ quantity }).eq('id', itemId);
  revalidatePath('/cart');
  return { success: true };
```

```typescript
// 綠界 CheckMacValue 產生
function generateCheckMac(params: Record<string, string>): string {
  // 1. 按 key 排序
  const sorted = Object.keys(params)
    .sort()
    .map(k => `${k}=${params[k]}`)
    .join('&');

  // 2. 加上 HashKey & HashIV
  const raw = `HashKey=${ECPAY_CONFIG.HashKey}&${sorted}&HashIV=${ECPAY_CONFIG.HashIV}`;

  // 3. URL encode（小寫）
  const encoded = encodeURIComponent(raw).toLowerCase();

  // 4. SHA256
  return crypto.createHash('sha256').update(encoded).digest('hex').toUpperCase();
}
```

## 5. 商品管理

### 5.1 SKU 編碼規則

```
SKU 格式建議：[品牌]-[分類]-[款式]-[顏色]-[尺寸]

範例：
  NK-SH-AF1-WHT-42    → Nike 鞋子 Air Force 1 白色 42號
  AP-PH-IP15-BLK-256  → Apple 手機 iPhone15 黑色 256G
  HC-SK-MOIST-50ML    → 自有品牌 保養 保濕乳 50ml

規則：
  1. 全大寫英數字，用「-」分隔
  2. 每段不超過 5 字元
  3. 同系列商品前綴一致
  4. SKU 一旦上線就不要改（影響庫存追蹤）
```

### 5.2 商品變體管理

```
變體策略：
├── 單一商品（無變體）：直接在 products 表管理
├── 簡單變體（1 個選項）：如只有顏色
├── 組合變體（2+ 選項）：如顏色 x 尺寸
└── 注意：變體數 = 選項 1 數量 x 選項 2 數量（可能很多）

範例 — T-shirt：
  顏色：白/黑/灰（3）x 尺寸：S/M/L/XL（4）= 12 個變體
  每個變體有獨立 SKU、價格、庫存
```

```typescript
// 變體產生工具
function generateVariants(
  options: Record<string, string[]>,
  basePrice: number,
  baseSku: string
): ProductVariant[] {
  const keys = Object.keys(options);

  function combine(index: number, current: Record<string, string>): Record<string, string>[] {
    if (index >= keys.length) return [current];
    const key = keys[index];
    return options[key].flatMap(value =>
      combine(index + 1, { ...current, [key]: value })
    );
  }

  return combine(0, {}).map((optionValues, i) => ({
    name: Object.values(optionValues).join(' / '),
    sku: `${baseSku}-${Object.values(optionValues).map(v => v.substring(0, 3).toUpperCase()).join('-')}`,
    price: basePrice,
    stock_quantity: 0,
    option_values: optionValues,
    sort_order: i,
  }));
}

// 使用範例

  590,
  'TS-BASIC'
);
```

```typescript
// supabase/migrations/xxx_deduct_stock.sql
CREATE OR REPLACE FUNCTION deduct_stock(
  p_variant_id UUID,
  p_quantity INTEGER
) RETURNS BOOLEAN AS $$
DECLARE
  current_stock INTEGER;
BEGIN
  SELECT stock_quantity INTO current_stock
  FROM product_variants
  WHERE id = p_variant_id
  FOR UPDATE;  -- 行鎖，防止超賣

  IF current_stock < p_quantity THEN
    RETURN FALSE;
  END IF;

  UPDATE product_variants
  SET stock_quantity = stock_quantity - p_quantity
  WHERE id = p_variant_id;

  RETURN TRUE;
END;
$$ LANGUAGE plpgsql;
```

```typescript
export async function issueInvoice(order: {
  orderId: string;
  buyerEmail: string;
  items: Array<{
    name: string;
    quantity: number;
    unitPrice: number;
  }>;
  invoiceType: 'personal' | 'company' | 'donation';
  carrierType?: string;           // 1=會員 2=自然人 3=手機條碼
  carrierNum?: string;            // 載具號碼
  taxId?: string;                 // 統編
  companyName?: string;           // 公司抬頭
  donationCode?: string;          // 愛心碼
}) {
  const totalAmount = order.items.reduce(
    (sum, item) => sum + item.unitPrice * item.quantity, 0
  );

  // 商品名稱/數量/單價用 | 分隔
  const itemNames = order.items.map(i => i.name).join('|');
  const itemCounts = order.items.map(i => String(i.quantity)).join('|');
  const itemPrices = order.items.map(i => String(i.unitPrice)).join('|');

  const params: Record<string, string> = {
    MerchantID: INVOICE_CONFIG.MerchantID,
    RelateNumber: order.orderId,
    CustomerEmail: order.buyerEmail,
    // 稅率
    TaxType: '1',                  // 1=應稅 2=零稅率 3=免稅
    SalesAmount: String(totalAmount),
    InvType: order.invoiceType === 'company' ? '07' : '07',  // 07=一般 08=特種
    // 商品
    ItemName: itemNames,
    ItemCount: itemCounts,
    ItemWord: order.items.map(() => '個').join('|'),
    ItemPrice: itemPrices,
    ItemAmount: order.items.map(i => String(i.unitPrice * i.quantity)).join('|'),
    // 買方
    Print: '0',                    // 0=不印 1=印
    Donation: order.invoiceType === 'donation' ? '1' : '0',
  };

  // 依發票類型設定載具
  if (order.invoiceType === 'personal' && order.carrierType) {
    params.CarrierType = order.carrierType;
    params.CarrierNum = order.carrierNum || '';
  } else if (order.invoiceType === 'company') {
    params.CustomerIdentifier = order.taxId || '';
    params.CustomerName = order.companyName || '';
    params.Print = '1';
  } else if (order.invoiceType === 'donation') {
    params.LoveCode = order.donationCode || '168001';
  }

  // AES 加密
  const encrypted = aesEncrypt(JSON.stringify(params));

  const response = await fetch(INVOICE_CONFIG.ApiUrl, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      MerchantID: INVOICE_CONFIG.MerchantID,
      RqHeader: { Timestamp: Math.floor(Date.now() / 1000) },
      Data: encrypted,
    }),
  });

  const result = await response.json();
  const decrypted = JSON.parse(aesDecrypt(result.Data));

  return {
    invoiceNumber: decrypted.InvoiceNo,
    invoiceDate: decrypted.InvoiceDate,
    randomNumber: decrypted.RandomNumber,
  };
}

// AES 加密（綠界電子發票用 AES-128-CBC）
function aesEncrypt(data: string): string {
  const cipher = crypto.createCipheriv(
    'aes-128-cbc',
    INVOICE_CONFIG.HashKey,
    INVOICE_CONFIG.HashIV,
  );
  let encrypted = cipher.update(encodeURIComponent(data), 'utf8', 'base64');
  encrypted += cipher.final('base64');
  return encrypted;
}

function aesDecrypt(data: string): string {
  const decipher = crypto.createDecipheriv(
    'aes-128-cbc',
    INVOICE_CONFIG.HashKey,
    INVOICE_CONFIG.HashIV,
  );
  let decrypted = decipher.update(data, 'base64', 'utf8');
  decrypted += decipher.final('utf8');
  return decodeURIComponent(decrypted);
}
```

### 10.1 訂單狀態機

```

│ shipped      │ delivered, returned                   │
```

```typescript
// lib/order-state-machine.ts

type OrderStatus =
  | 'pending' | 'paid' | 'processing' | 'shipped'
  | 'delivered' | 'completed' | 'cancelled'
  | 'refunding' | 'refunded'
  | 'return_requested' | 'returned';

const TRANSITIONS: Record<OrderStatus, OrderStatus[]> = {
  pending:          ['paid', 'cancelled'],
  paid:             ['processing', 'refunding', 'cancelled'],
  processing:       ['shipped', 'refunding'],
  shipped:          ['delivered', 'returned'],
  delivered:        ['completed', 'return_requested'],
  completed:        ['return_requested'],
  cancelled:        [],
  refunding:        ['refunded'],
  refunded:         [],
  return_requested: ['returned', 'completed'],
  returned:         ['refunding'],
};

export function canTransition(from: OrderStatus, to: OrderStatus): boolean {
  return TRANSITIONS[from]?.includes(to) ?? false;
}

export async function transitionOrder(
  orderId: string,
  newStatus: OrderStatus,
  note?: string
) {
  // 1. 取得目前訂單
  const { data: order } = await supabase
    .from('orders')
    .select('status')
    .eq('id', orderId)
    .single();

  if (!order) throw new Error('訂單不存在');

  // 2. 驗證轉換合法性
  if (!canTransition(order.status as OrderStatus, newStatus)) {
    throw new Error(
      `無法從 ${order.status} 轉為 ${newStatus}`
    );
  }

  // 3. 更新狀態
  const updates: Record<string, any> = {
    status: newStatus,
    updated_at: new Date().toISOString(),
  };

  // 特定狀態的額外欄位
  if (newStatus === 'paid') updates.paid_at = new Date().toISOString();
  if (newStatus === 'shipped') updates.shipped_at = new Date().toISOString();
  if (newStatus === 'completed') updates.completed_at = new Date().toISOString();

  await supabase.from('orders').update(updates).eq('id', orderId);

  // 4. 記錄狀態變更歷史
  await supabase.from('order_status_logs').insert({
    order_id: orderId,
    from_status: order.status,
    to_status: newStatus,
    note: note || '',
  });

  // 5. 觸發後續動作
  await handleStatusChange(orderId, order.status as OrderStatus, newStatus);
}

async function handleStatusChange(
  orderId: string,
  from: OrderStatus,
  to: OrderStatus
) {
  switch (to) {
    case 'paid':
      // 扣庫存 + 通知商家備貨
      await deductInventory(orderId);
      await notifyMerchant(orderId, '有新訂單待備貨');
      break;
    case 'shipped':
      // 通知買家已出貨
      await notifyBuyer(orderId, '您的訂單已出貨');
      break;
    case 'delivered':
      // 通知買家已送達
      await notifyBuyer(orderId, '您的訂單已送達，請確認商品');
      break;
    case 'cancelled':
      // 如果已付款要退款 + 還原庫存
      if (from === 'paid' || from === 'processing') {
        await restoreInventory(orderId);
        await initiateRefund(orderId);
      }
      break;
    case 'refunded':
      // 還原庫存
      await restoreInventory(orderId);
      await notifyBuyer(orderId, '退款已完成');
      break;
  }
}
```

### 11.1 退換貨政策（台灣法規）

```
消保法規定（網路購物）：

│   ├── 報紙、期刊、雜誌
│   ├── 數位內容（已提供線上下載）
│   └── 國際航空、訂房等已排定日期的服務
└── 瑕疵品：不受 7 天限制，可要求換貨或退款

建議退貨政策：
├── 7 天內無條件退貨（含運費）
├── 7-14 天退貨（運費買家自付）
├── 瑕疵品 30 天內免費退換
└── 退款方式：原付款方式退回（信用卡退刷 / ATM 匯款）
```

```
退貨申請流程：
  1. 買家在訂單頁按「申請退貨」
  2. 選擇退貨原因 + 上傳照片（瑕疵品）
  3. 商家審核（1-2 工作天）
  4. 核准 → 提供退貨物流單號（或超商退貨碼）
  5. 買家寄回商品
  6. 商家收到 → 檢查商品
  7. 確認 OK → 發起退款
  8. 退款完成 → 通知買家

換貨流程：
  1-6 同上
  7. 確認 OK → 寄出新商品
  8. 買家收到新商品 → 完成
```

```typescript
// lib/coupon.ts

export async function applyCoupon(
  couponCode: string,
  cartItems: CartItem[],
  userId?: string
): Promise<CouponResult> {
  // 1. 查詢優惠券
  const { data: coupon } = await supabase
    .from('coupons')
    .select('*')
    .eq('code', couponCode.toUpperCase())
    .eq('is_active', true)
    .single();

  if (!coupon) return { valid: false, error: '優惠券不存在或已失效' };

  // 2. 時間驗證
  const now = new Date();
  if (coupon.starts_at && now < new Date(coupon.starts_at))
    return { valid: false, error: '優惠券尚未生效' };
  if (coupon.expires_at && now > new Date(coupon.expires_at))
    return { valid: false, error: '優惠券已過期' };

  // 3. 使用次數驗證
  if (coupon.usage_limit && coupon.usage_count >= coupon.usage_limit)
    return { valid: false, error: '優惠券已達使用上限' };

  // 4. 個人使用次數
  if (userId && coupon.per_user_limit) {
    const { count } = await supabase
      .from('coupon_usages')
      .select('*', { count: 'exact' })
      .eq('coupon_id', coupon.id)
      .eq('user_id', userId);
    if ((count || 0) >= coupon.per_user_limit)
      return { valid: false, error: '您已使用過此優惠券' };
  }

  // 5. 最低金額門檻
  const subtotal = cartItems.reduce((sum, i) => sum + i.price * i.quantity, 0);
  if (coupon.min_order_amount && subtotal < coupon.min_order_amount)
    return {
      valid: false,
      error: `需滿 NT$${coupon.min_order_amount} 才能使用此優惠券`,
    };

  // 6. 計算折扣
  let discount = 0;
  switch (coupon.type) {
    case 'percentage':
      discount = Math.round(subtotal * (coupon.value / 100));
      if (coupon.max_discount) discount = Math.min(discount, coupon.max_discount);
      break;
    case 'fixed_amount':
      discount = coupon.value;
      break;
    case 'free_shipping':
      discount = 0; // 運費另計
      break;
  }

  return {
    valid: true,
    couponId: coupon.id,
    type: coupon.type,
    discount,
    freeShipping: coupon.type === 'free_shipping',
    description: coupon.description,
  };
}
```

```typescript
// 風控檢查（下單前）
export async function fraudCheck(order: OrderInput): Promise<FraudResult> {
  const checks: FraudCheck[] = [];

  // 1. IP 檢查 — 同一 IP 短時間大量下單
  const recentOrders = await supabase
    .from('orders')
    .select('id', { count: 'exact' })
    .eq('ip_address', order.ipAddress)
    .gte('created_at', new Date(Date.now() - 60 * 60 * 1000).toISOString());

  if ((recentOrders.count || 0) > 5) {
    checks.push({ rule: 'ip_velocity', risk: 'high', detail: '同 IP 1 小時內超過 5 筆訂單' });
  }

  // 2. Email 檢查 — 免費信箱 + 隨機字串
  const emailDomain = order.email?.split('@')[1];
  const suspiciousPattern = /^[a-z0-9]{10,}@(gmail|yahoo|hotmail)/;
  if (order.email && suspiciousPattern.test(order.email)) {
    checks.push({ rule: 'suspicious_email', risk: 'medium', detail: '疑似自動產生的 Email' });
  }

  // 3. 金額檢查 — 異常高額訂單
  if (order.total > 50000) {
    checks.push({ rule: 'high_amount', risk: 'medium', detail: '訂單金額超過 50,000' });
  }

  // 4. 地址檢查 — 收件地址和帳單地址不同
  if (order.shippingAddress !== order.billingAddress) {
    checks.push({ rule: 'address_mismatch', risk: 'low', detail: '收件地址與帳單地址不同' });
  }

  // 5. 首次購買 + 高額
  if (!order.userId && order.total > 10000) {
    checks.push({ rule: 'guest_high_amount', risk: 'medium', detail: '訪客高額訂單' });
  }

  const riskScore = checks.reduce((sum, c) => {
    return sum + (c.risk === 'high' ? 30 : c.risk === 'medium' ? 15 : 5);
  }, 0);

  return {
    approved: riskScore < 50,
    riskScore,
    checks,
    requireManualReview: riskScore >= 30 && riskScore < 50,
  };
}
```
