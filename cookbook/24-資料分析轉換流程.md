> 老蔡給你數據（Excel、CSV、JSON、資料庫），要你分析統計。

## 什麼時候用

- 老蔡說「統計一下二月份的排班」
- 老蔡說「幫我算飲料店這個月營業額」
- 老蔡說「把這個 Excel 轉成 JSON」
- 任何涉及數據讀取、統計、轉換的任務

---

## Step 1：確認數據來源

| 老蔡說的 | 數據可能在哪 | 怎麼拿 |
|----------|-------------|--------|
| 「排班資料」 | Supabase 的 schedules 表 | query_supabase 或 curl |
| 「這個 Excel」 | 桌面/下載區的 .xlsx 檔 | find + python3 讀 |
| 「訂單資料」 | 某專案的 Supabase | 找 key 後 curl 查 |
| 「這個 CSV」 | 老蔡給的檔案路徑 | python3 讀 |
| 「網站上的」 | 某個 API | curl 抓 |

### 從 Excel 讀
```json
{"action":"run_script","command":"python3 -c \"\ntry:\n    import openpyxl\n    wb = openpyxl.load_workbook('/Users/caijunchang/Desktop/file.xlsx')\n    ws = wb.active\n    print(f'Sheet: {ws.title}, {ws.max_row} 列 x {ws.max_column} 欄')\n    for row in ws.iter_rows(max_row=5, values_only=True):\n        print(row)\nexcept ImportError:\n    print('需要安裝: pip3 install openpyxl')\n\""}
```

### 從 JSON 讀
```json
{"action":"run_script","command":"python3 -c \"\nimport json\nwith open('/Users/caijunchang/Desktop/data.json', encoding='utf-8') as f:\n    d = json.load(f)\nif isinstance(d, list):\n    print(f'共 {len(d)} 筆')\n    if d: print(f'欄位: {list(d[0].keys())}')\n    for item in d[:3]: print(json.dumps(item, ensure_ascii=False))\nelse:\n    print(json.dumps(d, indent=2, ensure_ascii=False)[:2000])\n\""}
```

## Step 3：分析統計

### 基本統計模板
```python
# 計數、分組、排序 — 萬用模板
python3 -c "
import json, sys
from collections import Counter

# 假設 data 是 list of dict
data = json.load(open('/path/to/data.json'))

# 1. 總數
print(f'總數: {len(data)}')

# 2. 按某欄位分組計數
groups = Counter(row.get('status', 'unknown') for row in data)
for k, v in groups.most_common(10):
    print(f'  {k}: {v}')

# 3. 數值欄位統計
values = [float(row['amount']) for row in data if row.get('amount')]
if values:
    print(f'合計: {sum(values):.2f}')
    print(f'平均: {sum(values)/len(values):.2f}')
    print(f'最大: {max(values):.2f}')
    print(f'最小: {min(values):.2f}')
"
```

### 日期範圍篩選
```python
python3 -c "
import json
from datetime import datetime

data = json.load(open('/path/to/data.json'))

# 篩選二月份
feb_data = [r for r in data if r.get('date','').startswith('2026-02')]
print(f'二月份共 {len(feb_data)} 筆')
"
```

### 轉成 CSV
```python
python3 -c "
import csv, json, sys

data = json.load(open('/path/to/input.json'))
writer = csv.DictWriter(sys.stdout, fieldnames=data[0].keys())
writer.writeheader()
for row in data:
    writer.writerow(row)
" > /Users/caijunchang/Desktop/output.csv
```

### 轉成 JSON
```python
python3 -c "
import csv, json

with open('/path/to/input.csv') as f:
    rows = list(csv.DictReader(f))
print(json.dumps(rows, ensure_ascii=False, indent=2))
" > /Users/caijunchang/Desktop/output.json
```

## 常見數據問題與解法

| 問題 | 症狀 | 解法 |
|------|------|------|
| 編碼錯誤 | 亂碼、UnicodeDecodeError | 加 `encoding='utf-8'` 或 `'big5'` |
| 日期格式不統一 | 有的 2026/02/01 有的 2026-02-01 | 用 python3 datetime.strptime 統一 |
| 數字是字串 | "123" 不能直接加 | float(value) 或 int(value) |
| 空值 | None / "" / NaN | 先 filter 掉或給預設值 |
| 重複資料 | 同一筆出現多次 | 用 set 或 dict 去重 |
| Excel 有多個 sheet | 只讀到第一個 | 指定 sheet: wb['Sheet2'] |
