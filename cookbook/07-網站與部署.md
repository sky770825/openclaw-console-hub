## 一、建置指令

### 前端（React + Vite）
```bash
cd /Users/caijunchang/openclaw任務面版設計

# 開發模式（localhost:5173，自動 proxy API 到 :3011）
npm run dev

# 正式建置
npm run build        # 產出 dist/

# 型別檢查
npx tsc --noEmit

# Lint
npm run lint
```

### 後端（Express + TypeScript）
```bash
cd /Users/caijunchang/openclaw任務面版設計/server

# 開發模式（自動重載）
npm run dev

# 正式建置
npm run build        # 產出 server/dist/

# 啟動正式版
node dist/index.js   # 在 port 3011 提供 API + 靜態前端
```

### 一鍵正式啟動
```bash
cd /Users/caijunchang/openclaw任務面版設計/server
nohup node dist/index.js > /tmp/openclaw-server.log 2>&1 &
```

## 三、Port 管理

| Port | 服務 | 用途 |
|------|------|------|
| 3011 | Express Server | API + 靜態前端 |
| 5173 | Vite Dev Server | 開發模式前端 |
| 5678 | n8n | 工作流自動化 |
| 11434 | Ollama | 本地 AI 模型 |

### 查看 port 使用
```bash
# 查特定 port
lsof -i :3011 -sTCP:LISTEN

# 查所有 openclaw 相關 port
lsof -i :3011 -i :5173 -i :5678 -i :11434
```

### 釋放 port
```bash
# 單一 port
kill $(lsof -i :3011 -sTCP:LISTEN -t)

# 全部
bash /Users/caijunchang/openclaw任務面版設計/scripts/free-ports.sh
```

## 四、版本管理

### 目前版本
```bash
grep '"version"' /Users/caijunchang/openclaw任務面版設計/package.json
grep '"version"' /Users/caijunchang/openclaw任務面版設計/server/package.json
```

### 版本號位置（更新版本要同時改這三個地方）
1. `package.json` → `"version"`
2. `server/package.json` → `"version"`
3. `server/src/index.ts` → 版本字串

### 自動遞增版本
```bash
bash /Users/caijunchang/openclaw任務面版設計/scripts/auto-bump-version.sh
```

### 版本規則
- patch +1 = 每天 00:01 自動遞增
- minor +1 = 大功能上線
- major +1 = 重大架構變更
