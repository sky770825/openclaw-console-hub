## 五、資料型別

### Task（最常用）
```typescript
interface Task {
  id: string;
  name: string;
  description: string;
  status: 'draft' | 'ready' | 'running' | 'review' | 'done' | 'blocked';
  priority: 1 | 2 | 3 | 4 | 5;     // 1 最高
  owner: string;
  tags: string[];

  // 執行
  agent?: { type: 'cursor' | 'codex' | 'openclaw' | 'auto' };
  lastRunStatus?: 'queued' | 'running' | 'success' | 'failed' | 'timeout';
  result?: string;

  // 安全
  riskLevel?: 'low' | 'medium' | 'high' | 'critical';
  rollbackPlan?: string;
  acceptanceCriteria?: string[];
}
```

### MDCI 文明指數（6 軸）
```typescript
interface MDCIAxis {
  communication: number;  // 通信
  culture: number;        // 文化
  energy: number;         // 能源
  technology: number;     // 科技
  life: number;           // 生命
  defense: number;        // 防禦
}
// 每軸 0-100，總分 = 平均
```

## 六、WebSocket 事件

### 前端監聽的事件
| 事件 | 說明 |
|------|------|
| `task:created` | 新任務 |
| `task:updated` | 任務更新 |
| `task:deleted` | 任務刪除 |
| `run:started` | 開始執行 |
| `run:completed` | 執行完成 |
| `alert:new` | 新警報 |
| `federation:attack` | 聯盟攻擊事件 |
| `auto-executor:dispatch` | 任務派發 |

### 連線方式
```
ws://localhost:3011（自動升級 HTTP → WS）
```
