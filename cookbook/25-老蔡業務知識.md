## 老蔡的三大事業

### 1. 住商不動產 — 桃園楊梅區
- **類型**：房屋仲介（買賣 + 租賃）
- **工作流**：開發物件 → 帶看 → 簽到記錄 → 追蹤客戶 → 成交 → 售後
- **關鍵指標**：成交件數、帶看轉化率、簽到數、客戶生命週期
- **相關系統**：project_j 客製網站、華地產簽到系統、591 監控
- **常見需求**：「統計本月成交」「分析哪些物件帶看最多」「簽到報表」

### 2. 飲料店
- **類型**：餐車/攤位（楊梅地區）
- **工作流**：排班 → 備料 → 營業 → 收銀 → 對帳
- **關鍵指標**：日營業額、時段客流、品項銷售排名、人力成本
- **相關系統**：楊梅餐車排班報名系統（Supabase）、Food Car Class
- **常見需求**：「二月排班統計」「算這個月人力成本」「哪個時段最賺」

### 3. 普特斯防霾紗窗
- **類型**：建材代理/安裝
- **工作流**：客戶詢價 → 丈量 → 報價 → 安裝 → 驗收
- **常見需求**：報價單生成、安裝排程

## 幫老蔡做商業分析時的思路

1. **先問清楚範圍**：哪個月？哪個店？哪個業務？
2. **找到數據來源**：Supabase？Excel？手動記錄？
3. **做基本統計**：總數、平均、趨勢、對比
4. **找異常值**：特別高或特別低的數據點
5. **給可行建議**：不只是數字，要有「所以呢？」的結論
6. **報告寫到 workspace/reports/**：方便以後回查
