## 各版本詳細評估

### v2.5.0 — crew-doctor 星群醫生
- **改動**：自動偵測 bot 卡死/故障，產出健康報告
- **效果**：穩定運作，能偵測異常
- **結論**：保留

### v2.5.1 — create_task 防重複修復
- **改動**：API 回傳格式 {tasks:[]} 而非陣列
- **效果**：修復了 create_task 錯誤判斷重複的 bug
- **結論**：保留（bug fix）

### v2.5.2 — crew-inbox 協作引擎（297 行新檔案）
- **改動**：crew-inbox.ts 新增，30 秒掃描 inbox、類型/優先級解析、crewThink 處理、自動歸檔
- **效果**：星群閉環協作核心，bot 之間可以互派任務
- **結論**：保留（核心功能）

### v2.5.3 — 星群協作強化
- **改動**：
  1. 巡邏從 3 人擴到 6 人（加入阿策/阿工/阿商）
  2. Flash maxOutputTokens 1024→2048
  3. Claude CLI 改 stdin pipe（修復 arg 長度截斷）
  4. 新增 crew_dispatch action
  5. VALID_BOT_IDS 過濾 inbox 孤兒目錄
- **效果**：
  - crew_dispatch 有效（小蔡成功派 5 次任務）
  - Claude stdin 修復有效（阿工 score 3→7）
  - 但 6 人同時出動打爆 Gemini Pro API（大面積超時→fallback Flash→品質下降）
- **結論**：
  - 保留：crew_dispatch、Claude stdin、inbox 過濾、maxOutputTokens 提升
  - 已修正：6 人同時出動問題（v2.5.6 分批解決）

### v2.5.4 — 小蔡自修星群解鎖
- **改動**：
  1. security.ts + isPatchPathSafe 放行 crew-bots/ 子目錄
  2. patch_file prompt 格式修正（search/replace → old/new）
  3. Supabase 真實欄位清單 + 不存在欄位黑名單
  4. write_file path 格式提示強化
- **效果**：
  - patch_file 格式修正直接解決了 3 次失敗
  - Supabase 欄位教學減少幻覺查詢
  - crew-bots/ 寫入放行讓小蔡能自己優化星群
- **結論**：保留（全部正向改動）

### v2.5.5 — patrol 幻覺欄位修正
- **改動**：
  1. patrol prompt 裡 name→title，移除 priority/owner
  2. security.ts 空 if 改寫
  3. crew-poller let→const
- **效果**：修復巡邏查詢一直查空結果的問題
- **結論**：保留（bug fix）

### v2.5.6 — Pro timeout + 巡邏分批
- **改動**：
  1. Gemini Flash 30s / Pro 60s 分開 timeout
  2. 巡邏每批最多 3 人，批間隔 5 秒
- **效果**：待觀察（剛上線）
- **結論**：保留

## 最優配置基準線（v2.5.6 當前）

### 星群
- **巡邏**：6 人分 2 批出動（3+3），批間隔 5 秒
- **Gemini timeout**：Flash 30s / Pro 60s
- **maxOutputTokens**：Flash 2048（summarizeReply）/ 4096（callGeminiAPI）
- **Claude CLI**：stdin pipe 傳 prompt，120s timeout，全局冷卻 15s
- **最優分數基準**：阿秘 9、阿研 9、阿數 8.9（v2.5.2 三人巡邏時期）
- **目前分數**：阿秘 9、阿工 7、阿策 6、阿數 4（需持續觀察分批後是否回升）

### 小蔡
- **主力模型**：Gemini 3 Pro（64%），Opus（29%），Sonnet（7%）
- **Action 成功率**：77.7%（80/103）
- **平均回覆**：439 字
- **最優表現時段**：用 Gemini 3 Pro 連續做事，action 成功率高
- **已知問題**：升級鏈過度觸發（72 次）、Claude 短回覆（9 次 <50 字）

### 安全
- **小蔡可改**：crew-bots/ 子目錄、workspace、前端、scripts
- **小蔡不可改**：server/src（crew-bots 除外）、靈魂文件、.env、package.json
