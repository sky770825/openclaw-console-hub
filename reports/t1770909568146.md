# ClawFlow 插件熱更新技術規格

## 功能概述
Skill 動態載入、熱插拔，無需重啟應用。

## 核心功能

### 動態載入

```typescript
class SkillManager {
  // 載入 Skill
  async load(skillPath: string): Promise<Skill>
  
  // 卸載 Skill
  async unload(skillName: string): Promise<void>
  
  // 重新載入
  async reload(skillName: string): Promise<void>
  
  // 列出已載入
  list(): Skill[]
}
```

### 熱更新流程

```
1. 偵測檔案變更 (watchFiles)
2. 驗證新 Skill (validate)
3. 載入到記憶體 (loadInMemory)
4. 執行遷移 (runMigrations)
5. 切換路由 (switchRouter)
6. 保留舊版本 (rollback)
```

### 版本管理

```typescript
interface SkillVersion {
  name: string;
  version: string;
  path: string;
  status: 'active' | 'previous' | 'disabled';
  loadedAt: Date;
}

// 同時支援多版本
skillManager.versions('github-skill')
  // [
  //   { version: '2.0.0', status: 'active' },
  //   { version: '1.9.0', status: 'previous' }
  // ]
```

### 狀態保持

```typescript
// 熱更新時保留記憶體狀態
const state = currentSkill.getState();
newSkill.setState(state);
```
