# 智慧表單系統 - P1 前端基礎建設報告

## 完成項目
1. **框架搭建**: 採用 Vite + Vue 3 + TypeScript。
2. **UI 框架**: 整合 Tailwind CSS，提供原子化 CSS 支援與快速樣式開發。
3. **路由結構**: 
   - 設定 Vue Router。
   - 實作主佈局 (MainLayout) 嵌套路由。
4. **RWD 佈局**:
   - 實作響應式側邊欄 (Sidebar)。
   - 行動端自動隱藏選單並提供漢堡選單 (Hamburger Menu)。
5. **基礎頁面**:
   - `HomeView`: 儀表板數據概覽。
   - `FormsView`: 表單清單管理介面。

## 目錄結構
- `src/layouts/`: 存放通用佈局。
- `src/views/`: 存放頁面組件。
- `src/router/`: 路由設定檔。
