## 4. M3 Mac Studio 96GB 規格建議配置
針對 96GB 記憶體，建議以下環境配置：
- **Unified Memory Allocation**: 針對大型語言模型 (LLM) 本地推論，建議配置 GPU 記憶體使用上限為 80GB (透過 `sysctl iogpu.wired_limit_mb`)。
- **Build Stack**:
    - **Node.js**: 建議 20.x 以上版本。
    - **Python**: 建議 3.11+ 以利用 M3 核心優化。
- **Docker**: 建議在 Docker Desktop 設置中將 Memory 限制調高至 64GB+。
