# Proactive Evolution Workflow (Xiao Cai)

## Phase 1: Idea Generation (Draft)
- Capture random inspirations.
- Use `idea_manager.py add` to formalize the structure.

## Phase 2: Structural Evaluation
- **Impact (1-5)**: How much does this improve the current system?
- **Feasibility (1-5)**: How difficult is it to implement?

## Phase 3: Status Tracking
- **PROPOSED**: Initial entry.
- **EVALUATING**: Researching technical requirements.
- **IN_PROGRESS**: Active development.
- **IMPLEMENTED**: Fully deployed.

## Tools
- `ideas_db.json`: Source of truth.
- `IDEA_REPORT.md`: Automatically generated dashboard.
