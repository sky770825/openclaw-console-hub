# GitHub Skill PR 審查助手技術規格

## 功能概述
自動審查 GitHub PR，檢查代碼質量、測試覆蓋率、衝突檢測、安全問題。

## 核心功能

### 自動檢查項目
```
✓ 程式碼風格檢查 (ESLint/Prettier)
✓ 測試覆蓋率檢查
✓ 依賴衝突檢測
✓ 安全漏洞掃描 (npm audit)
✓ 型別檢查 (TypeScript)
✓ 文檔更新檢查
✓ Commit 訊息規範
✓ 檔案大小檢查
```

```yaml
  - type: test-coverage

  - type: documentation
    enabled: true

```

## 技術實作

```typescript
// 檢查流程
async function reviewPR(pr: PullRequest) {
  const checks = [
    runESLint(pr.files),
    runTypeScript(pr.files),
    runTests(pr.files),
    checkTestCoverage(pr.files, 80),
    runSecurityScan(pr.files),
    validateCommitMessages(pr.commits),
    checkDocumentation(pr.files)
  ];
  
  const results = await Promise.all(checks);
  const summary = generateSummary(results);
  await postComment(pr, summary);
  
  return summary;
}
```
