## 實戰案例 1：每日新聞摘要自動化

### 場景
每天早上 8 點，自動搜尋科技新聞，生成摘要，發送到 Telegram。

### 設定檔
```json
{
  "cron": {
    "jobs": [
      {
        "name": "daily-tech-news",
        "schedule": "0 8 * * *",
        "task": {
          "type": "agent",
          "prompt": "搜尋今日科技新聞，生成 5 條摘要，發送到 Telegram"
        }
      }
    ]
  }
}
```

### Agent 腳本
```javascript
// skills/daily-news.js
module.exports = {
  name: 'daily-news',
  
  async run({ tools }) {
    // 1. 搜尋新聞
    const news = await tools.webSearch({
      query: 'tech news today AI',
      count: 10
    });
    
    // 2. AI 生成摘要
    const summary = await tools.llm.complete({
      prompt: `將以下新聞整理成 5 條摘要：\n${news}`
    });
    
    // 3. 發送通知
    await tools.telegram.send({
      chatId: process.env.TELEGRAM_CHAT_ID,
      message: `📰 今日科技新聞：\n\n${summary}`
    });
    
    return { success: true, articles: news.length };
  }
};
```

## 實戰案例 3：智能客服自動回覆

### 場景
自動讀取客戶郵件，分類問題類型，生成回覆草稿。

### Agent 配置
```javascript
// agents/support-agent.js
module.exports = {
  name: 'support-agent',
  model: 'gpt-4o-mini',
  
  systemPrompt: `
你是一個專業的客戶服務助理。
請根據客戶郵件內容：
1. 分類問題類型（技術/帳務/一般）
2. 提取關鍵資訊
3. 生成專業回覆草稿
4. 標記緊急程度
  `,
  
  tools: ['email', 'crm', 'knowledge-base'],
  
  async onEmailReceived(email) {
    const analysis = await this.analyze(email);
    
    if (analysis.urgency === 'high') {
      await this.notifySlack({ channel: '#urgent-support', email });
    }
    
    const draft = await this.generateReply(analysis);
    await this.saveDraft({ emailId: email.id, draft });
    
    return { classified: analysis.category, draft };
  }
};
```

# backup.sh

  ~/Documents/Important \
  ~/Projects/Config

# 2. 呼叫 OpenClaw Agent 處理後續

```javascript
// skills/backup-manager.js
async function uploadAndNotify({ file }) {
  // 上傳到 AWS S3
  const s3Url = await tools.aws.s3.upload({
    bucket: 'my-backups',
    key: `backups/${file}`,
    file
  });
  
  // 驗證備份完整性
  const checksum = await tools.file.checksum(file);
  
  // 發送確認信
  await tools.email.send({
    to: 'admin@example.com',
    subject: '✅ 備份完成通知',
    body: `
備份檔案：${file}
上傳位置：${s3Url}
檔案校驗：${checksum}
完成時間：${new Date().toLocaleString()}
    `
  });
  
  // 清理本地舊備份
  await tools.file.clean({ pattern: 'backup-*.tar.gz', keep: 3 });
}
```
