## 3. 上架平台

### 主要平台
1. **Gumroad** (推薦)
   - 優點: 數位產品友善、低手續費、全球支付
   - 費率: 10% + 處理費
   - 適合: 個人開發者

2. **GitHub Releases**
   - 優點: 開發者熟悉、免費
   - 適合: 開源專案

3. **npm Registry**
   - 優點: JavaScript 生態、版本管理
   - 適合: 程式導向用戶

### 次要平台
- **ProductHunt**: 新產品曝光
- **Reddit r/selfhosted**: 社群分享
- **Hacker News**: 技術社群

## 4. 免費版設計原則

### 功能限制策略
```
Freemium 模式:
├── 免費版: 80% 功能，用量限制
├── 專業版: 100% 功能，無限制
└── 企業版: 100% 功能 + 支援服務
```

### 轉換漏斗設計
1. **發現** → ProductHunt/Reddit 曝光
2. **試用** → 免費 Skill 下載
3. **體驗** → 每日/每月配額使用
4. **限制** → 用量達上限提示升級
5. **轉換** → Gumroad 購買 Pro
