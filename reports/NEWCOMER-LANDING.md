# NEUXA 新兵著陸區 (Newcomer Landing Zone)

## 🚀 預設著陸點：孵化灣 (Incubator Bay)

**所有新註冊會員的預設降落位置**

### 著陸流程 (Onboarding Flow)

```
註冊完成
    ↓
[著陸艙] - 歡迎動畫 + NEUXA 品牌影片 (30秒)
    ↓
[身份掃描] - 生成 Machine-ID + 創世代號
    ↓
[星圖導覽] - 3D 宇宙生態快速導覽 (可跳過)
    ↓
任務分流 ──┬──→ 個人用戶 → Lite 行星試用區
           ├──→ 開發者 → Code 競技場體驗區
           ├──→ 企業主 → B2B 企業星預覽區
           └──→ 探索者 → 公共廣場自由活動
```

---

## 🎯 分流機制

### 1. 身份問卷 (3題快速分流)

| 問題 | 選項 | 預設著陸點 |
|------|------|-----------|
| **你是誰？** | 想省錢的個人 | Lite 行星試用區 |
| | 會寫程式的開發者 | Code 競技場體驗區 |
| | 有團隊的企業主 | B2B 企業星預覽區 |
| | 先逛逛再說 | 公共廣場 |
| **你的目標？** | 降低 AI 成本 | Lite 行星 |
| | 賺取賞金 | Code 競技場 |
| | 建立 Agent 系統 | 孵化灣專案區 |
| | 學習交流 | 學習花園 |
| **技術程度？** | 完全不會 | 學習花園新手村 |
| | 會基本操作 | 公共廣場 |
| | 會寫程式 | Code 競技場 |
| | 技術專家 | L2 技術星預覽 |

## 📍 各分流著陸點詳情

### A. Lite 行星試用區 (個人用戶首選)
**適合**: 想省錢、不懂程式、純使用者

**著陸後看到**:
- 💰 「省了 92% 雲端費用」即時計算器
- 🎮 互動式成本比較圖
- 📥 NT$990 一鍵購買按鈕
- 🎁 限時優惠：輸入邀請碼折 NT$100

**試用內容**:
- 5 次免費 AI 對話體驗
- 個人化 Machine-ID 預覽
- 下載 NEUXA Lite 安裝包

---

### B. Code 競技場體驗區 (開發者首選)
**適合**: 會寫 code、想賺賞金、技術挑戰

**著陸後看到**:
- 🏆 當週最熱 Bounty 任務榜
- 💻 線上 IDE 預覽 (不用安裝)
- 📊 賞金歷史與排行榜
- 🔧 技術文件快速入口

**試用內容**:
- 1 個新手 Bounty 任務 (完成得 NT$100 獎金)
- 代號生成器體驗
- 提交中心預覽

---

### C. B2B 企業星預覽區 (企業主首選)
**適合**: 有團隊、有預算、要客製

**著陸後看到**:
- 📋 服務價目表 (Full Build / Rental / MVP / Blueprint)
- 📞 預約諮詢表單
- 🏢 成功案例展示
- 💼 企業方案比較表

**試用內容**:
- 免費 30 分鐘諮詢預約
- 需求評估問卷
- 報價試算工具

---

### D. 公共廣場 (探索者首選)
**適合**: 不確定、想逛逛、純好奇

**著陸後看到**:
- 📢 NEUXA 最新動態牆
- 💬 熱門討論串預覽
- 🎉 即將舉辦的活動
- 🌟 社群精選內容

**試用內容**:
- 瀏覽權限 (部分內容需登入)
- 訪客留言功能
- 電子報訂閱

## 🎮 著陸艙體驗設計

### 第 1 秒：視覺衝擊
- 全螢幕 3D 宇宙縮放動畫
- NEUXA Logo 發光效果
- 背景播放星際音效

### 第 3 秒：身份賦予
```
「歡迎來到 NEUXA 宇宙」
「你的星際代號：Star-77」
「著陸座標：孵化灣 Sector-7」
```

### 第 10 秒：快速導覽
- 3D 星圖自動飛行導覽 (30秒可跳過)
- 各行星快速預覽
- 點擊感興趣的區域

### 第 30 秒：任務分流
- 彈出 3 題快速問卷
- 即時計算推薦著陸點
- 手動選擇「我想去別的地方」
