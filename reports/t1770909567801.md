# ClawFlow 日誌追蹤系統技術規格

## 功能概述
結構化日誌、分佈式追蹤 (Distributed Tracing)、分級輸出。

## 核心功能

### 日誌等級

```typescript
enum LogLevel {
  DEBUG = 0,    // 開發調試
  INFO = 1,     // 一般資訊
  WARN = 2,     // 警告
  ERROR = 3,    // 錯誤
  FATAL = 4     // 致命錯誤
}
```

### 結構化日誌

```json
{
  "timestamp": "2026-02-13T02:06:00Z",
  "level": "info",
  "logger": "skill-executor",
  "message": "Task completed",
  "traceId": "abc123def456",
  "spanId": "xyz789",
  "context": {
    "taskId": "task-123",
    "skillName": "github-skill",
    "duration": 2500
  },
  "metadata": {
    "environment": "production",
    "version": "2.0.1"
  }
}
```

### 分佈式追蹤

```typescript
// 生成追蹤 ID
const traceId = generateTraceId();  // abc123...

// 在整個執行流程傳遞
skillA.execute({ traceId });
  → skillB.execute({ traceId });
    → externalAPI.call({ traceId });

// 最後在日誌系統中關聯
logs.filter({ traceId: 'abc123' })
// 查看該操作的所有日誌
```

### 日誌輸出

```
[2026-02-13 10:06:30] [INFO] [skill-executor] Task started (traceId: abc123)
[2026-02-13 10:06:32] [DEBUG] [github-skill] Connecting to GitHub API
[2026-02-13 10:06:33] [INFO] [github-skill] Fetched 100 issues
[2026-02-13 10:06:35] [INFO] [skill-executor] Task completed in 2500ms
```
