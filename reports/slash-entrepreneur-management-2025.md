# 斜槓創業家時間管理與多事業財務規劃

> 來源：Gemini Flash 2.5 搜尋整理
> 日期：2026-02-16
> 適用對象：斜槓創業家（多事業經營者）

## 1. 多工作業時間分配技巧

- **優先順序矩陣 (Eisenhower Matrix)**：區分重要且緊急、重要但不緊急、緊急但不重要、不重要也不緊急
- **時間區塊 (Time Blocking)**：將一天劃分為不同區塊，每個區塊專注於特定事業
- **番茄工作法**：25分鐘專注工作，5分鐘休息，提高效率避免 Burnout
- **任務委派與外包**：將非核心或重複性工作委託給他人或外包
- **彈性工作排程**：預留彈性時間應對突發狀況

## 2. 現金流管理原則

| 原則 | 說明 |
|------|------|
| 建立獨立帳戶 | 為每個事業建立獨立銀行帳戶，避免混淆 |
| 現金流預測 | 定期預測現金流入和流出，提前發現財務問題 |
| 緊急備用金 | 每個事業預留3-6個月營運資金 |
| 嚴格控制支出 | 審慎評估每項支出，尋找降低成本的機會 |
| 定期檢視財報 | 審閱損益表、資產負債表和現金流量表 |

## 3. 稅務規劃基礎

- **了解稅務法規**：熟悉所得稅、營業稅等相關規定
- **合法節稅管道**：費用列報、投資抵減等合法節稅方式
- **保留完整憑證**：保留所有收入和支出憑證以便報稅
- **尋求專業諮詢**：諮詢會計師或稅務顧問確保合規性
- **設立個人公司或工作室**：以利稅務規劃和風險管理
