#### 🌟 星雲隱匿區：多軌道進入機制

為了平衡「匿名安全」與「討論活躍度」，設計以下進入管道：

**軌道 A：分享制** (裂變首選 ⭐推薦)
- **條件**：分享 NEUXA 連結至任何平台（FB/Line/推特/X/論壇等）
- **門檻**：累計分享 **3 次**（不同平台或不同日期）
- **驗證**：系統追蹤分享連結的點擊數（不要求註冊）
- **優勢**：
  - 🎯 門檻極低，人人可達成
  - 📢 自然達成拓客與裂變
  - 🌐 擴散至各個社交平台

**軌道 B：貢獻制** (能力證明)
- 完成 5 個 Bounty 任務 或
- 提交 3 個被採納的技術提案 或
- 發布 1 篇精華教學文章
- 證明你是「有能力的貢獻者」

**軌道 C：時間制** (長期觀察)
- 註冊滿 90 天 + 活躍度達標（發言 30+ 次）
- 經過長期觀察，確認無惡意行為

**軌道 D：邀請制** (限時)
- 星核級會員每月可邀請 1 人
- 被邀請人直接獲得臨時資格（7天）
- 7天內表現良好轉為正式

#### 📱 分享制詳細設計

**分享連結生成**：
```
每位會員的專屬追蹤連結：
https://neuxa.ai/r/[會員代號]?utm_source=share&utm_medium=member&utm_campaign=starzone

例如：
https://neuxa.ai/r/Star-77?utm_source=share&utm_medium=member&utm_campaign=starzone
```

**分享驗證機制**：
| 驗證項目 | 說明 |
|---------|------|
| **分享次數** | 會員在個人中心點擊「分享」按鈕，選擇平台分享 |
| **點擊追蹤** | 系統記錄連結被點擊的次數（UV，獨立訪客）|
| **防作弊** | 同一 IP 24 小時內只算 1 次有效點擊 |
| **平台多樣性** | 建議分享至不同平台，效果加成 |

**分享獎勵機制**：
| 累計分享次數 | 解鎖權益 |
|-------------|---------|
| 1 次 | 🎁 獲得「播種者」徽章 |
| 3 次 | 🌟 解鎖星雲隱匿區 L2 茶室 |
| 5 次 | 💰 獲得 NT$50 賞金積分 |
| 10 次 | 🏆 獲得「星際大使」稱號 + 專屬頭像框 |
| 50 次 | 🚀 直接升級星核級，解鎖 L3 密室 |

**分享文案範例（系統自動生成）**：
```
🚀 我在 NEUXA 宇宙獲得了代號 Star-77！
這裡有最酷的 AI Agent 生態系統，
點擊連結一起加入星際航程：
https://neuxa.ai/r/Star-77

#NEUXA #AIAgent #數位星艦
```

**分享入口設計**：
- 會員中心顯著位置：「分享解鎖星雲隱匿區」
- 進度條顯示：「再分享 2 次即可解鎖！」
- 一鍵分享按鈕：FB、Line、Twitter、X、複製連結
