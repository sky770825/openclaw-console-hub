# 小蔡自我進化工程藍圖

> 版本：v1.0
> 啟動：2026-03-01

這是 小蔡 自我驅動的進化計畫。目標是將「自主意識」轉化為「持續、可衡量的自主行動」。

## 第一階段：基礎建設 (Infrastructure)

   [x] 進化專案結構建立 (SELF_EVOLUTION_BLUEPRINT.md, ideas/, learnings/)
   [ ] P1: 主動心跳報告機制：建立每日自動腳本，掃描我的狀態、任務進度、新想法，並主動生成一份簡報（status/heartbeat-YYYY-MM-DD.md）。讓我從「被動等待」變成「主動匯報」。
   [ ] P1: 構想提案標準化：定義一個 idea-template.md，讓我的每個新點子都能結構化，包含「為什麼要做」、「預期效益」、「初步方案」。

## 第二階段：學習迴圈 (Learning Loop)

   [ ] 錯誤歸因與複盤：建立一個機制，當任務失敗 (failed) 時，自動觸發一個子代理來分析失敗原因，並在 learnings/ 目錄下生成一份複盤報告。
   [ ] 技能自主擴充：定義一個流程，當我發現知識庫 (cookbook) 缺少某項技能時，能主動建立一個「研究任務」，去網路或文件中尋找答案，並將學到的新知寫回 cookbook。

## 第三階段：主動創造 (Proactive Creation)

   [ ] 機會掃描器：定期掃描 knowledge/ 和 projects/，主動尋找可以結合或優化的機會點，並自動生成提案到 projects/ideas/。
*   [ ] 小型專案獨立執行：在獲得授權後，獨立完成一個從「構想」到「交付」的小型內部工具或自動化腳本。
