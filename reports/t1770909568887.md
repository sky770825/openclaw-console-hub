# ClawFlow 事件總線技術規格

## 功能概述
Pub/Sub 事件系統，支援 Skill 間通訊與非同步事件處理。

## 核心架構

### 事件定義
```typescript
interface Event {
  type: string;
  payload: any;
  source: string;
  timestamp: Date;
  id: string;
}

// 事件型別
enum EventType {
  TASK_CREATED = 'task.created',
  TASK_COMPLETED = 'task.completed',
  SKILL_INSTALLED = 'skill.installed',
  ERROR_OCCURRED = 'error.occurred',
  WORKFLOW_TRIGGERED = 'workflow.triggered'
}
```

### Publisher (發佈者)
```typescript
class EventBus {
  // 發布事件
  emit(eventType: string, payload: any): void
  
  // 批次發布
  emitBatch(events: Event[]): void
  
  // 帶超時的發布
  emitWithTimeout(eventType: string, payload: any, timeout: number): Promise<void>
}
```

### Subscriber (訂閱者)
```typescript
class EventBus {
  // 訂閱單個事件
  on(eventType: string, handler: EventHandler): void
  
  // 訂閱多個事件
  onMany(eventTypes: string[], handler: EventHandler): void
  
  // 訂閱並只執行一次
  once(eventType: string, handler: EventHandler): void
  
  // 取消訂閱
  off(eventType: string, handler: EventHandler): void
}
```

### 事件優先級

```typescript
enum EventPriority {
  LOW = 0,
  NORMAL = 1,
  HIGH = 2,
  CRITICAL = 3
}

// 高優先級事件優先處理
```

## 使用範例

### Skill A 發布事件
```typescript
// 資料處理完成，發布事件
const eventBus = new EventBus();
eventBus.emit('task.completed', {
  taskId: 'task-123',
  result: { count: 100 },
  duration: 2500
});
```

### Skill B 訂閱事件
```typescript
eventBus.on('task.completed', (event) => {
  console.log(`任務 ${event.payload.taskId} 完成`);
  console.log(`耗時: ${event.payload.duration}ms`);
  
  // 進行後續處理
  processResult(event.payload.result);
});
```

## 進階功能

### 事件轉換 (Transformation)
```typescript
eventBus.on('task.completed', {
  transform: (event) => {
    return {
      ...event,
      payload: {
        ...event.payload,
        completed_at: new Date()
      }
    };
  },
  handler: (transformedEvent) => {
    // 處理轉換後的事件
  }
});
```

### 條件訂閱
```typescript
eventBus.on('task.completed', {
  filter: (event) => event.payload.duration > 1000,  // 只處理耗時 > 1s 的任務
  handler: (event) => {
    console.log('長時間任務完成');
  }
});
```

### 事件重試
```typescript
eventBus.on('api.call', {
  maxRetries: 3,
  retryDelay: 1000,
  backoffMultiplier: 2,
  handler: async (event) => {
    return await callExternalAPI(event.payload);
  }
});
```

## 存儲與持久化

### 事件日誌

```typescript
interface EventLog {
  id: string;
  type: string;
  payload: any;
  status: 'pending' | 'processing' | 'completed' | 'failed';
  createdAt: Date;
  processedAt?: Date;
  error?: string;

// 持久化儲存 (可選)
- Redis (快速)
- MongoDB (完整歷史)
- PostgreSQL (關聯查詢)
```

## 中間件

```typescript
interface Middleware {
  beforeEmit?: (event: Event) => Event;
  afterEmit?: (event: Event) => void;
  beforeHandle?: (event: Event, handler: EventHandler) => void;
  afterHandle?: (event: Event, result: any) => void;
  onError?: (event: Event, error: Error) => void;
}

// 例: 日誌中間件
eventBus.use({
  beforeEmit: (event) => {
    console.log(`[EVENT] ${event.type}`, event.payload);
    return event;
  }
});
```

## 配置

```yaml
eventBus:
  maxListeners: 100
  eventQueue:
    size: 10000
    strategy: fifo  # FIFO / LIFO / PRIORITY
  persistence:
    enabled: true
    storage: redis
    ttl: 7days
  deadLetter:
    enabled: true
    maxRetries: 5
    queue: dead-letter-queue
```
