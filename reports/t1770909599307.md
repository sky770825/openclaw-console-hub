**任務類型**: 開發

## 規格說明

### 目標
實作完整的自主探索邏輯，使 Autopilot Agent 能自動發現、執行和報告任務。

### 核心機制

#### 1. 任務發現與排序
```
1. 定時查詢 /api/openclaw/tasks (status=ready)
2. 按 priority DESC、createdAt ASC 排序
3. 智能分類：[Kimi]/[Cursor]/[CoDEX]/[Ollama]/[OR-Free]
4. 根據 Agent 類型分配執行器
```

#### 2. 執行引擎
- **同步執行**: 5 個任務並行 / 10 秒週期
- **智能限流**: 每分鐘最多 12 個任務（避免過載）
- **失敗重試**: 3 次重試 + 指數退避

#### 3. 結果上報
- 寫入 `memory/autopilot-results/{task-id}.md`
- Telegram 通知 (✅ 完成 / ❌ 失敗)
- 更新任務狀態: running → done / failed

#### 4. 監控與告警
- 執行成功率追蹤
- 異常任務自動標記 [STUCK]
- 30 分鐘無進展自動重啟
