# 美業網站後台管理介面開發報告

## 任務概述
開發一個直觀且具備 CRUD 功能的後台管理介面，支持平台管理員對用戶、店家、服務及預約的管理。

## 核心功能實作
- **儀表板 (Dashboard)**: 提供即時數據統計（使用者數、店家數、服務數、預約數）。
- **使用者管理**: 支持列表顯示、權限識別（會員 vs 店家管理員）、狀態監控及 CRUD 操作入口。
- **店家管理**: 針對美業店家資料的維護介面。
- **預約管理**: 監控全站預約狀態，支持待確認訂單的視覺化提醒。
- **技術棧**: React 18 + Vite + Tailwind CSS + Lucide Icons。

## 檔案結構
- `/sandbox/admin-dashboard`: 完整的源代碼目錄。
- `/scripts/launch_admin_dashboard.sh`: 啟動腳本。
