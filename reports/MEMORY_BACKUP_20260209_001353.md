# MEMORY.md - 長期記憶關鍵字索引（省 Token）

**原則：只有記憶庫（MEMORY_FULL.md、memory/YYYY-MM-DD.md）裡有對應內容的主題才列關鍵字；沒有資料不設索引。**

**用法：** 輸入出現下方關鍵字時，先 `memory_search` 或讀 MEMORY_FULL.md 對應段落，再回答。

**主題與命中：** 當前主題見 **MEMORY_TOPIC.md**（若有）；該主題相關關鍵字視為高優先。若某關鍵字連續多輪（如 3 輪）未命中，可考慮從本索引移除或合併。**比例分配：** 最近 N 輪（3／5／10）內若某區塊 ≥ 70%，本輪只載入主導 + 有匹配的區塊，不重複引入低比例區塊；狀態見 memory/index_state.json。詳見 docs/記憶索引-主題與命中率設計.md、記憶索引-比例分配與實作流程.md。
