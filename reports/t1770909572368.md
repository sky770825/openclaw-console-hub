## 2. 核心功能規格

### 2.1 連線管理
```typescript
interface DBConfig {
  type: 'mysql' | 'postgresql' | 'mongodb';
  host: string;
  port: number;
  username: string;
  password: string;
  database: string;
  ssl?: boolean;
  poolSize?: number;
}

class DatabaseManager {
  connect(config: DBConfig): Promise<Connection>
  disconnect(): Promise<void>
  testConnection(): Promise<boolean>
}
```

### 2.2 CRUD 操作
```typescript
interface CRUDOperations {
  // Create
  insert(table: string, data: object): Promise<InsertResult>
  insertMany(table: string, data: object[]): Promise<InsertResult>
  
  // Read
  find(table: string, query: Query): Promise<any[]>
  findOne(table: string, query: Query): Promise<any>
  
  // Update
  update(table: string, query: Query, data: object): Promise<UpdateResult>
  updateMany(table: string, query: Query, data: object): Promise<UpdateResult>
  
  // Delete
  delete(table: string, query: Query): Promise<DeleteResult>
  deleteMany(table: string, query: Query): Promise<DeleteResult>
}
```

### 2.3 查詢構建器
```typescript
interface QueryBuilder {
  select(fields: string[]): QueryBuilder
  from(table: string): QueryBuilder
  where(condition: WhereClause): QueryBuilder
  orderBy(field: string, direction: 'ASC' | 'DESC'): QueryBuilder
  limit(n: number): QueryBuilder
  offset(n: number): QueryBuilder
  execute(): Promise<any[]>
}
```

## 3. 資料庫特定實作

### 3.1 MySQL
- 使用 `mysql2` 套件
- 支援 prepared statements
- 連線池管理

### 3.2 PostgreSQL
- 使用 `pg` 套件
- 支援 JSON/JSONB 操作
- 支援陣列類型

### 3.3 MongoDB
- 使用 `mongodb` 官方驅動
- 支援 Aggregation Pipeline
- 支援 Index 管理
