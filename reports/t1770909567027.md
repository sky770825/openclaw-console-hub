# ClawFlow 任務調度器核心技術規格

## 功能概述
並發控制、優先級佇列、逾時處理。

## 任務佇列

```typescript
interface TaskQueue {
  // 添加任務
  enqueue(task: Task): void
  
  // 取出下一個任務
  dequeue(): Task | null
  
  // 優先級佇列
  priority: 'fifo' | 'lifo' | 'priority'
}

interface Task {
  id: string;
  priority: 0-10;        // 0=lowest, 10=highest
  payload: any;
  timeout: number;       // 毫秒
  retries: number;
  status: TaskStatus;
}
```

## 並發控制

```typescript
const scheduler = new TaskScheduler({
  maxConcurrent: 5,      // 最多同時執行 5 個任務
  maxQueueSize: 1000,    // 佇列最多 1000 個待執行任務
  workerCount: 5         // 5 個執行緒
});

// 動態調整
scheduler.setMaxConcurrent(10);
```

## 優先級範例

```
佇列:
[高]  Task A (priority: 10)
[中]  Task B (priority: 5)
[低]  Task C (priority: 1)

執行順序: A → B → C (優先級越高越先執行)
```

## 超時與取消

```typescript
// 設定超時
const task = new Task({
  timeout: 30000  // 30 秒超時
});

// 執行時超時會自動取消
scheduler.on('timeout', (task) => {
  console.log(`Task ${task.id} timeout`);
  // 進行清理
});
```
