## 📄 產品詳細描述 (Description)

### 開場 (Hook)
```
厭倦了每天重複的工作？ClawFlow 讓你用自然語言建立自動化工作流程。

這不是另一個複雜的 workflow builder — 這是為開發者打造的開源 AI Agent 框架，本地運行、完全掌控、無限擴展。
```

### 核心功能 (Features)

**🔧 開箱即用的 Skill 生態**
- GitHub: Issue/PR 自動管理、審查提醒
- Email: 智能分類、自動回覆、範本系統
- Notion: 資料庫同步、頁面自動化
- Web Scraper: 定時監控、資料清洗
- Messaging: Telegram/Slack/Discord 自動化

**🤖 本地優先的 AI 整合**
- 原生支援 Ollama，無需雲端 API
- 支援 OpenRouter、OpenAI、Anthropic 等多模型
- 資料不離開你的機器，隱私有保障

**⚡ 開發者友好的設計**
- TypeScript 全程開發，型別安全
- 熱插拔 Skill 插件系統
- CLI + 視覺化編輯器雙模式
- BullMQ 任務佇列，穩定可靠

**🌐 瀏覽器自動化**
- Playwright 驅動的網頁操作
- 登入狀態持久化
- 反爬蟲對策內建

### 快速開始 (Quick Start)
```bash
# 安裝
npm install -g clawflow

# 初始化專案
clawflow init my-automation

# 加入所需 Skill
clawflow add github email notion

# 編輯 workflow.yaml
# 執行
clawflow run
```

## ❓ FAQ 常見問題

**Q: ClawFlow 與 n8n/Zapier 有什麼不同？**
A: ClawFlow 專為開發者設計，程式碼優先、本地執行、完全開源。n8n 雖然也是開源，但設定較複雜；Zapier 則是昂貴的雲端服務。ClawFlow 結合兩者優點：開源自由 + 簡單易用。

**Q: 需要會寫程式才能使用嗎？**
A: 基礎使用不需要，YAML 設定即可。進階功能可以寫 JavaScript/TypeScript 擴展 Skill。

**Q: 支援哪些 AI 模型？**
A: 本地：Ollama (Llama, DeepSeek, Qwen 等)；雲端：OpenAI, Anthropic, OpenRouter, Google Gemini 等。

**Q: 資料安全嗎？**
A: 完全本地執行，不傳輸資料到第三方。API Key 儲存在本地加密檔案。

**Q: 可以商用嗎？**
A: 可以！MIT License 允許自由使用、修改、商用。

**Q: 有企業版嗎？**
A: 核心功能完全免費。未來會推出企業級支援與進階功能。

**Q: 如何開發自己的 Skill？**
A: 使用 `clawflow create-skill` 命令，基於官方模板快速開發。文件：https://docs.clawflow.dev/skills

**Q: 支援哪些平台？**
A: macOS、Linux、Windows (WSL)。推薦使用 macOS 或 Linux 獲得最佳體驗。
