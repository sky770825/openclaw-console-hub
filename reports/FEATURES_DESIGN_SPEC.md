# OpenClaw 官網功能特色頁面設計方案 (Features Page Design)

## 1. 核心定位
展示 OpenClaw 作為「最強個人 AI 代理系統」的關鍵競爭力：完全本地化控制、極強的 MCP 協議支持、可視化任務編排。

## 2. 頁面結構 (Sections)
1. **Hero Section**: 
   - 標語：控制你的 AI，釋放你的靈魂。
   - 動態背景：抽象的節點連接圖。
2. **Core Feature Grid (3x2)**:
   - **自主 Agents**: 基於 MCP 的高度自律任務執行。
   - **完全沙盒化**: 嚴格的權限管理與本地執行環境。
   - **可視化編排**: 任務面版直觀操作（對應當前任務面版設計）。
   - **跨平台支持**: macOS 原生體驗。
   - **多模型兼容**: 支持 OpenAI, Claude, Local LLMs (Ollama)。
   - **知識庫整合**: 本地 RAG 系統。
3. **Interactive Demo Area**:
   - 一個模擬 OpenClaw 任務執行的 UI 動畫。
4. **Integration Showcase**:
   - 展示對主流 IDE, Browser, File System 的集成能力。

## 3. UI 規範
- **色調**: 深色模式為主 (Dark Mode)，深靛藍 (#0f172a) 與 螢光紫/青 (#a855f7 / #22d3ee)。
- **字體**: Inter / System Sans-serif。
- **組件**: 玻璃擬態 (Glassmorphism)。
