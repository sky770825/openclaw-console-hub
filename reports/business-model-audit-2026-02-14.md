## 二、各事業現況分析

### 2.1 住商biz_realestate（桃園loc_yangmei區）

#### 業務特性
- **區域特性**：桃園loc_yangmei區，新興重劃區與傳統社區並存
- **客戶類型**：首購族、換屋族、投資客、退休置產
- **成交週期**：平均 45-90 天
- **競爭環境**：大型連鎖房仲 + 個人工作室並存

#### 現行作業流程
```
房源取得 → 拍照/量測 → 文案撰寫 → 上架平台 → 回應詢問 
→ 安排帶看 → 價格談判 → 成交簽約 → 過戶交屋
```

#### 痛點盤點
| 痛點 | 影響程度 | 發生頻率 |
|------|---------|---------|
| 優質房源文案耗時 | 高 | 每日 |
| 潛在買方篩選耗人力 | 高 | 每日 |
| 客戶跟進不及時 | 中高 | 持續 |
| 帶看安排效率低 | 中 | 每週 |
| 市場分析報告製作慢 | 中 | 每月 |

#### 可變現機會點
1. **自動化房源文案生成**：節省 70% 文案撰寫時間
2. **智能客戶分級系統**：精準識別高意向買家
3. **自動跟進系統**：提升客戶留存率
4. **區域市場分析自動化**：快速產出專業報告

### 2.2 biz_drinks（自有店鋪）

#### 業務特性
- **店型**：自有店鋪，租金成本低
- **客群**：路過客、附近上班族、學生、社區居民
- **營業時段**：早午餐、下午茶、傍晚尖峰
- **毛利結構**：飲品毛利 60-75%，材料成本可控

#### 現行作業流程
```
原料備貨 → 開店營業 → 點單製作 → 服務客戶 
→ 收店盤點 → 社群經營 → 促銷規劃
```

#### 痛點盤點
| 痛點 | 影響程度 | 發生頻率 |
|------|---------|---------|
| 離峰時段產能閒置 | 高 | 每日 |
| 客群輪廓模糊 | 中高 | 持續 |
| 回購率無法追蹤 | 高 | 持續 |
| 促銷決策憑直覺 | 中 | 每月 |
| 社群內容產出耗時 | 中 | 每週 |

#### 可變現機會點
1. **會員系統 + 行為分析**：提升回購率與客單價
2. **動態定價與促銷**：離峰時段產能優化
3. **自動化社群經營**：持續曝光不間斷
4. **原料庫存預測**：減少浪費與缺貨
