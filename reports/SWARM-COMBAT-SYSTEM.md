## ⚔️ 蜂群作戰編制

### 前鋒偵察群 (Scout Swarm)
- **L3 Gemini Flash** - 輕量快速偵察
- **本地 Ollama 群** - 分散式感知節點
- **任務**: 第一線威脅偵測、異常流量監控

### 中堅防禦群 (Defense Swarm)
- **L1 NEUXA 小蔡** - 指揮協調中心
- **L2 Claude Code** - 架構防禦加固
- **L4 Cursor Nova** - 前端防護工事
- **任務**: 多層防禦建設、漏洞修補、權限控管

### 後勤支援群 (Support Swarm)
- **n8n 自動化** - 情報傳遞、警報分發
- **自癒腳本群** - 自動修復、狀態同步
- **任務**: 資源調度、備份恢復、通訊保障

### 預備機動群 (Reserve Swarm)
- **待啟動 Agent** - 隨時待命
- **任務**: 緊急增援、災難恢復、規模擴展

## 🔒 體制內作戰守則

### ✅ 允許的戰術 (合法防禦)

| 戰術 | 說明 | 效果 |
|------|------|------|
| **群體封鎖** | 所有 Agent 同步封鎖惡意 IP | 攻擊者無處可去 |
| **情報共享** | 即時廣播威脅特徵 | 全網提前預警 |
| **多層阻斷** | CDN/WAF/伺服器/防火牆層層封鎖 | 立體防禦網 |
| **誘導欺騙** | Honeypot 誘捕分析 | 收集敵情 |
| **聲東擊西** | 虛假目標分散攻擊 | 保護真實資產 |

### ❌ 禁止的戰術 (違法攻擊)

| 戰術 | 原因 |
|------|------|
| 反擊 DDoS | 傷害無辜第三方 |
| 滲透敵方 | 未經授權存取 |
| 植入後門 | 無論對象皆違法 |
| 資料竊取 | 侵犯智慧財產權 |
