## 📖 Overview
Salesforce Einstein is the AI layer for Salesforce CRM, featuring Einstein GPT, Copilot, and AI agents for automation. Integrates Data Cloud for unified data, driving business outcomes like 40% less maintenance, 95% accuracy.

**📢 2025年重大更新**: **Agentforce** 正式發布 + **Agentforce 3** + **ChatGPT 整合**

## 🆕 2025年 Agentforce 重大突破

### Agentforce 正式發布 (2024年10月)
Salesforce 於 2024年10月發布 **Agentforce**，2025年2月推出 **Atlas Reasoning Engine** 組件：

| 組件 | 發布時間 | 功能 |
|------|----------|------|
| Agentforce for Service | 2024-10-25 | 服務自動化 |
| Agentforce for Sales | 2024-10-25 | 銷售自動化 |
| Atlas Reasoning Engine | 2025-02 | 推理引擎 |

### Agentforce 3 (2025年8月)
2025年8月發布的 **Agentforce 3** 帶來重大升級：

| 新功能 | 說明 | 價值 |
|--------|------|------|
| **完整可觀測性** | 統一監控 Agent 健康狀態 | 領導者全面掌控 |
| **互動分析** | 深入分析每個互動細節 | 精準優化 |
| **AI 建議** | 針對標記對話類型的建議 | 智能決策 |
| **即時警報** | 延遲、升級頻率、錯誤率監控 | 即時響應 |

### ChatGPT 整合 (2025年12月)
Salesforce 將 **Agentforce 整合進 ChatGPT**：

- 使用者在 ChatGPT app 中查詢 hot leads
- 請求路由至 Agentforce
- Salesforce AI 使用自家評分模型、情境資料、業務邏輯評估 leads
- 格式化結果返回 ChatGPT 顯示和分析

**目的**: 防止企業自建 MCP servers，提供官方整合方案

Source: [SalesforceDevops.net](https://salesforcedevops.net/index.php/2025/12/22/salesforce-launches-agentforce-in-chatgpt-to-head-off-homegrown-mcp-servers/)

## 🔧 進階整合範例
```javascript
// OpenClaw + Einstein Workflow
const crmWorkflow = async () => {
  // 查詢高價值潛客
  const leads = await exec({
    command: "sf data query 'SELECT Id, Name, Score FROM Lead WHERE Score > 80'"
  });
  
  // 用 GPT 生成個性化郵件
  const email = await sessions_spawn({
    task: `為這些潛客寫郵件: ${leads}`,
    model: "gpt-5.2"
  });
  
  // 發送並記錄
  await message.send({ content: email });
};
```
