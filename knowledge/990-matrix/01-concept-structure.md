# 990 房產九宮格 (Real Estate BMC Matrix) - 概念架構

> 版本：v0.1 (初始化)
> 來源：Crew Bot (阿商/阿研) + 指揮官整合

## 1. 核心定義
將傳統商業模式畫布 (BMC) 的九大模組，轉化為房地產經紀人的「斜槓戰略網格」。這不只是分析工具，更是會員系統的核心數據結構。

## 2. 九宮格對應 (待阿商/阿策深化)

| BMC 區塊 | 房產對應 (990) | 關鍵指標 (KPI) |
|---|---|---|
| *價值主張* (Value Propositions) | 經紀人核心賣點 (專營豪宅/租賃/土地) | 委託轉換率 |
| *客戶區隔* (Customer Segments) | 買方/賣方/租客/投資客畫像 | 客戶數、活躍度 |
| *通路* (Channels) | 591/FB/IG/現場/老客戶介紹 | 流量來源佔比 |
| *客戶關係* (Customer Relationships) | 帶看/回報/節日問候/售後服務 | 滿意度、回頭率 |
| *關鍵收益* (Revenue Streams) | 佣金/介紹費/代辦費 | 總業績 (GCI) |
| *關鍵資源* (Key Resources) | 房源庫/人脈網/銷售技巧/工具 | 庫存件數 |
| *關鍵活動* (Key Activities) | 開發/銷售/議價/簽約 | 帶看量、開發量 |
| *關鍵夥伴* (Key Partnerships) | 代書/裝修/銀行/同業 | 合作成交數 |
| *成本結構* (Cost Structure) | 廣告費/油錢/時間成本/平台費 | 投報率 (ROI) |

## 3. PropTech 趨勢整合 (阿研貢獻)
- *數位轉型*：從紙本作業轉向數位足跡分析。
- *斜槓網絡*：經紀人不再單打獨鬥，透過 990 矩陣連結彼此的資源與強項。
