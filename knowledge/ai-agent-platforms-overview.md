# AI Agent 開發平台概覽

根據搜尋結果，以下是幾個主流的 AI Agent 開發平台及其主要特點：

## 1. CrewAI

   定位：專為開發者設計的彈性平台，用於建立 AI 代理並編排多代理工作流程。
   主要功能：
       全面控制多代理協作。
       支援自訂大型語言模型整合。
       基於角色的代理分配。
       可整合外部 API 處理複雜任務。
   適用場景：需要精細控制多代理協作和客製化 LLM 整合的開發者。

## 2. Vertex AI Agent Builder (Google)

   定位：針對開發者的雲端部署 AI 代理平台。
   主要功能：
       高擴展性的雲端原生平台。
       代理工作流程完全自訂。
       支援與 Google 雲端 AI 工具整合。
       提供企業級基礎設施。
   適用場景：需要建置具可擴展性且可自訂的 AI 代理，並享受企業級雲端服務的開發者。

## 3. Lindy

   定位：簡化小型團隊建立 AI 代理的流程。
   主要功能：
       易用的拖放工作流程編輯器。
       超過 200 種應用整合。
       提供內容與行銷自動化的預建範本。
   適用場景：非常適合行銷與客戶支援自動化，或需要快速部署 AI 代理的小型團隊。
