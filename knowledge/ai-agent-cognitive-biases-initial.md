# AI Agent 決策與認知偏差 (初步匯總)

## AI Agent 與認知偏差

AI Agent 定義
AI Agent 是一種能夠感知周圍環境、自主做出決策並動態調整以執行複雜任務的人工智慧系統。與傳統生成式AI需要人類逐一輸入指令不同，AI Agent 具備記憶、推理、規劃、使用工具及多方協作的能力，能從頭到尾自主處理複雜任務，實現更高程度的自動化。它們的目標是達成給定的目標，並能自行推論達成目標所需的過程並執行多個任務。

人類認知偏差
人類在決策、思考和判斷時，常因過往經驗和經歷而產生許多認知偏差（Cognitive Bias）。這些偏差會影響信念的形成、商業與經濟決策，並可能導致決策偏離理性和最佳選擇。常見的認知偏差包括：
   易得性捷思法（Heuristic Thinking）：根據容易取得的印象判斷事件可能性。
   倖存者偏誤（Survivorship Bias）：只注意成功者而忽略失敗者。
   確認偏誤（Confirmation Bias）：傾向於尋找、解釋和偏愛證實自己既有信念的資訊。
   投射偏誤（Projection Bias）：將自身想法投射到他人身上。
   過度自信偏誤（Overconfidence Effect）：因過去經驗產生過度自信而欠缺客觀分析。
   錨定效應（Anchoring Bias）：估值時受第一個接收到的訊息影響。
*   自動化偏差（Automation Bias）：過度依賴自動化系統，可能導致錯誤。
