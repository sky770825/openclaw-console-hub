### 向量搜尋系統

#### Qdrant + bge-m3（主力）

| 項目 | 數值 |
|------|------|
| Collection | `memory_smart_chunks` |
| Points | 330 |
| Dimensions | 1024 |
| Distance | Cosine |
| Status | green |
| Embedding 模型 | bge-m3（多語言，中文精準度 90% Top-1） |

**向量化腳本**: `~/.openclaw/workspace/scripts/vectorize-knowledge.py`

```bash
# 追加模式
python3 ~/.openclaw/workspace/scripts/vectorize-knowledge.py

# 清空重建（推薦）
python3 ~/.openclaw/workspace/scripts/vectorize-knowledge.py --rebuild
```

**覆蓋範圍**：
- `sop-知識庫/`（18 篇 SOP）
- `dar-指令集/`（12 篇指令）
- `knowledge/`（3+ 篇知識文件）
- `AGENTS.md`（核心規則）
- `memory/PROFILE.md`（達爾自我描述）

#### QMD 本地搜尋引擎（輔助）

| 項目 | 數值 |
|------|------|
| 總文件 | 270 |
| 向量 chunks | 744 |
| 資料庫 | ~/.cache/qmd/index.sqlite (6.7 MB) |
| GPU | Apple M5 Metal |

**5 個 Collections**：

| Collection | 檔案數 | 用途 |
|-----------|--------|------|
| sop | 19 | SOP 規範 |
| instruction | 12 | 達爾指令集 |
| knowledge | 54 | 知識文件 |
| docs | 48 | 一般文件 |
| memory | 137 | 記憶檔案 |

**搜尋策略建議**：
- 知道確切關鍵字 → `qmd-run search "關鍵字" -c sop`（BM25，快）
- 模糊語義搜尋 → Qdrant + bge-m3（90% Top-1）
- 最準結果 → `qmd-run query "問題"`（BM25 + 向量 + Reranking）
