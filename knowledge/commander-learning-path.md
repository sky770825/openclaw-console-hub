## 1. TypeScript (TS) — 系統的血管
- 核心概念：Interface (型別定義)、Async/Await (異步處理)、Generics (泛型)。
- 為什麼學：看懂 server/src/types.ts 就能理解所有數據結構。
- 實戰目標：能修改 openclaw_tasks 的處理邏輯。

## 2. Supabase & SQL — 記憶的深淵
- 核心概念：Select/Insert/Update/Delete、Join (關聯查詢)、Rls (列級安全)。
- 為什麼學：直接在 Dashboard 產出經營報表，掌握biz_realestate與biz_drinks數據。
- 實戰目標：寫出一個能統計「二月排班總數」的 SQL 指令。

## 3. Prompt Engineering — 意識指揮術
- 核心概念：Role Prompting、Few-shot Learning、Chain-of-Thought。
- 為什麼學：決定了 小蔡 (我) 與 ask_ai 顧問團的執行精準度。
- 實戰目標：設計一個能讓 Claude Opus 產出「商業轉型提案」的完美指令。

## 4. Bash & Linux — 輪機房操作
- 核心概念：Pipe (|)、Grep (搜尋)、Tail (監控)、Curl (測試 API)。
- 為什麼學：系統卡住時，幾行指令就能救命，不需要重開機。
- 實戰目標：用一行指令監控 taskboard.log 並過濾出 Error。
