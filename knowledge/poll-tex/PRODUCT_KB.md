# 普特絲防霾紗窗產品知識庫

## 1. 產品核心技術
- **技術來源**：荷蘭 Poll-tex 原裝進口。
- **核心原理**：
    - **靜電吸附**：紗線中混紡天然四價貴金屬，形成穩定正電荷層，吸附帶電荷的空氣微粒（PM2.5、花粉）。
    - **物理攔截**：1.38 x 0.35mm 細緻網格，阻擋蚊蟲與部分灰塵。
- **壽命與維護**：
    - 靜電力不因水洗消失（通過 50 分鐘 30 次機洗測試）。
    - 清潔建議：住宅區 3 個月一次，重劃區 1 個月一次。使用清水沖洗或濕抹布擦拭，不需刷子或清潔劑。

## 2. 認證與功效
- **PM2.5 濾淨率**：98.8%（交大環工所測試）。
- **過敏原過濾率**：90%（歐盟 ECARF 抗敏認證）。
- **蚊蟲阻擋率**：100%（包含小黑蚊）。
- **異味過濾**：對煙味、燒金紙味有平均 48.1% 的阻隔力（德國 DMT 實驗室）。

2. **折紗/捲紗**：若原本是隱形紗窗（折紗），更換普特絲需改裝專用框架，費用另計。
3. **高空作業**：若需吊車或特殊外牆施工，費用另計。
4. **包框工程**：陽台包框每才約 $1,000（含鋁料）。

## 4. 關鍵評估指標 (Qualified Lead Indicators)
1. **過敏需求**：家中有成員過敏（鼻塞、氣喘、小孩）。
2. **環境因素**：鄰近馬路、工地、鄰居抽菸、有蚊蟲困擾。
3. **窗型結構**：橫拉窗（最易安裝）、推射窗、折紗（需改框）。
4. **預算認知**：是否了解防霾紗窗單價高於一般紗窗（約 10-20 倍）。
