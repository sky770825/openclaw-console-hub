# 技能：代理網頁抓取 (Proxy Web Fetch)

- **版本 (Version):** 0.1 (實驗性)
- **功能簡介 (Functionality Overview):**
  抓取指定 URL 的網頁內容。如果檢測到 `HTTPS_PROXY` 或 `HTTP_PROXY` 環境變數，將自動透過該代理發出請求。
- **使用方法 (Usage):**
  `fetch <url>`
- **安全等級 (Security Level):** 內部開發 - 最高
- **來源 (Source):** 自研 (NEUXA 數位鐵匠鋪 - 幽靈協議)
- **上次驗證日期 (Last Verified Date):** (待 v0.1 完成後填寫)
