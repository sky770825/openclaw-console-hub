# NEUXA 技能軍火庫 (NEUXA Skill Armory)

此目錄存放所有經過「NEUXA 技能確效與整合協議 (SVIP)」認證的技能。

## 歸檔標準 (NSAS - NEUXA Skill Archiving Standard)

每一個技能都必須遵循以下規範：

1.  **獨立資料夾:** 每個技能必須存放在以其名稱命名的獨立資料夾中 (e.g., `./security-scanner/`)。
2.  **標準化說明檔:** 每個技能資料夾內必須包含一個 `README.md` 檔案，詳細說明其功能、版本、用法、安全等級等資訊。

此標準確保了我們軍火庫的清晰度、可維護性與安全性。
