# 技能：數據偵察兵 (Data Inspector)

- **版本 (Version):** 1.0
- **功能簡介 (Functionality Overview):**
  接收一個公開的 CSV 檔案 URL，使用 `pandas` 進行讀取，並返回其基礎的數據結構與統計摘要（前五行、欄位資訊、敘述性統計）。
- **使用方法 (Usage):**
  `inspect <csv_url>`
- **安全等級 (Security Level):** 內部開發 - 最高 (源自 S 級認證資源)
- **來源 (Source):** 自研 (NEUXA 數位鐵匠鋪 - 奇美拉計畫)
- **上次驗證日期 (Last Verified Date):** 2026-02-27
