# Skill: Security Scanner

## Description
Scans a public Git repository for vulnerabilities using Trivy.

## Usage
- `scan <repo_url>`: Scans the specified public repository URL.
- `scan --scan-path <local_path>`: Scans a local directory.

## Example
`scan https://github.com/aquasecurity/trivy-ci-test`
`scan --scan-path ../../skills`
