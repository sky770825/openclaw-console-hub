# 訂閱者分群策略

## 核心分群

| 分群 | 定義 | 比例目標 |
|------|------|----------|
| 🆕 新訂閱者 | 加入 < 30 天 | 持續增長 |
| ✅ 活躍客戶 | 90 天內有預約 | > 40% |
| 😴 沉睡客戶 | 90+ 天未預約 | < 30% |
| ⭐ VIP | 累計 ≥ 5 次預約 | 培養至 15% |
| 👋 流失風險 | 開信率連續 3 期 < 10% | < 10% |

## 各分群內容策略

### 🆕 新訂閱者
- **頻率**: 前 7 天密集（歡迎序列），之後每週 1 次
- **內容**: 品牌故事、服務介紹、首次預約優惠
- **目標**: 轉換為首次預約客戶
- **CTA**: 「預約首次體驗」

### ✅ 活躍客戶
- **頻率**: 每兩週 1 次
- **內容**: 新服務推薦、保養知識、會員專屬活動
- **目標**: 提升客單價、交叉銷售
- **CTA**: 「探索新服務」「預約下次護理」

### 😴 沉睡客戶
- **頻率**: 每月 1 次
- **內容**: 回饋優惠、新服務公告、季節保養提醒
- **目標**: 喚回預約
- **CTA**: 「專屬回歸禮：首次回訪 8 折」

### ⭐ VIP
- **頻率**: 每週 1 次
- **內容**: 獨家預覽、優先預約、生日驚喜、邀請制活動
- **目標**: 維持忠誠、創造口碑
- **CTA**: 「VIP 專屬預約」「推薦好友享雙重優惠」

### 👋 流失風險
- **頻率**: 觸發式（最後一封再參與郵件）
- **內容**: 「我們想念你」+ 問卷調查（為什麼不開信？）
- **目標**: 重新激活或清理名單
- **CTA**: 「告訴我們你的想法」
- 若仍無互動 → 60 天後移出活躍名單（保持名單健康度）

## 分群觸發條件

| 事件 | 動作 |
|------|------|
| 新訂閱 | 加入「新訂閱者」→ 啟動歡迎序列 |
| 完成首次預約 | 移至「活躍客戶」 |
| 累計 5 次預約 | 升級為「VIP」 |
| 90 天未預約 | 移至「沉睡客戶」→ 觸發回饋郵件 |
| 連續 3 期未開信 | 標記「流失風險」→ 觸發再參與郵件 |
| 再參與郵件後仍未互動 (60天) | 移出活躍名單 |

## 資料收集

需要追蹤的欄位：
- 訂閱日期
- 最後預約日期
- 累計預約次數
- 偏好服務類型（臉部/身體/美甲/美睫）
- 開信/點擊行為
- 來源管道（官網/社群/門市）

## 名單健康維護

- **月度清理**: 移除硬退信地址
- **季度檢視**: 分群比例是否健康
- **年度重整**: 全面 re-engagement campaign
