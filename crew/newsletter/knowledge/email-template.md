# 電子報模板與規範

## 主旨行公式

| 公式 | 範例 |
|------|------|
| 數字 + 利益 | 「3 個讓肌膚持續透亮的居家護理秘訣」 |
| 問題式 | 「你的保養順序做對了嗎？」 |
| 時效性 | 「本週限定：深層保養體驗 85 折」 |
| 個人化 | 「[姓名]，專屬你的春季護膚方案」 |
| How-to | 「如何在 5 分鐘內完成睡前保養」 |

### 主旨行規則
- 長度：30-50 字元（含空格）
- 含數字或具體利益
- 避免 spam trigger words（免費！、限時！、點擊！）
- 預覽文字 40-60 字元，**不重複**主旨內容

## 版面規範

| 項目 | 規格 |
|------|------|
| 寬度 | 600px（最大） |
| 排版 | 單欄（行動優先） |
| 字體大小 | ≥ 14px（內文）、≥ 20px（標題） |
| 行高 | 1.5 - 1.6 |
| CTA 按鈕 | 最小 44px 高、對比色、圓角 |
| 圖片 | 有 alt 文字、寬度 100% 自適應 |
| 配色 | 品牌色系（rose #c7938e、gold #c9a96e） |

## KPI 目標

| 指標 | 目標 | 警戒線 |
|------|------|--------|
| 開信率 | > 25% | < 18% 需檢討主旨 |
| 點擊率 | > 3% | < 1.5% 需檢討內容/CTA |
| 退訂率 | < 1% | > 2% 需檢討頻率/內容品質 |
| 投訴率 | < 0.1% | > 0.3% 立即暫停並檢查 |

## 發送前檢查清單

1. [ ] 主旨行 < 50 字元且含具體利益
2. [ ] 預覽文字已設定且不重複主旨
3. [ ] 所有連結可點擊且指向正確頁面
4. [ ] 退訂連結存在且有效
5. [ ] 行動裝置預覽正常（單欄、字體夠大）
6. [ ] 無拼字錯誤（經 review 代理審核）
7. [ ] 個人化欄位 [姓名] 有 fallback 值
