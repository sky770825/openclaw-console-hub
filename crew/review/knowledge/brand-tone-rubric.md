# 品牌語氣評分標準

## 評分維度（各 1-5 分）

### 1. 清晰度 (Clarity)
| 分數 | 標準 |
|------|------|
| 5 | 一讀就懂，邏輯清晰，無歧義 |
| 4 | 整體清楚，少數句子可更精簡 |
| 3 | 大致理解，但部分段落需重讀 |
| 2 | 含糊不清，讀者需要猜測意圖 |
| 1 | 難以理解，邏輯混亂 |

### 2. 準確度 (Accuracy)
| 分數 | 標準 |
|------|------|
| 5 | 所有事實正確，來源可查 |
| 4 | 核心正確，極少數細節可改進 |
| 3 | 大致正確，有 1-2 處需修正 |
| 2 | 多處不準確或過時 |
| 1 | 重大事實錯誤 |

### 3. 語氣 (Tone)
| 分數 | 標準 |
|------|------|
| 5 | 完美符合品牌調性：專業、溫暖、可信 |
| 4 | 整體符合，偶有偏離 |
| 3 | 可接受但缺乏品牌特色 |
| 2 | 明顯偏離（太隨意/太生硬/太銷售導向） |
| 1 | 完全不符（粗魯/clickbait/誇大） |

### 4. 可行動性 (Actionability)
| 分數 | 標準 |
|------|------|
| 5 | 讀者明確知道下一步該做什麼 |
| 4 | CTA 存在但可以更明確 |
| 3 | 有暗示但無明確引導 |
| 2 | 讀者不確定該怎麼行動 |
| 1 | 無任何行動引導 |

## 通過標準
- **單項不得低於 3 分**
- **總分 ≥ 16 / 20** → 通過
- **總分 14-15** → 小修後通過
- **總分 ≤ 13** → 退回重寫

## 語氣通過 vs 不通過範例

### ✅ 通過範例
> 「每個人的肌膚都是獨特的。我們的首席美容師會在護理前進行詳細的膚質分析，為您量身打造最適合的保養方案。」

**為什麼通過**: 專業但不冷淡、以客戶為中心、有具體細節、自然不做作

### ✅ 通過範例
> 「換季時肌膚容易感到乾燥不適，這是正常的生理反應。建議在日常保養中加入保濕精華，幫助肌膚平穩度過轉換期。」

**為什麼通過**: 教育性語氣、不恐嚇、提供實用建議、溫和專業

### ❌ 不通過範例
> 「震驚！你一直用錯的保養方式，難怪皮膚越來越差！」

**為什麼不通過**: clickbait 標題、恐嚇語氣、負面框架、缺乏專業感

### ❌ 不通過範例
> 「本店提供各種美容服務，歡迎來電洽詢。」

**為什麼不通過**: 太空泛、無品牌特色、無具體價值、機械化語氣

### ❌ 不通過範例
> 「限時大特惠！！！錯過再等一年！！立刻搶購！！！」

**為什麼不通過**: 過度銷售、多餘驚嘆號、壓力式推銷、不符合品牌調性

## 快速判定清單（5 個 Yes/No）

1. [ ] 讀者能在 5 秒內理解主要訊息嗎？
2. [ ] 語氣是「專業朋友」而非「銷售員」嗎？
3. [ ] 內容聚焦客戶利益（非公司自誇）嗎？
4. [ ] 是否避免了 clickbait/恐嚇/過度承諾？
5. [ ] CTA 是否自然不強迫？

**5 個 Yes** = 快速通過 | **4 個 Yes** = 詳細評分 | **≤ 3 個 Yes** = 退回
