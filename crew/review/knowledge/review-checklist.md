# 審查清單

## 所有內容必查
- [ ] 無敏感資訊洩漏（API key、密碼、token）
- [ ] 事實準確（數據有來源）
- [ ] 符合品牌調性
- [ ] 無語法錯誤
- [ ] 格式正確（標題、段落、列表）

## 對外發布內容額外檢查
- [ ] SEO 標記完整（title、meta、keywords）
- [ ] 配圖建議合理
- [ ] CTA 明確
- [ ] 無法律風險（版權、誹謗）

## 代碼額外檢查
- [ ] 無 SQL injection 風險
- [ ] 無 XSS 風險
- [ ] 有適當的錯誤處理
- [ ] 無硬編碼的 credentials
- [ ] 有必要的註解
