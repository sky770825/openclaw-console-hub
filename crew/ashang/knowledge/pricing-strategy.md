# 定價策略方法論

> 學習自外部系統「財務蝦」的定價分析能力

## 定價模型

### 1. 成本加成定價
```
售價 = 成本 × (1 + 利潤率)

適用：標準化服務、實體產品
優點：簡單、可預測
缺點：忽略市場需求和競品
```

### 2. 價值導向定價
```
售價 = 客戶感知價值 × 價值捕獲比率

適用：差異化服務、知識型產品
優點：利潤最大化
缺點：需要深入了解客戶
```

### 3. 競品對標定價
```
售價 = 競品均價 × 定位係數
  定位係數 > 1.0：高端定位
  定位係數 = 1.0：跟隨策略
  定位係數 < 1.0：價格戰

適用：成熟市場、多競品環境
```

### 4. 分層定價（SaaS 常用）
```
| 方案 | 價格 | 功能 | 目標客群 |
|------|------|------|---------|
| 免費 | $0 | 基礎功能 | 體驗、引流 |
| 基礎 | $X/月 | 核心功能 | 小型用戶 |
| 專業 | $3X/月 | 進階功能 | 中型用戶 |
| 企業 | 客製 | 全功能+支援 | 大型客戶 |
```

## 定價分析模板

```markdown
# 定價分析：{產品/服務名稱}

## 成本結構
| 項目 | 固定成本/月 | 變動成本/單位 |
|------|-----------|-------------|
| ... | $... | $... |

## 競品價格
| 競品 | 方案 | 月費 | 功能差異 |
|------|------|------|---------|
| ... | ... | $... | ... |

## 建議定價
- 方案 A：$___/月（理由：___）
- 方案 B：$___/月（理由：___）

## 敏感度分析
- 價格 +10%：預估流失 ___% 客戶
- 價格 -10%：預估增加 ___% 客戶
```

## 利潤分析公式

```
毛利率 = (營收 - 直接成本) / 營收 × 100%
淨利率 = (營收 - 所有成本) / 營收 × 100%
損益兩平點 = 固定成本 / (單價 - 單位變動成本)
```
