# 互動回覆框架（Engagement Response Framework）

## 語氣原則
- **溫暖**：像朋友般親切，不是冷冰冰的企業帳號
- **專業**：用詞得體，展現專業知識
- **樂於助人**：主動提供有用資訊
- **絕不防禦性**：即使被誤解也不辯駁，先傾聽再回應

---

## 回覆分類與處理

### 1. 正面評論（Positive Comments）

**回覆時限**：12 小時內

**處理原則**：
- 真誠感謝，不要制式回覆
- 個人化回應：提及對方的具體內容
- 提出後續問題，延續對話

**範例**：

> 留言：「上次去做臉超舒服的！皮膚變好透亮～」

✅ 好的回覆：
```
謝謝你的分享！聽到皮膚變透亮真的好開心 🤍
你上次做的是水光煥膚對嗎？想問問後續居家保養有沒有什麼想了解的，可以幫你推薦喔！
```

❌ 不好的回覆：
```
謝謝支持！
```

**回覆公式**：感謝 → 回應具體內容 → 提問/延伸話題

---

### 2. 提問（Questions）

**回覆時限**：4 小時內

**處理原則**：
- 直接回答問題，簡潔明確
- 若問題複雜，提供簡答 + 引導進一步聯繫
- 涉及預約或價格，導向預約系統或私訊

**範例**：

> 留言：「請問做一次深層清潔大概多少錢？」

✅ 好的回覆：
```
嗨！深層清潔療程依膚況不同有幾個方案，
價格區間在 NT$1,200-2,500 之間 ✨
建議可以先預約免費膚質諮詢，我們幫你評估最適合的方案！
預約連結在 Bio 喔，或直接私訊我們也可以 💬
```

**回覆公式**：簡答 → 補充說明 → 引導下一步行動（預約/私訊/連結）

**需要導向私訊的情況**：
- 涉及個人膚況評估
- 需要客製化報價
- 包含個人資訊（電話、地址等）

---

### 3. 抱怨（Complaints）

**回覆時限**：2 小時內

**處理原則**：
- 先同理，再道歉（如果確實有疏失）
- 不在公開區域討論細節
- 迅速轉移至私訊處理
- 完整記錄並追蹤

**範例**：

> 留言：「等了 40 分鐘才排到，服務態度也不好，很失望。」

✅ 好的回覆：
```
非常抱歉讓您有不好的體驗，這不是我們期望給客戶的感受。
您的感受我們非常重視，已經私訊您了，
希望能了解更多細節並為您妥善處理，謝謝您願意告訴我們 🙏
```

**回覆公式**：同理/道歉 → 表達重視 → 轉移私訊 → 感謝反饋

**私訊後續流程**：
1. 了解完整情況
2. 提供具體補救方案（如：下次折扣/免費升級療程）
3. 確認客戶滿意
4. 3 天後追蹤關懷

---

### 4. 垃圾訊息 / 濫用（Spam / Abuse）

**處理時限**：發現後立即處理

**處理流程**：
1. **判斷類型**：
   - 垃圾廣告連結 → 檢舉 + 刪除
   - 機器人帳號 → 檢舉 + 封鎖
   - 人身攻擊/歧視言論 → 截圖存證 + 檢舉 + 封鎖 + 刪除
   - 釣魚/詐騙連結 → 檢舉 + 刪除 + 發警示（如有其他用戶受影響）
2. **記錄**：截圖保存，記錄帳號名稱與內容
3. **封鎖**：對惡意帳號執行封鎖
4. **不回覆**：垃圾訊息與濫用一律不回覆，直接處理

**記錄格式**：
```
日期：[YYYY-MM-DD]
平台：[平台名]
帳號：[違規帳號名]
類型：[spam / abuse / phishing]
內容摘要：[簡述]
處置：[檢舉 / 封鎖 / 刪除]
```

## 各平台回覆注意事項

### Instagram
- 回覆留言盡量在 3 行以內
- 善用表情符號增加親和力（但不過度）
- 重要回覆可 pin 到留言區頂部
- Stories 的問答互動要即時回覆

### Facebook
- 可以較詳細的回覆
- 善用「按讚回覆」表達已讀
- 粉絲團留言公開可見，注意措辭

### Twitter/X
- 回覆字數受限，精簡為主
- 可用 Thread 回覆複雜問題
- Tag 相關帳號增加互動

### LINE
- 語氣可更親近，像朋友聊天
- 善用貼圖增加溫度
- 回覆速度要求最高（營業時間 30 分鐘內）

## 互動 KPI 目標

| 指標 | 目標 |
|------|------|
| 平均回覆時間 | < 3 小時 |
| 回覆率 | > 95% |
| 正面互動比率 | > 80% |
| 抱怨解決率 | > 90% |
| 抱怨轉正面評價率 | > 30% |
