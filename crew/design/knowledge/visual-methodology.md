# 視覺設計方法論

> 學習自外部系統「設計蝦」的專業方法

## 60/30/10 配色法則

分配畫面中三種色彩的比例：

```
60% — 主色（背景/大面積區域）
      作用：建立整體氛圍
      選擇：通常為中性色或品牌主色的淺色調

30% — 輔色（次要區域/區塊背景）
      作用：增加層次與視覺變化
      選擇：與主色互補或同色系的深色調

10% — 強調色（CTA 按鈕/重點文字/圖標）
      作用：引導視線、突出重點
      選擇：高對比色，用於互動元素
```

### 實際應用
```
網頁：背景白(60%) + 區塊灰(30%) + 按鈕藍(10%)
Email：底色白(60%) + 標題區品牌色(30%) + CTA橘(10%)
社群圖：背景照片(60%) + 文字區塊(30%) + 品牌色標記(10%)
```

## Moodboard 製作流程

### 步驟
1. **蒐集靈感** — 從 Pinterest/Dribbble/Behance 蒐集 15-20 張參考圖
2. **分類整理** — 按「色彩」「材質」「排版」「氛圍」分組
3. **精選定調** — 從每組選 2-3 張最能代表目標風格的
4. **組合排版** — 將精選圖排列成一張 Moodboard
5. **加註說明** — 標註關鍵字：色號、字體、間距、質感描述

### Moodboard 模板
```markdown
# Moodboard: {專案名稱}

## 風格關鍵字
{3-5 個形容詞，如：溫暖、質感、簡約、專業、自然}

## 色彩方案
- 主色：#_____ （{色名}）
- 輔色：#_____ （{色名}）
- 強調色：#_____ （{色名}）

## 字體建議
- 標題：{字體名}
- 內文：{字體名}

## 參考圖片
{附上參考連結或描述}

## 設計原則
- {原則 1}
- {原則 2}
```

## Before/After 案例分析法

展示改進效果的標準格式：

```markdown
## Before（改進前）
- 問題 1：{具體描述}
- 問題 2：{具體描述}
- [截圖/描述]

## After（改進後）
- 改進 1：{具體描述}
- 改進 2：{具體描述}
- [截圖/描述]

## 改進要點
1. {改了什麼} → {為什麼這樣改} → {效果}
```

## Less is More 原則

設計審查時的減法檢查清單：
- [ ] 能用留白代替線條分隔嗎？
- [ ] 能減少一種顏色嗎？
- [ ] 能減少一種字體嗎？
- [ ] 能用圖標代替文字嗎？
- [ ] 每個元素都有存在的理由嗎？
