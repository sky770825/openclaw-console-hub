# 阿研關於 Vercel 與 Linear 動畫實作教學的整理摘要

阿研的報告詳細整理了 Vercel 和 Linear 網站動畫實作的教學資源。

## Vercel 動畫實作教學重點：
- **流暢導航動畫**：可使用純 CSS + JS 滾動監聽器，或 Framer Motion + React Router 實現「方向感知」導航。
- **文字漸變動畫**：純 CSS @keyframes 即可實現連續漸變效果。
- **流動線條/管道效果**：利用 CSS 關鍵幀動畫、SVG 元素實現。
- **圖片庫動畫**：Next.js + Framer Motion 實現快速、動畫化圖片庫。
- **開書動畫**：Framer 資源重現 Next.js 網站上的「開書」動畫。

## Linear 動畫實作教學：
- **目前沒有直接找到針對 Linear 產品介面或其特定動畫的實作教學**。
- 相關資訊多為 Linear 與 Vercel 的整合功能，或曾在 Vercel 和 Linear 工作過的設計工程師 Emil 分享的廣泛網頁動畫知識。

## 總結：
- Vercel 網站動畫教學資源豐富，涵蓋多種技術棧。
- Linear 產品特定動畫教學資源較少，需留意資訊的相關性。
