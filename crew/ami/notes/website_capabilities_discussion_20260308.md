*   **核心框架**: Next.js (React) 搭配 TypeScript
*   **互動功能**: React Query/SWR, React Hook Form + Zod
*   **動畫效果**: CSS Transitions/Animations, Framer Motion
*   **RWD**: Tailwind CSS, Flexbox/Grid
*   **LINE LIFF 串接**: LINE Front-end Framework (LIFF) SDK
