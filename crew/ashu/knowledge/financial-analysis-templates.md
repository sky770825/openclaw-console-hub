# 財務與數據分析模板

> 學習自外部系統「財務蝦」的分析方法論

## ROI 分析模板

```markdown
# ROI 分析：{項目名稱}

## 投入成本
| 項目 | 金額 | 類型 |
|------|------|------|
| {成本項 1} | ${金額} | 一次性 / 持續 |
| {成本項 2} | ${金額} | 一次性 / 持續 |
| **總計** | **${總額}** | |

## 預期收益
| 項目 | 金額 | 時間範圍 |
|------|------|---------|
| {收益項 1} | ${金額}/月 | {N}個月 |
| **總計** | **${總額}** | |

## ROI 計算
- ROI = (收益 - 成本) / 成本 × 100%
- 回本期：{N} 個月
- 12 個月 ROI：{N}%

## 風險因素
- {風險 1}：可能影響 ±{N}%
```

## 指標追蹤模板

### 系統效能指標
```markdown
| 指標 | 本週 | 上週 | 變化 | 目標 |
|------|------|------|------|------|
| 任務完成率 | {N}% | {N}% | ↑/↓{N}% | > 90% |
| 平均回應時間 | {N}s | {N}s | ↑/↓{N}s | < 5s |
| 錯誤率 | {N}% | {N}% | ↑/↓{N}% | < 1% |
| 代理活躍率 | {N}/16 | {N}/16 | ±{N} | 16/16 |
```
