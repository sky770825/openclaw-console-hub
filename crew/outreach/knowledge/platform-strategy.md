# 各平台外展策略

## 1. Podcast 合作

| 項目 | 策略 |
|------|------|
| 目標規模 | 1K–50K 聽眾的利基節目 |
| 主題切入 | AI 自動化 × 美業創業、科技工具應用、一人公司效率 |
| 最佳接觸方式 | Email（節目頁面通常有聯絡信箱） |
| 回應時間 | 平均 7-14 天 |
| 成功率基準 | 15-20%（利基節目較高） |
| 準備事項 | 3 個具體主題提案、主講者簡介、過往受訪連結 |

### 策略重點
- 優先選擇有「創業」「科技」「美業」標籤的節目
- 提案中強調可提供獨特觀點（AI + 美業的交叉領域少有人談）
- 錄製前準備 5 分鐘的核心故事（品牌起源 + 轉折 + 成果）
- 節目上線後在自家社群大量分享，建立長期關係

## 2. KOL / 微網紅合作

| 項目 | 策略 |
|------|------|
| 目標規模 | 5K–50K 粉絲的微網紅 |
| 平台 | Instagram（主力）、YouTube（次要）、小紅書（探索） |
| 最佳接觸方式 | Instagram DM 或 Email（查看 Bio 連結） |
| 回應時間 | 平均 3-7 天 |
| 成功率基準 | 10-15% |
| 合作形式 | 服務體驗 → 真實分享（非業配感） |

### 策略重點
- 微網紅比大 KOL 更適合：互動率高、信任感強、成本低
- 篩選標準：粉絲互動率 > 3%、內容與美業/生活風格相關
- 合作模式：免費體驗服務 → 對方自由創作分享（不過度限制腳本）
- 避免一次性合作，建立 3 次以上的長期關係
- 要求對方標註地理位置 + 品牌帳號

## 3. 媒體報導

| 項目 | 策略 |
|------|------|
| 台灣美業媒體 | FashionGuide、女人我最大、ELLE Taiwan |
| 台灣科技媒體 | 數位時代、科技新報、Inside |
| 通用媒體 | 商業周刊（創業故事）、天下雜誌（科技應用） |
| 最佳接觸方式 | Email（編輯部信箱或記者個人） |
| 回應時間 | 平均 7-21 天 |
| 成功率基準 | 5-10%（媒體較嚴格） |

### 策略重點
- **美業角度**（給美業媒體）：「AI 如何改變傳統美業的經營方式」
- **科技角度**（給科技媒體）：「台灣團隊打造 AI 代理自動化系統」
- 準備完整 Press Kit：品牌故事、高解析度圖片、數據亮點、創辦人語錄
- 搭配時事熱點投稿（如 AI 熱潮、美業數位轉型趨勢）
- 記者偏好獨家或首發，提供排他性可提高刊登機會

## 4. 活動 / 研討會

| 項目 | 策略 |
|------|------|
| 目標活動 | 美業博覽會、創業小聚、AI 應用論壇 |
| 參與方式 | 講者（優先）、攤位、工作坊、贊助商 |
| 接觸方式 | 活動主辦方網站報名 或 主辦人 LinkedIn |
| 準備時間 | 活動前 2-3 個月開始接觸 |
| 成功率基準 | 20-30%（活動方持續需要講者） |

### 策略重點
- 講者身份 > 攤位參展（曝光度更高、成本更低）
- 準備 2 個演講主題提案（15 分鐘 + 30 分鐘版本）
- 活動現場收集名片/LINE，48 小時內跟進
- 活動結束後在社群發佈活動回顧，標註主辦方

## 跨平台通用原則

1. **個人化第一** — 永遠不發群發訊息，每封都客製化
2. **價值先行** — 先說「我能為你帶來什麼」，不是「請幫我宣傳」
3. **耐心跟進** — 第一次無回應很正常，7 天後禮貌跟進
4. **長期思維** — 每次合作都是建立關係，不是一次性交易
5. **數據追蹤** — 所有接觸都記錄在 prospect-tracking 系統中
