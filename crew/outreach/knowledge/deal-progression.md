# Deal Progression 合作推進指南

## 收到正面回覆後的流程

### Step 1：確認合作細節
收到「有興趣」的回覆後，立即確認以下事項：

- **合作格式**：評測、教學、聯名、Podcast 訪談、其他
- **時程**：內容製作期限、發布日期、活動日期
- **交付物**：雙方各自需要提供什麼（素材、產品、文案、連結等）
- **補償方式**：參見下方補償模式
- **審核流程**：內容發布前是否需要雙方確認

### Step 2：通知相關 Agent
根據合作需求，brief 以下 agent：

- **Content Agent**：如果需要文案、腳本、talking points
  - 提供：合作對象資訊、內容格式要求、品牌訊息重點、截止日期
- **Design Agent**：如果需要視覺素材（圖片、banner、logo 使用）
  - 提供：素材尺寸規格、品牌視覺指南、截止日期

### Step 3：排程與協調
- 與合作對象確認雙方可配合的時間
- 建立時程表：素材準備 → 內容製作 → 審核 → 發布
- 提前至少 3 天完成所有素材交付

### Step 4：執行日檢查清單
合作執行當天（發布日/錄製日/活動日）：

- [ ] 所有素材已交付且經雙方確認
- [ ] 連結、折扣碼、UTM 等追蹤機制已設定
- [ ] 合作對象已收到最終版本的所有資料
- [ ] 通知達爾合作即將上線
- [ ] 準備好即時回應（留言、問題、突發狀況）

### Step 5：合作完成後
- [ ] 發送感謝訊息（24 小時內）
- [ ] 收集成效數據（觀看數、互動數、導流數、轉換數）
- [ ] 整理成效報告並向達爾回報
- [ ] 在社群分享/轉發合作內容
- [ ] 提出未來合作建議（如成效良好）
- [ ] 更新 tracking.json 狀態為 `completed`

## 補償模式

| 模式 | 適用情境 | 說明 |
|------|----------|------|
| **產品交換** | 微型 KOL（< 10K followers） | 免費使用 OpenClaw 完整功能 + 專屬帳號 |
| **互相曝光** | 同量級的創作者/品牌 | 雙方各自在自有平台推廣對方 |
| **分潤** | 有導購能力的 KOL | 專屬推薦碼，依註冊/付費轉換分潤 |
| **固定費用** | 中大型 KOL（> 20K followers） | 按合作內容支付固定費用，需達爾審批預算 |
| **混合** | 依協商 | 上述模式組合，例如產品交換 + 分潤 |

### 預算審批
- 產品交換與互相曝光：outreach agent 可自行決定
- 分潤合作：需達爾確認分潤比例
- 固定費用：必須達爾審批，提供預估 ROI

## 合約 / 協議要點

即使是非正式合作，也建議以文字（email 或訊息）確認以下事項：

### 必要條款
1. **合作範圍**：具體的交付物與內容描述
2. **時程**：素材交付日期、內容發布日期、合作期間
3. **使用權**：雙方可以如何使用合作產出的內容（轉發、二次使用、期限）
4. **補償細節**：具體的補償內容與支付時間/方式
5. **取消條款**：任一方可在何種條件下取消，以及取消後的處理方式

### 建議條款
6. **修改次數**：內容審核的修改上限（建議最多 2 次）
7. **排他性**：合作期間是否禁止與競品合作
8. **數據分享**：合作後雙方分享哪些成效數據
9. **保密**：補償細節是否保密

### 注意事項
- 所有協議內容需經達爾最終確認
- 保留所有溝通紀錄（截圖或轉寄）
- 正式合約（如涉及金額較大）需法律審查
