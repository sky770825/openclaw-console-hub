# 在地 SEO 策略

> 學習自外部系統的地域型關鍵字方法論

## 在地 SEO 核心概念

在地 SEO 針對「地點 + 服務」的搜尋意圖優化，讓附近的潛在客戶找到你。

```
搜尋範例：
「台北 軟裝設計」
「大安區 居家佈置」
「空間改造 推薦」
```

## 關鍵字結構

### 三層關鍵字策略
```
核心詞（競爭高，流量大）
  例：軟裝設計、居家佈置、空間改造

長尾詞（競爭低，意圖明確）
  例：小坪數客廳軟裝推薦、租屋改造預算 3 萬

地域詞（在地化，高轉換）
  例：台北軟裝設計師、信義區居家佈置服務
```

### 地域關鍵字公式
```
{城市/區域} + {服務類型}
{城市/區域} + {服務類型} + {修飾詞}

修飾詞：推薦、費用、評價、比較、ptt、dcard
```

## Google 商家檔案（GMB）優化

```
必填項目：
- [ ] 商家名稱（含核心關鍵字）
- [ ] 地址（精確到門牌）
- [ ] 營業時間
- [ ] 電話
- [ ] 網站連結
- [ ] 服務類別（主類別 + 副類別）
- [ ] 商家描述（含關鍵字，750 字以內）

加分項目：
- [ ] 每週至少 1 則貼文
- [ ] 回覆所有評論（24 小時內）
- [ ] 上傳高品質照片（每月至少 5 張）
- [ ] Q&A 區預先填寫常見問題
```

## 在地內容策略

### 內容類型
| 類型 | 範例 | SEO 價值 |
|------|------|---------|
| 地區指南 | 「台北 10 間值得參考的軟裝案例」 | 高（地域+長尾） |
| 案例展示 | 「信義區 25 坪公寓改造紀錄」 | 高（地域+視覺） |
| 在地活動 | 「2026 台北居家設計展攻略」 | 中（時效性） |
| 常見問題 | 「台北軟裝設計費用多少？」 | 高（精準意圖） |
| 比較文章 | 「IKEA vs 客製軟裝：優缺點分析」 | 中（資訊意圖） |

### Schema 標記（結構化資料）
```json
{
  "@type": "LocalBusiness",
  "name": "品牌名稱",
  "address": {
    "@type": "PostalAddress",
    "addressLocality": "台北市",
    "addressRegion": "信義區"
  },
  "geo": { "latitude": "25.03", "longitude": "121.57" },
  "priceRange": "$$"
}
```

## 在地連結建設

| 來源 | 方法 | 難度 |
|------|------|:----:|
| 本地商家目錄 | 登錄 Google/Bing/Apple Maps | 低 |
| 行業協會 | 加入公會，取得反向連結 | 中 |
| 本地媒體 | 投稿/受訪取得報導連結 | 高 |
| 合作夥伴 | 互相推薦、客座文章 | 中 |
| 社區論壇 | PTT/Dcard 專業回答取得自然連結 | 中 |
