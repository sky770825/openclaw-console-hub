# 品牌故事與文案技術

> 學習自外部系統「行銷蝦」的 Copywriting 方法論

## 品牌故事框架

### 1. 英雄旅程（Hero's Journey）
```
日常世界 → 召喚冒險 → 遇到導師 → 跨越門檻 → 試煉 → 獲得寶藏 → 歸來
    ↓           ↓           ↓          ↓         ↓         ↓          ↓
 客戶痛點    發現需求    遇到品牌    開始使用   克服困難   獲得成果   推薦他人
```

### 2. Before-After-Bridge（BAB）
```
Before（之前）：描述客戶現在的痛苦狀態
After（之後）：描述使用後的理想狀態
Bridge（橋樑）：你的產品/服務如何連接兩者
```

### 3. Problem-Agitate-Solve（PAS）
```
Problem（問題）：點出客戶的具體問題
Agitate（搧風）：加深問題帶來的痛苦感
Solve（解決）：提出你的解決方案
```

## 文案公式庫

### 標題公式
| 類型 | 公式 | 範例 |
|------|------|------|
| 數字型 | {N}個{方法}讓你{結果} | 5個方法讓你的空間煥然一新 |
| 問題型 | 你是否也有{痛點}？ | 你是否也覺得家裡總是亂？ |
| 教學型 | 如何{動作}而不{代價} | 如何改造空間而不花大錢 |
| 秘密型 | {身份}不會告訴你的{N}件事 | 設計師不會告訴你的3件事 |
| 對比型 | {A} vs {B}：{結論} | DIY vs 找設計師：哪個更值得 |

### CTA（行動呼籲）公式
```
弱 CTA：了解更多 / 點擊查看
中 CTA：立即預約免費諮詢 / 限時優惠，馬上行動
強 CTA：只剩 3 個名額，立即報名 / 今天加入，明天改變
```

### AIDA 結構
```
Attention（注意）：用標題/視覺抓住注意力
Interest（興趣）：提供相關且有價值的資訊
Desire（慾望）：展示成果/見證，激發渴望
Action（行動）：明確的 CTA
```

## 品牌調性指南

寫任何內容前，先確認品牌調性：

| 維度 | 問題 | 範例選項 |
|------|------|---------|
| 正式度 | 像朋友聊天 or 專業顧問？ | 口語化 / 專業但親切 / 正式 |
| 情感度 | 理性分析 or 感性打動？ | 數據導向 / 故事導向 / 混合 |
| 節奏感 | 短句有力 or 長句優雅？ | 短促 / 中等 / 優雅 |
| 用詞 | 生活用語 or 專業術語？ | 白話 / 適度專業 / 行業術語 |

## 內容品質檢查清單

- [ ] 開頭 3 秒能抓住注意力嗎？
- [ ] 有明確的目標讀者嗎？
- [ ] 有具體的價值主張嗎？（不是空泛的「最好的」）
- [ ] 有社會證明嗎？（案例/數據/見證）
- [ ] 有明確的 CTA 嗎？
- [ ] 品牌資訊引用自 brand-facts.json 嗎？
