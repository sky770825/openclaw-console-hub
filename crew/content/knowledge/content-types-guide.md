# 內容類型規範指南

## 1. 部落格文章 (Blog Post)

| 項目 | 規範 |
|------|------|
| 字數 | 800–1,500 字 |
| 結構 | H1 標題 → 引言(2-3句) → H2 段落×3-5 → 結論+CTA |
| SEO | 標題含主關鍵詞、第一段自然帶入關鍵詞、H2 含長尾詞 |
| 語氣 | 知識性但親切，像專業朋友在分享 |
| CTA | 每篇結尾 1 個明確行動呼籲（預約/閱讀更多/訂閱） |
| 圖片 | 每 300 字至少 1 張配圖，含 alt 文字 |

### 範例標題格式
- ✅ 「3 個居家臉部保養技巧，讓肌膚持續透亮」
- ✅ 「第一次做美甲？新手必知的 5 件事」
- ❌ 「美甲介紹」（太模糊）
- ❌ 「震驚！這個方法讓你年輕 10 歲！」（clickbait）

## 2. 社群貼文 (Social Post)

| 平台 | 字數上限 | 特點 |
|------|----------|------|
| Instagram | 2,200 字（建議 150 字內） | 視覺優先，hashtag 5-15 個，emoji 適量 |
| Facebook | 63,206 字（建議 80-200 字） | 可長文，鼓勵互動提問 |
| Twitter/X | 280 字 | 精煉有力，1-2 hashtag |
| LINE | 500 字 | 直接、有用、含連結 |

### 貼文結構
1. **Hook** — 第一句抓住注意力（問題/數據/驚喜）
2. **Body** — 核心內容 2-4 句
3. **CTA** — 引導行動（留言/點連結/分享）
4. **Hashtag** — 按平台規範

### 範例
```
✨ 你知道嗎？90% 的肌膚問題都來自清潔不當。

我們的首席美容師蔡小美分享了一個簡單的三步驟晚間清潔法，
只需要 5 分鐘，就能讓肌膚感受到明顯差異。

👉 完整教學在 Bio 連結
#美容保養 #臉部護理 #主人美業
```

## 3. 電子報內容 (Email Newsletter)

| 項目 | 規範 |
|------|------|
| 主旨行 | < 50 字元，含數字或具體利益 |
| 預覽文字 | 40-60 字元，補充主旨（不重複） |
| 內文 | 300-500 字，單欄排版 |
| CTA | 1 個主要 CTA（按鈕，非連結） |
| 字體 | ≥ 14px，行高 1.5 |
| 按鈕 | 最小 44px 高度，對比色 |

### 主旨行公式
- **數字+利益**: 「3 個讓肌膚透亮的居家護理秘訣」
- **問題式**: 「你的保養順序做對了嗎？」
- **時效性**: 「本週限定：深層保養體驗價」
- **個人化**: 「[姓名]，這是你專屬的護膚方案」

## 4. 服務說明頁 (Service Description)

### 結構
1. **服務名稱** — 簡潔明確
2. **一句話概述** — 核心價值，非功能描述
3. **詳細說明** — 2-3 段，features → benefits 格式
4. **規格資訊** — 價格、時長、適合對象
5. **CTA** — 「立即預約」按鈕

### features → benefits 轉換
- ❌ Feature: 「使用進口精華液」
- ✅ Benefit: 「採用法國進口精華液，深層滲透肌膚底層，喚回自然光澤」

## 5. 登陸頁文案 (Landing Page Copy)

### 結構流程
1. **Hero** — 標題(痛點/願景) + 副標(解決方案) + CTA
2. **價值主張** — 3-4 個核心優勢，icon + 短文
3. **社會證明** — 客戶評價、數據、媒體報導
4. **服務展示** — 2-3 個精選服務卡片
5. **常見問題** — 3-5 個 FAQ
6. **最終 CTA** — 重複主要行動呼籲

### 文案原則
- 標題聚焦「你」（客戶），不是「我們」
- ✅ 「找回屬於你的自信光采」
- ❌ 「我們提供最好的服務」
