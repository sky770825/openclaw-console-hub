# 內容日曆規劃流程

## 內容組合比例

| 類型 | 比例 | 說明 |
|------|------|------|
| 教育型 | 60% | 保養知識、美業趨勢、技巧教學 |
| 推廣型 | 25% | 服務介紹、優惠活動、新品上架 |
| 互動型 | 15% | 問答、投票、UGC、幕後花絮 |

## 週度規劃節奏

### 每週一（規劃日）
1. 回顧上週內容表現（開信率、互動率、流量）
2. 從 journal 代理取得上週重要事件與趨勢
3. 從 seo 代理取得關鍵詞建議
4. 規劃本週 5-7 則內容主題
5. 分配至各平台（blog / social / email）

### 每週二至四（製作日）
- 撰寫初稿
- 整合 SEO 關鍵詞
- 提交 review 代理審核

### 每週五（排程日）
- 審核通過的內容排入發佈佇列
- 確認社群貼文圖文配對
- 預覽電子報版面

## 月度內容主題

| 月份 | 季節主題 | 內容方向 |
|------|----------|----------|
| 1-2月 | 新年煥新 | 新年保養計畫、開運美甲 |
| 3-4月 | 春季保養 | 換季保養、過敏肌對策 |
| 5-6月 | 夏日防護 | 防曬、控油、清爽保養 |
| 7-8月 | 盛夏美麗 | 身體護理、美甲流行色 |
| 9-10月 | 秋季修護 | 曬後修復、保濕加強 |
| 11-12月 | 冬季寵愛 | 聖誕特惠、年末犒賞、禮品卡 |

## 內容審核流程

```
content 初稿
  → seo 關鍵詞檢查
  → review 品質審核
  → design 視覺確認（如需配圖）
  → 排程發佈
  → 追蹤成效 → 回饋下週規劃
```

## 內容表現追蹤指標

| 內容類型 | 主要指標 | 目標 |
|----------|----------|------|
| Blog | 頁面瀏覽 / 停留時間 | >500 views, >2min |
| Social | 互動率 (likes+comments/reach) | >3% |
| Email | 開信率 / 點擊率 | >25% / >3% |

## 跨代理協作

- **journal → content**: 每日事件摘要作為內容靈感來源
- **seo → content**: 關鍵詞建議融入標題與內文
- **content → social**: 提供文案素材，由 social 代理適配各平台
- **content → newsletter**: 提供精選內容摘要，由 newsletter 代理編排
- **content → review**: 所有內容發佈前必須通過審查
