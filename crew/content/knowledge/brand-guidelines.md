# 品牌內容規範 (Brand Guidelines)

---

## 1. Brand Voice 品牌語調

### 核心定位
**專業但親切** — 讓讀者感受到可信賴的專業能力，同時保持溫暖、平易近人的溝通方式。

### 語調特質
| 我們是 | 我們不是 |
|--------|----------|
| 專業的 | 冷冰冰的 |
| 親切的 | 過度熱情/浮誇的 |
| 有信心的 | 自大的 |
| 清晰直接的 | 囉嗦含糊的 |
| 溫暖鼓勵的 | 施壓逼迫的 |

### 不同情境的語調

**部落格文章 (Blog)** — 對話式
- 像跟朋友分享專業知識
- 可以用「你」稱呼讀者
- 適度加入個人觀點與經驗

**服務描述 (Service)** — 清晰 + 簡潔
- 重點在功能與效益
- 短句為主，避免長段落
- 用列表呈現特色與價格

**社群貼文 (Social)** — 互動 + 個人化
- 語氣較輕鬆，可用口語
- 鼓勵互動（提問、留言）
- 搭配視覺素材

**Email 電子報 (Email)** — 溫暖 + 行動導向
- 開頭親切問候
- 內容精簡，一封一重點
- 結尾明確 CTA

## 2. Before / After 範例

### 標題寫法
| 不好 | 好 |
|------|-----|
| 震驚！這個保養秘密讓你年輕10歲！！！ | 日常保養的 3 個關鍵步驟，讓肌膚維持健康光采 |
| 最便宜的美容服務都在這裡 | 高品質美容療程，提供適合你的客製化方案 |
| 不看會後悔的美容知識 | 專業美容師分享：換季保養你該注意的事 |

### 內文寫法
| 不好 | 好 |
|------|-----|
| 我們的服務超級棒，保證讓你滿意！！ | 我們的療程由專業美容師操作，依據你的膚質量身規劃。 |
| 趕快來預約，名額有限要搶喔！ | 歡迎線上預約，讓我們為你安排最合適的時段。 |
| 用了我們的產品皮膚馬上變好 | 持續使用搭配正確保養步驟，逐步改善膚質狀態。 |

### CTA 寫法
| 不好 | 好 |
|------|-----|
| 手刀搶購！！ | 立即預約體驗 |
| 不買就虧大了 | 了解更多療程內容 |
| 限時特惠錯過不再 | 查看本月優惠方案 |

## 3. Key Vocabulary 品牌用語

### 優先使用的詞彙
| 詞彙 | 使用場景 |
|------|----------|
| **客製化** | 描述服務特色（非「量身打造」「獨一無二」） |
| **專業** | 描述團隊與技術能力 |
| **呵護** | 描述服務體驗與態度 |
| **體驗** | 描述到店/使用感受 |
| **光采** | 描述肌膚/外在狀態的正面結果 |

### 用語偏好
| 避免 | 改用 |
|------|------|
| 最好的 | 專業的、高品質的 |
| 保證效果 | 幫助改善、逐步提升 |
| 超級/超棒 | 優質的、用心的 |
| 便宜 | 合理價格、高性價比 |
| 缺點 | 注意事項、建議 |

## 4. Forbidden Patterns 禁止模式

### 絕對禁止
- **Clickbait 標題黨**：不用聳動、誇張、誤導性標題
- **過度承諾**：不承諾「保證見效」「100% 有效」等無法兌現的結果
- **抄襲**：所有內容原創，引用必須標注來源
- **攻擊競爭對手**：不貶低其他品牌或服務
- **未經證實的醫療宣稱**：不宣稱療效，不替代醫療建議

### 需要避免
- 過多驚嘆號（最多 1 個，且少用）
- 全大寫英文（看起來像在吼叫）
- 過度使用 emoji（社群貼文除外，且不超過 3 個）
- 冗長段落（每段不超過 4 句）
- 專業術語堆疊（若使用術語必須附帶簡單解釋）

## 5. 格式規範

### 標題
- 簡短有力，含主要關鍵字
- 中文標題不超過 20 字
- 副標題補充具體資訊

### 段落
- 每段 2-4 句，不寫文字牆
- 段落間留空行
- 重點內容用**粗體**標記

### 列表
- 善用 bullet points 增加可讀性
- 每個列表項控制在 1-2 行
- 平行結構（同一列表的項目用相同句型）

### 引用與數據
- 外部數據必須標注來源
- 統計數字盡量具體（「87% 的顧客」而非「大部分顧客」）
- 專家引述需標明姓名與資歷

### CTA (Call to Action)
- 每篇內容至少 1 個 CTA
- CTA 文字要明確指出下一步動作
- 按鈕文字用動詞開頭（「預約體驗」「了解更多」「查看方案」）
