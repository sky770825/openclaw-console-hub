# Exercise 003: Cookbook Search Quality Test

## Task
1. Run 3 semantic searches with different queries:
   - "how to deploy website"
   - "Telegram bot error handling"
   - "database schema migration"
2. For each query, check if the top result is relevant (score > 0.5)
3. Write results to `~/.openclaw/workspace/reports/exercise-003-result.md`

## Expected Output
A report showing query, top result title, score, and relevance judgment.
