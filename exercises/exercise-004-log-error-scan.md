# Exercise 004: Server Log Error Scan

## Task
1. Read the last 200 lines of `~/.openclaw/automation/logs/taskboard.log`
2. Find all ERROR and WARN level entries
3. Group errors by type/source
4. Write analysis to `~/.openclaw/workspace/reports/exercise-004-result.md`

## Expected Output
A report listing unique error types, frequency, and any patterns.
