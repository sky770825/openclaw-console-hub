# Exercise 002: Task Board Summary

## Task
1. Query Supabase `openclaw_tasks` table for all tasks with status 'pending' or 'in_progress'
2. Count tasks by status and priority
3. Identify any tasks that have been pending for more than 3 days
4. Write a summary report to `~/.openclaw/workspace/reports/exercise-002-result.md`

## Expected Output
A markdown report with task counts, priority breakdown, and stale task warnings.
