# Exercise 001: API Health Check Analysis

## Task
1. Call `curl http://localhost:3011/api/health` and read the response
2. Check if all services (supabase, telegram, n8n, websocket) are configured
3. Report memory usage (rss, heapUsed) and whether it's within healthy range (< 200MB rss)
4. Write a brief summary to `~/.openclaw/workspace/reports/exercise-001-result.md`

## Expected Output
A markdown report with service status table and memory analysis.
